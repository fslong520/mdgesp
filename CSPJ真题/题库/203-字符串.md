### 第16题（2020）阅读程序：字符串编码解码（判断题）
```cpp
#include <iostream>
#include <string>
using namespace std;

char encoder[26], decoder[26];
string st;

int main() {
    for (int i = 0; i < 26; ++i)
        encoder[i] = 'A' + i;
    for (int i = 0; i < 26; ++i)
        decoder[i] = 'A' + i;
    int n;
    cin >> n;
    for (int i = 0; i < n; ++i) {
        int x, y;
        cin >> x >> y;
        char t = encoder[x];
        encoder[x] = encoder[y];
        encoder[y] = t;
    }
    int k = 0;
    for (int i = 0; i < 26; ++i) {
        bool flag = true;
        for (int j = 0; j < 26; ++j) {
            if (encoder[j] == 'A' + i) {
                flag = false;
                break;
            }
        }
        if (flag) {
            encoder[k] = 'A' + i;
            ++k;
        }
    }
    for (int i = 0; i < 26; ++i)
        decoder[encoder[i] - 'A'] = i + 'A';
    cin >> st;
    for (int i = 0; i < st.length(); ++i)
        st[i] = decoder[st[i] - 'A'];
    cout << st;
    return 0;
}
```
- **题型**：程序阅读（判断题）
- **知识点**：字符串、编码
- **第1题**：输入的字符串应当只由大写字母组成，否则在访问数组时可能越界。（）
  - A. 正确
  - B. 错误
  - **答案：A**
  - **解析**：decoder数组只定义了26个大写字母的位置。

### 第22题（2021）阅读程序：Base64解码（判断题）
```cpp
#include <iostream>
#include <string>
using namespace std;

char base[64];
char table[256];

void init() {
    for (int i = 0; i < 26; i++) base[i] = 'A' + i;
    for (int i = 0; i < 26; i++) base[26 + i] = 'a' + i;
    for (int i = 0; i < 10; i++) base[52 + i] = '0' + i;
    base[62] = '+', base[63] = '/';
    for (int i = 0; i < 256; i++) table[i] = 0xff;
    for (int i = 0; i < 64; i++) table[base[i]] = i;
    table['='] = 0;
}

string decode(string str) {
    string ret;
    int i;
    for (i = 0; i < str.size(); i += 4) {
        ret += table[str[i]] << 2 | table[str[i + 1]] >> 4;
        if (str[i + 2] != '=')
            ret += (table[str[i + 1]] & 0x0f) << 4 | table[str[i + 2]] >> 2;
        if (str[i + 3] != '=')
            ret += table[str[i + 2]] << 6 | table[str[i + 3]];
    }
    return ret;
}

int main() {
    init();
    cout << int(table[0]) << endl;
    string str;
    cin >> str;
    cout << decode(str) << endl;
    return 0;
}
```
- **题型**：程序阅读（判断题）
- **知识点**：字符串、Base64编码
- **第22题**：输出的第二行一定是由小写字母、大写字母、数字和"+"、"/"、"="构成的字符串。（）
  - A. 正确
  - B. 错误
  - **答案：B**
  - **解析**：decode函数输出的是解码后的原始字符串，不一定是Base64字符。

## 2016年

### 第10题（2016）关于字符串的正确说法是？
- A. 字符串是一种特殊的线性表
- B. 串的长度必须大于零
- C. 字符串不可以用数组表示
- D. 空格串就是空串
- **答案：A**
- **解析**：字符串是线性表的一种（每个字符一个元素）。空串长度可为0，空格串≠空串。
