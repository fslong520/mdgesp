## 2021年

### 第16题（2021）阅读程序：位运算（判断题）
```cpp
#include <iostream>
using namespace std;

int n;
int a[1000];

int f(int x) {
    int ret = 0;
    for (; x; x &= x - 1) ret++;
    return ret;
}

int g(int x) {
    return x & -x;
}

int main() {
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++)
        cout << f(a[i]) + g(a[i]) << ' ';
    cout << endl;
    return 0;
}
```
- **题型**：程序阅读（判断题）
- **知识点**：位运算
- **第16题**：输入的 n 等于 1001 时，程序不会发生下标越界。（）
  - A. 正确
  - B. 错误
  - **答案：B**
  - **解析**：数组大小1000，输入1001会越界。

### 第16题（2022）阅读程序：位运算（判断题）
```cpp
#include <iostream>
using namespace std;

int main() {
    unsigned short x, y;
    cin >> x >> y;
    x = (x | x << 2) & 0x33;
    x = (x | x << 1) & 0x55;
    y = (y | y << 2) & 0x33;
    y = (y | y << 1) & 0x55;
    unsigned short z = x | y << 1;
    cout << z << endl;
    return 0;
}
```

  - A. 正确
  - B. 错误
  - **答案：A**
  - **解析**：输入不超过15，short足够，unsigned不影响结果。

## 2025年

### 第2题（2025）int x=255; cout<<(x&(x-1)); 输出？ <!-- 已用 -->
- A. 255
- B. 254
- C. 128
- D. 0
- **答案：B**
- **解析**：x=255(11111111₂)，x&(x-1)消除最右1得11111110₂=254。

## 2024年

### 第4题（2024）0至7的4位二进制格雷码是？ <!-- 已用 -->
- A. 0000,0001,0011,0010,0110,0111,0101,1000
- B. 0000,0001,0011,0010,0110,0111,0100,0101
- C. 0000,0001,0011,0010,0100,0101,0111,0110
- D. 0000,0001,0011,0010,0110,0111,0101,0100
- **答案：D**
- **解析**：相邻两个格雷码只有1位不同。D项0000→0001→0011→0010→0110→0111→0101→0100满足性质。

## 2019年

### 第2题（2019）二进制数11 1011 1001 0111和01 0110 1110 1011逻辑与的结果？ <!-- 已用 -->
- A. 01 0010 1000 1011
- B. 01 0010 1000 0001
- C. 01 0010 1000 0011
- D. 01 0010 1001 0011
- **答案：C**
- **解析**：逐位与运算，全1为1否则为0，得01 0010 1000 0011。

### 第14题（2018）统计二进制1个数的缺失代码？
- A. x>>=1
- B. x&=x-1
- C. x|=x>>1
- D. x<<=1
- **答案：B**
- **解析**：x&=x-1 消除最右边的1，循环次数即1的个数。
