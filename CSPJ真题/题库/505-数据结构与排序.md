# 505 数据结构与排序

## NOIP2018 T4 对称二叉树 (tree) — 二叉树递归

【题目描述】
对称二叉树：左右子树交换后结构相同且权值相等。给定二叉树，求最大对称二叉子树的节点数。

输入：第一行n。第二行n个权值vi。接下来n行li ri（左右孩子编号，-1表示不存在）。
输出：最大对称子树节点数。
范围：n≤10^6, vi≤1000。

![图](images/noip2018/page07/page07.png)

样例输入：
```
10
2 2 5 5 5 5 4 4 2 3
9 10
-1 -1
-1 -1
-1 -1
-1 -1
-1 2
3 4
5 6
-1 -1
7 8
```
样例输出：
```
3
```

## CSP-J 2022 T3 逻辑表达式 (expr) — 栈/表达式解析

【题目描述】
计算C++逻辑表达式（0/1, &, |, 括号），并统计短路次数：&短路（a=0则b不计算）和|短路（a=1则b不计算）。嵌套在已短路部分内的不计。

输入：一行字符串s表示逻辑表达式。
输出：第一行表达式值（0或1），第二行两个整数：&短路次数和|短路次数。
范围：|s|≤10^6。

![图](images/2022/page06/page06.png)

样例输入：
```
0&(1|0)|(1|1|1&0)
```
样例输出：
```
1
1 2
```

## CSP-J 2021 T2 插入排序 (sort) — 模拟排序

【题目描述】
q次操作：1 x v修改a[x]为v；2 x询问在插入排序下a[x]的最终位置。

输入：第一行n q。第二行n个整数。接下来q行操作。
输出：每个类型2操作输出位置。
范围：n≤8000, q≤2×10^5, ai≤10^9。

![图](images/2021/page04/page04.png)

样例输入：
```
6 4
3 1 4 5 2 6
2 4
1 2 3
2 4
2 6
```
样例输出：
```
4
3
5
```

## CSP-J 2021 T4 小熊的果篮 (fruit) — 链表模拟

【题目描述】
n个水果排一排（0=苹果,1=橘子）。每次取最长同色连续段中第一个水果，输出编号，重复至取完。

输入：第一行n。第二行n个整数0或1。
输出：n个整数，取出顺序。
范围：n≤2×10^5。

![图](images/2021/page11/page11.png)

样例输入：
```
8
0 1 1 0 0 1 0 0
```
样例输出：
```
1 4 5 3 6 2 7 8
```
