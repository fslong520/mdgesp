# 动态规划

## NOIP2017 普及组

### T3 棋盘 (chess)

![图](images/noip2017/page05/page05.png)

【题目描述】
有一个 m×m 的棋盘，棋盘上有一些格子有颜色，颜色为 0（红色）或 1（黄色），其余格子为无色。你需要从 (1,1) 走到 (m,m)，只能站在有颜色的格子上。每次只能向上下左右四个方向走一格。

如果相邻的两个格子颜色相同，则不需要花费金币；如果颜色不同，则需要花费 1 金币。你可以花费 2 金币施展魔法，将一个与当前所在格子相邻的无色格子暂时变为指定颜色（0 或 1），但魔法不能连续使用——即你必须先走到一个有颜色的格子上，才能再次使用魔法。

求从 (1,1) 到 (m,m) 的最小花费。如果无法到达，输出 -1。

【输入格式】
第一行两个整数 m 和 n，分别表示棋盘大小和有色格子数量。
接下来 n 行，每行三个整数 x, y, c，表示第 x 行第 y 列的格子颜色为 c。保证 (1,1) 和 (m,m) 都在输入中。

【输出格式】
一行一个整数，表示最小花费。如果无法到达，输出 -1。

【样例输入】
```
5 7
1 1 0
1 2 0
2 2 1
3 3 1
3 4 0
4 4 1
5 5 0
```

【样例输出】
```
8
```

【数据范围与约定】
- m ≤ 100
- n ≤ 1000

## NOIP2018 普及组

### T3 摆渡车 (bus)

![图](images/noip2018/page05/page05.png)

【题目描述】
有 n 个人在路边等摆渡车。第 ti 分钟有一个人到达。摆渡车从起点出发，到达终点后立即返回，往返一趟需要 m 分钟。摆渡车可以在任意时刻出发（即可以选择任意整数分钟出发）。

每个人从到达时刻开始等车，直到被摆渡车接走。摆渡车每次可以接走车上所有人（包括之前上车但还没到终点的人），但每个人只能被接走一次。

求所有人的等车时间之和的最小值。一个人的等车时间等于他上车的时刻减去他到达的时刻。

【输入格式】
第一行两个整数 n 和 m，分别表示等车人数和摆渡车往返一趟所需时间。
第二行 n 个整数 t1, t2, ..., tn，表示每个人到达的时刻（按非递减顺序给出）。

【输出格式】
一行一个整数，表示所有人的等车时间之和的最小值。

【样例输入】
```
5 1
0 3 4 4 5
```

【样例输出】
```
0
```

【数据范围与约定】
- n ≤ 500
- m ≤ 100
- 0 ≤ ti ≤ 4 × 10^6

## CSP-J 2019

### T3 纪念品 (souvenir)

![图](images/2019/page06/page06.png)

【题目描述】
小W是国王，他有n种纪念品，每种纪念品无限多个。小W打算把纪念品卖给游客。

小W在第1天到第T天每天对每种纪念品制定一个价格。第i种纪念品在第j天的价格为pi,j。游客每天可以选择买入若干个纪念品（每种纪念品可以买入任意多个），也可以选择卖出若干个纪念品（卖出的纪念品必须是之前买入的）。

小W希望知道在第T天结束时，自己最多能获得多少金币。初始时小W没有纪念品，但有M金币。注意同一天买入的纪念品可以在同一天卖出。

【输入格式】
第一行三个整数 T, n, M，分别表示天数、纪念品种类数和初始金币数。
接下来 T 行，每行 n 个整数，第 j 行第 i 个整数表示第 j 天第 i 种纪念品的价格。

【输出格式】
一行一个整数，表示第 T 天结束时小W最多能获得的金币数。

【样例输入】
```
3 2 10
3 2
4 3
5 4
```

【样例输出】
```
12
```

【数据范围与约定】
- T ≤ 100
- n ≤ 100
- M ≤ 10^3
- 1 ≤ pi,j ≤ 100

## CSP-J 2020

### T4 方格取数 (grid)

![图](images/2020/page09/page09.png)

【题目描述】
在一个 n 行 m 列的方格中，每个格子有一个整数。你需要从 (1,1) 走到 (n,m)，每次只能向右、向上或向下走一格（不能走到已经走过的格子）。

求经过的所有格子中的数字之和的最大值。

【输入格式】
第一行两个整数 n 和 m，分别表示方格的行数和列数。
接下来 n 行，每行 m 个整数，表示方格中每个格子的数字。

【输出格式】
一行一个整数，表示经过格子数字之和的最大值。

【样例输入】
```
3 3
1 2 3
4 5 6
7 8 9
```

【样例输出】
```
35
```

【数据范围与约定】
- n, m ≤ 1000

## CSP-J 2022

### T4 上升点列 (point)

![图](images/2022/page09/page09.png)

【题目描述】
平面上有 n 个给定的点，坐标为 (xi, yi)。你可以自由添加最多 k 个整点（坐标为整数的点）到平面上。

你需要选择若干个点（包括给定的点和你添加的点），将它们排列成一个序列 (p1, p2, ..., pm)，使得：
1. 对于 1 ≤ i < m，pi 和 pi+1 之间的欧氏距离恰好为 1。
2. 序列中所有点的 x 坐标和 y 坐标都是非递减的（即 p1.x ≤ p2.x ≤ ... ≤ pm.x 且 p1.y ≤ p2.y ≤ ... ≤ pm.y）。

求序列的最大长度 m。

【输入格式】
第一行两个整数 n 和 k，分别表示给定点数和最多可添加的点数。
接下来 n 行，每行两个整数 xi 和 yi，表示给定点的坐标。

【输出格式】
一行一个整数，表示序列的最大长度。

【样例输入】
```
8 2
3 1
3 2
3 3
3 6
1 2
2 2
5 5
5 3
```

【样例输出】
```
8
```

【数据范围与约定】
- n ≤ 500
- k ≤ 100
- xi, yi ≤ 10^9

## CSP-J 2025

### T3 异或和 (xor)

![图](images/2025/page08/page08.png)

【题目描述】
给定一个长度为 n 的序列 a1, a2, ..., an 和一个非负整数 k。你需要选择若干个互不相交的区间，使得每个区间的异或和都恰好等于 k。求最多能选多少个区间。

两个区间互不相交意味着它们没有公共元素。一个区间的异或和定义为区间内所有元素的异或值。

【输入格式】
第一行两个整数 n 和 k，分别表示序列长度和目标异或值。
第二行 n 个整数 a1, a2, ..., an，表示给定的序列。

【输出格式】
一行一个整数，表示最多能选的区间数。

【样例输入1】
```
4 2
2 1 0 3
```

【样例输出1】
```
2
```

【样例输入2】
```
4 3
2 1 0 3
```

【样例输出2】
```
2
```

【样例输入3】
```
4 0
2 1 0 3
```

【样例输出3】
```
1
```

【数据范围与约定】
- n ≤ 5 × 10^5
- 0 ≤ k < 2^20
- 0 ≤ ai < 2^20
