### 第28题（2022）阅读程序：二分查找求平方根（判断题）
```cpp
#include <iostream>
using namespace std;

int n, k;

int solve1() {
    int l = 0, r = n;
    while (l <= r) {
        int mid = (l + r) / 2;
        if (mid * mid <= n) l = mid + 1;
        else r = mid - 1;
    }
    return l - 1;
}

double solve2(double x) {
    if (x == 0) return x;
    for (int i = 0; i < k; i++)
        x = (x + n / x) / 2;
    return x;
}

int main() {
    cin >> n >> k;
    double ans = solve2(solve1());
    cout << ans << ' ' << (ans * ans == n) << endl;
    return 0;
}
```
- **题型**：程序阅读（判断题）
- **知识点**：二分查找、数学
- **第28题**：该算法最准确的时间复杂度分析结果为O(log n + k)。（）
  - A. 正确
  - B. 错误
  - **答案：A**
  - **解析**：solve1二分查找O(log n)，solve2迭代k次O(k)，总复杂度O(log n + k)。
