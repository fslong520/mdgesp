# C++语法基础

## 2025年

### 第9题（2025）关于C++ string类的说法正确的是？
- A. string长度创建后不能改变
- B. 可用+连接string和char
- C. length()和size()返回值可能不同
- D. string以\0结尾且计入length()
- **答案：B**
- **解析**：string长度可动态改变（A错），length()=size()（C错），\0不计入length（D错）。+可连接string与char（B对）。

### 第10题（2025）solve(x,y)调用后x和y的值？
```cpp
void solve(int&a, int b){
    a=a+b; b=a-b; a=a-b;
}
```
- A. 5,10
- B. 10,5
- C. 10,10
- D. 5,5
- **答案：C**
- **解析**：a是x的引用，b是y的传值副本。a=15,b=5,a=10 → x=10,y=10（y不变）。

## 2024年

### 第6题（2024）哪个不是C++基本数据类型？
- A. int
- B. float
- C. struct
- D. char
- **答案：C**
- **解析**：struct是复合类型（结构体），int/float/char是基本数据类型。

### 第7题（2024）哪个不是C++循环语句？
- A. for
- B. while
- C. do-while
- D. repeat-until
- **答案：D**
- **解析**：repeat-until是Pascal语言的循环语句，C++不支持。

### 第8题（2024）(char)('a'+13)等于？
- A. 'm'
- B. 'n'
- C. 'z'
- D. '3'
- **答案：B**
- **解析**：'a' ASCII=97，97+13=110对应'n'。

- **答案：B**

### 第3题（2023）修改union Data的value成员存3.14，正确方式是？
- A. data.value = 3.14;
- B. value.data = 3.14;
- C. data->value = 3.14;
- D. value->data = 3.14;
- **答案：A**
- **解析**：union类型变量用点(.)访问成员。data.value=3.14正确。

## 2022年

### 第3题（2022）int *p=&x; int *q=&y; p=q; 的行为是？
- A. 将x的值赋为201
- B. 将y的值赋为101
- C. 将q指向x的地址
- D. 将p指向y的地址
- **答案：D**
- **解析**：p=q使p和q都存储y的地址，即p指向y。

## 2019年

### 第4题（2019）程序段s=0; for(i=1;i<=c;i++)s=s-1; 的等价语句？
- A. s=a-c
- B. s=a-c
- C. s=a-c
- D. s=a-c
- **答案：A**
- **解析**：循环c次每次减1等价于s=a-c（若初始s=a）。但题目原题s初始为a，故s=a-c。具体选项需核对原题。

## 2016年

### 第12题（2016）循环c次加1等价于？
- A. s=a+b
- B. s=a+c
- C. s=s+c
- D. s=b+c
- **答案：B**
- **解析**：循环从1到c每次s+1，等价于s=a+c。

### 第13题（2016）代码输出k,n的值？
```cpp
int k=4,n=0;
while(n<k){ n++; if(n%3!=0) continue; k--; }
```
- A. 2,2
- B. 2,3
- C. 3,2
- D. 3,3
- **答案：D**
- **解析**：n=1(跳过)→n=2(跳过)→n=3(k→3)→n=4(n<3?不，退出)。最终k=3,n=3。
