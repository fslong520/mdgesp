## NOIP2016 普及组

### T1 铅笔 (Pencil)

小明有 n 支铅笔需要准备。铅笔以盒为单位出售，每盒有 k 支，价格为 p 元。小明最少需要花多少钱？

#### 输入格式

一行三个整数 n, k, p，分别表示需要的铅笔数量、每盒的铅笔数量、每盒的价格。

#### 输出格式

一行一个整数，表示最少需要花的钱数。

#### 样例输入

```
57 2 2
```

#### 样例输出

```
54
```

#### 样例输入2

```
50 30 30
```

#### 样例输出2

```
60
```

#### 数据范围与约定

| 测试点编号 | n | k | p |
|-----------|---|---|---|
| 1~3 | n≤100 | k=1 | p=1 |
| 4~6 | n≤10000 | k=1 | p=1 |
| 7~8 | n≤10000 | k=10 | p=1 |
| 9~10 | n≤10000 | k=100 | p=100 |

其中 n, k, p 均为正整数，且 n ≤ 10000。

给定一个8位数字 d，请判断它是哪种情况：

- 如果 d 不是一个合法的日期，输出 1。

#### 输入格式

一行一个8位整数 d。

#### 输出格式

一行一个整数，表示判断结果。

#### 样例输入1

```
20110101
```

#### 样例输出1

```
3
```

#### 样例输入2

```
20000101
```

#### 样例输出2

```
2
```

年份在 1000~9999 之间。

### T2 龙虎斗 (Fight)

![图](images/noip2018/page03/p03.png)

![图](images/noip2018/page04/p04.png)

一共有 n 个兵营，编号从 1 到 n，排成一排。其中第 m 个兵营是分界点，编号 1~m 的兵营属于龙军，编号 m+1~n 的兵营属于虎军。

每个兵营 i 有 c_i 个士兵。龙军的气势定义为：所有龙军兵营的（士兵数 × 兵营编号）之和。虎军的气势定义为：所有虎军兵营的（士兵数 × 兵营编号）之和。

现在龙军已经在 p1 号兵营增加了 s1 个工兵。为了让两军气势差的绝对值最小，需要在虎军的某个兵营 p2 增加 s2 个工兵。

求在哪个兵营 p2 增加工兵可以使两军气势差的绝对值最小。

#### 输入格式

第一行一个整数 n。

第二行 n 个整数 c_1, c_2, ..., c_n，表示每个兵营的士兵数。

第三行四个整数 m, p1, s1, s2，分别表示分界点、龙军工兵出现的位置、龙军工兵数量、虎军工兵数量。

#### 输出格式

一行一个整数 p2，表示虎军应该在哪个兵营增加工兵。如果有多个答案，输出编号最小的。

#### 样例输入

```
8
1 2 3 4 5 6 7 8
4 4 1 1
```

#### 样例输出

```
7
```

#### 数据范围与约定

- 1 ≤ n ≤ 10^5
- 1 ≤ m ≤ n
- 1 ≤ c_i ≤ 1000
- 1 ≤ p1 ≤ m
- 1 ≤ s1, s2 ≤ 1000

```
5 3
1 5 3 4 2
2 3
1 2 6
2 2
```

#### 样例输出

```
3
5
```

#### 数据范围与约定

- 1 ≤ n ≤ 8000

第一行一个整数 n，表示操作次数。

接下来 n 行，每行一个操作，格式为 `Operation addr`。

#### 输出格式

对于每个操作，输出一行 "OK" 或 "FAIL"。

#### 样例输入

```
5
Server 1.2.3.4:5
Server 1.2.3.4:5
Client 1.2.3.4:5
Client 1.2.3.4:6
Client 1.2.3.4:5
```

#### 样例输出

```
OK
FAIL
OK
FAIL
OK
```

#### 数据范围与约定

- 1 ≤ n ≤ 1000
- 地址格式合法
