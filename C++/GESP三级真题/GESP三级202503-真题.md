## 第 26 题 <a href="https://fslong.iok.la/problem/GESP20250303T1" target="_blank">26、2025（25 分）</a>

**试题名称：** 2025

**时间限制：** 1.0 s

**内存限制：** 512.0 MB

### 题目描述
小 A 有一个整数 x，他想找到最小的正整数 y 使得下式成立：

(x & y) + (x | y) = 2025

其中 & 表示二进制按位与运算，| 表示二进制按位或运算。如果不存在满足条件的 y，则输出 -1。

### 输入格式
一行，一个整数 x。

### 输出格式
一行，一个整数，若满足条件的 y 存在则输出 y，否则输出 -1。

### 样例
**输入样例 1**
```
1025
```
**输出样例 1**
```
1000
```

### 数据范围
对于所有测试点，保证 0 ≤ x < 2025。

### 提示
按位与运算，运算符为 &。
按位或运算，运算符为 |。

## 第 27 题 <a href="https://fslong.iok.la/problem/GESP20250303T2" target="_blank">27、词频统计（25 分）</a>

**试题名称：** 词频统计

**时间限制：** 1.0 s

**内存限制：** 512.0 MB

### 题目描述
在文本处理中，统计单词出现的频率是一个常见的任务。现在，给定 n 个单词，你需要找出其中出现次数最多的单词。在本题中，忽略单词中字母的大小写（即 Apple、apple、APPLE、aPPle 等均视为同一个单词）。

请你编写一个程序，输入 n 个单词，输出其中出现次数最多的单词。

### 输入格式
第一行，一个整数 n，表示单词的个数；

接下来 n 行，每行包含一个单词，单词由大小写英文字母组成。

输入保证，出现次数最多的单词只会有一个。

### 输出格式
输出一行，包含出现次数最多的单词（输出单词为小写形式）。

### 样例
**输入样例 1**
```
6
Apple
banana
apple
Orange
banana
apple
```
**输出样例 1**
```
apple
```

### 数据范围
对于所有测试点，保证 1 ≤ n ≤ 100，每个单词的长度不超过 30，且仅由大小写英文字母组成。
