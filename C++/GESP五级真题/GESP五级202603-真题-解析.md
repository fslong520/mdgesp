## 一、选择题

**第 1 题** D — 带头结点的循环单链表中，判定链表为空只需判断头结点的 next 是否指向自身。A 错在单链表已知某结点指针无法 O(1) 删除前驱（需遍历找到前驱）；B 错在空循环链表头结点的 next 可指向自身，存在空指针含义的情况；C 错在循环双链表尾结点的 next 指向头结点而非 nullptr。

**第 2 题** C — 在双向循环链表的结点 p 前插入 s，需先让 s 的 next 指向 p，s 的 prev 指向 p 的前驱，再将 p 前驱的 next 指向 s，最后 p 的 prev 指向 s。此顺序保证不丢失原有链接。

**第 3 题** B — `cur->next = del->next` 将前驱结点的 next 跳过待删结点，直接指向后继，完成删除。

**第 4 题** A — gcd(48,18) → 48%18=12 → gcd(18,12) → 18%12=6 → gcd(12,6) → 12%6=0 → gcd(6,0) = 6。

**第 5 题** C — 线性筛内层循环遍历已发现的质数列表，故条件为 `j < primes.size()` 防止越界，同时用已有质数筛合数。

**第 6 题** C — 小于 i² 的 i 的倍数（如 2i, 3i, …）必含小于 i 的质因子，已被之前更小的质因子筛过，故从 i² 开始可避免重复标记。

**第 7 题** B — sort 后 a = {1,2,4,8,9}，二分答案求最大最小间距。过程：check(4)→cnt=2<3→r=3；check(2)→cnt=4≥3→l=3；check(3)→cnt=3≥3→l=3，输出 3。

**第 8 题** A — 区间 [l, r) 左闭右开，当 a[mid] ≥ x 时 mid 可能是答案，右边界收缩为 r = mid。

**第 9 题** D — 栈溢出通常导致程序异常终止（SIGSEGV），不会抛出可捕获的异常让程序继续执行。

**第 10 题** A — 二分答案求最大值：check 为真说明 mid 可行，尝试更大值 l = mid + 1；否则过大，缩小 r = mid - 1。

**第 11 题** B — 递推式 T(n) = 2T(n/2) + O(n)，由主定理得 O(n log n)。

**第 12 题** B — 合并两个升序序列时，取较小的元素先放入结果，故条件为 A[i] ≤ B[j]。

**第 13 题** C — 已有序数组以首元素为 pivot，每次划分极不平衡（一侧 0 个元素），递归树退化为链，时间复杂度 O(n²)。

**第 14 题** B — 归并排序是稳定的排序算法，故"快速排序和归并排序都是不稳定的"说法错误。

**第 15 题** B — 竖式除法每步：`rem = rem * 10 + a[i]` 后取商，再 `rem %= b` 得余数带入下一位。

## 二、判断题

**第 16 题** T — 数组随机访问 O(1)；链表在已知结点指针后，在该结点后插入新结点也只需修改相邻指针，为 O(1)。

**第 17 题** T — 代码使用左闭右开区间 [l, r)，当 a[mid] ≥ x 时收缩右边界，否则收缩左边界，是 lower_bound 的标准实现。

**第 18 题** F — 排序稳定性取决于比较相等元素时是否交换位置，与选取哪个元素作为枢轴无关。

**第 19 题** T — T(n) = 2T(n/2) + O(n) 满足主定理第二种情况，结果为 O(n log n)。

**第 20 题** F — 该函数仅实现了归并过程中统计跨左右两半的逆序对，但缺少递归分解的框架函数，无法独立统计整个 [l, r] 的全部逆序对。

**第 21 题** T — 若 n 为合数，必有一个质因子 ≤ √n。反之，若所有 ≤ √n 的质数都不能整除 n，则 n 必为质数，此为试除法判质的理论基础。

**第 22 题** T — 排序 O(n log n)，二分答案约 O(log range) 次迭代，每次 check 为 O(n)。值域为 O(n) 时 range = O(n)，总复杂度 O(n log n)。

**第 23 题** F — 最优子结构是动态规划的必要条件，贪心还需满足贪心选择性质（局部最优能推出全局最优），二者缺一不可。

**第 24 题** F — 线性筛的核心是"每个合数只被其最小质因子筛去"（非最大），通过 `if (i % primes[j] == 0) break;` 保证每个合数恰好被最小质因子标记一次。

**第 25 题** F — 尾递归函数可被编译器优化为循环（如 GCC -O2 的尾调用优化），无需显式栈。非尾递归虽可改写为非递归，但不一定都需要显式栈。
