# GESP C++ 真题集

CCF GESP（编程能力等级认证）C++ 历年真题归档。

## 目录结构

```
GESP三级202509/       ← 2025年09月 三级
GESP三级202512/       ← 2025年12月 三级
GESP三级202603/       ← 2026年03月 三级
GESP四级真题/         ← 四级历年真题
GESP五级真题/         ← 五级历年真题
GESP六级真题/         ← 六级历年真题
```

## 文件格式

| 后缀 | 说明 |
|:-----|:------|
| `真题.md` | Markdown 版试卷 |
| `真题-答案.md` | 答案表 |
| `真题-解析.md` | 答案解析 |
| `真题.pdf` | 官方 PDF 原卷 |
| `真题.json` | JSON 结构化试题（含题型/选项/答案） |
| `题解-{学生}.md` | 学生错题解析 |
| `-text.txt` | 五级试卷文本转写 |
