## 一、重点知识

### 1. 荒地种树问题（P7036）

这是一道典型的搜索与模拟题，核心思路：

- **暴力枚举**：遍历每个位置，尝试移除该位置的障碍物
- **连通性判断**：检查移除障碍物后，该位置是否与已种植的树连通
- **方向数组**：使用`di[]`和`dj[]`数组表示上下左右四个方向

```cpp
int di[]={-1,1,0,0};  // 上下左右
int dj[]={0,0,-1,1};

bool check(int k, int l) {
    for(int h=0; h<4; ++h) {
        int ki=k+di[h], kj=l+dj[h];
        if(ki<0||kj<0||ki>=n||kj>=m) continue;
        if(a[ki][kj]=='#') return 0;  // 相邻有障碍物，不连通
    }
    return 1;
}
```

### 2. 函数参数传递

**值传递 vs 引用传递**：
```cpp
void swap(int a, int &b) {
    // a是值传递，修改不影响原变量
    // b是引用传递，修改会影响原变量
    int temp = a;
    a = b;
    b = temp;
}
```
- `int a`：值传递，函数内修改不影响原变量
- `int &b`：引用传递，函数内修改会影响原变量

### 3. 斐波那契数列递推

```cpp
int f(int n) {
    if (n == 1 || n == 2) return n;
    int f1 = 1;  // f(n-2)
    int f2 = 2;  // f(n-1)
    int res = 0;
    for (int i = 3; i <= n; i++) {
        res = f1 + f2;  // 注意：是 = 不是 +=
        f1 = f2;
        f2 = res;
    }
    return res;
}
```
**关键点**：`res = f1 + f2` 不是 `res += f1 + f2`

### 4. 时间复杂度分析

```cpp
for (int i = 0; i < (1 << n); i++) {  // 2^n次
    for (int j = 0; j < n; j++) {     // n次
        // ...
    }
}
```
- 外层循环：`1 << n` = 2^n
- 内层循环：n次
- 总复杂度：O(n × 2^n)

### 5. 排序稳定性

- **稳定排序**：相同元素的相对顺序不变（冒泡、插入、归并）
- **不稳定排序**：选择排序、快速排序、堆排序

### 6. 冒泡排序过程

```
初始：{5, 3, 8, 1}
第1次：5 > 3，交换 → {3, 5, 8, 1}
第2次：5 < 8，不交换 → {3, 5, 8, 1}
第3次：8 > 1，交换 → {3, 5, 1, 8}
第一轮结束：{3, 5, 1, 8}
```

### 7. 文件输出重定向

```cpp
// 正确方式：
freopen("log.txt", "w", stdout);  // 方式1
std::ofstream outFile("log.txt"); outFile << "Happy Spring!" << endl;  // 方式2
ofstream log_file("log.txt"); cout.rdbuf(log_file.rdbuf());  // 方式3

// 错误方式：
std::ofstream outFile("log.txt");
cout << "Happy Spring!" << endl;  // cout没有关联到文件
```
