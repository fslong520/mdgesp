### 1、P9451 巧克力棒（100 分）

**题目描述**  
你和好朋友分吃一块巧克力。巧克力是 a×b 格的（a 行 b 列），你们想把它掰成 1×1 的小块分享。每次只能沿着格线把一块巧克力掰成两块。问：最少要掰几次？

**输入格式**  
一行两个整数 a 和 b（1 ≤ a, b ≤ 1000）。

**输出格式**  
一个整数，表示最少掰几次。

**样例**
```
输入：
3 4

输出：
11
```

**提示**  
每次掰一下，块数加 1。从 1 块到 a×b 块，需要掰 a×b-1 次。无论怎么掰，答案都一样。

**知识点**：数学思维

**提交链接**：<a href="https://fslong.iok.la/problem/P9455" target="_blank">P9451 - 巧克力棒</a>

### 2、P9452 密码锁（100 分）

**题目描述**  
你面前有一个密码锁，有 n 位数字，每位是 0~9。你可以把任意一位数字 +1 或 -1（0 减 1 变成 9，9 加 1 变成 0）。从全 0 开始，最少要操作几次才能变成目标密码？

**输入格式**  
第一行一个整数 n（1 ≤ n ≤ 100）。  
第二行 n 个数字（0~9），表示目标密码。

**输出格式**  
一个整数，表示最少操作次数。

**样例**
```
输入：
3
7 3 9

输出：
7
```

**提示**  
每一位是独立的。从 0 到目标数字，可以顺时针（+1）或逆时针（-1）转。取两者中较小的：`min(d, 10-d)`，然后全部加起来。

**知识点**：贪心、循环

**提交链接**：<a href="https://fslong.iok.la/problem/P9451" target="_blank">P9452 - 密码锁</a>

### 3、P9453 队友匹配（100 分）

**题目描述**  
有 n 个同学（n 一定是偶数），每人有一个能力值 a[i]。现在要把他们两人一组分成 n/2 组，使得每组内两人能力值之差的**总和最小**。求这个最小总差值。

**输入格式**  
第一行一个偶数 n（2 ≤ n ≤ 200）。  
第二行 n 个整数 a[i]（1 ≤ a[i] ≤ 10000），表示每个同学的能力值。

**输出格式**  
一个整数，表示最小的总差值。

**样例**
```
输入：
4
10 20 30 40

输出：
20
```

**提示**  
把能力值排序后，相邻两人一组就是最优解。代码量很短：排序 + 相邻两两求和。

**知识点**：排序、贪心

**提交链接**：<a href="https://fslong.iok.la/problem/P9452" target="_blank">P9453 - 队友匹配</a>

### 4、P9454 取物游戏（100 分）

**题目描述**  
小明和小红玩一个取物游戏。桌上有 n 个物品排成一行，第 i 个物品的价值为 v[i]。两人轮流从两端取物品（只能取最左边或最右边的），小明先取。两人都足够聪明，都会选择对自己最有利的策略。问：小明能获得的最大总价值是多少？

**输入格式**  
第一行一个整数 n（1 ≤ n ≤ 1000）。  
第二行 n 个整数 v[i]（1 ≤ v[i] ≤ 10000），表示每个物品的价值。

**输出格式**  
一个整数，表示小明能获得的最大总价值。

**样例**
```
输入：
4
8 15 3 7

输出：
23
```

**提示**  
定义 `dfs(i, j)` 表示当前玩家面对区间 [i, j] 能获得的最大净胜分。转移：`dfs(i, j) = max(v[i] - dfs(i+1, j), v[j] - dfs(i, j-1))`。最终小明得分 = (总分 + dfs(0, n-1)) / 2。用二维 `memo[i][j]` 记录已算过的值。

**知识点**：递归、记忆化搜索、博弈论

**提交链接**：<a href="https://fslong.iok.la/problem/P9453" target="_blank">P9454 - 取物游戏</a>

### 5、P9455 宝藏迷宫（100 分）

**题目描述**  
小红闯入了一个 n 行 m 列的宝藏迷宫。第 i 行第 j 列的宝藏价值为 a[i][j]（0 表示无宝藏，-1 表示障碍）。小红从左上角 (1,1) 出发，每次可以向右或向下移动一格（不能进入障碍格），要到达右下角 (n,m)。问：所有路径中，能收集的宝藏最大总价值是多少？如果无法到达，输出 -1。

**输入格式**  
第一行两个整数 n 和 m（1 ≤ n, m ≤ 500）。  
接下来 n 行，每行 m 个整数 a[i][j]（-1 ≤ a[i][j] ≤ 1000）。

**输出格式**  
一个整数，表示能收集的宝藏最大总价值。若无法到达，输出 -1。

**样例**
```
输入：
2 3
0 5 3
2 0 4

输出：
12
```

**提示**  
定义 `dfs(i, j)` 表示从 (i,j) 走到 (n,m) 能收集的最大价值。转移：`dfs(i, j) = a[i][j] + max(dfs(i+1, j), dfs(i, j+1))`。如果 (i,j) 是障碍，返回极小值。用二维 `memo[i][j]` 记录已算过的值。

**知识点**：递归、二维记忆化搜索、路径规划

**提交链接**：<a href="https://fslong.iok.la/problem/P9454" target="_blank">P9455 - 宝藏迷宫</a>
