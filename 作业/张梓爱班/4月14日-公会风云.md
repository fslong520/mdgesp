# 一、选择题（每题 10 分，共 100 分）

## 第 1 题【公会名册】(10分)

你是一名热爱冒险的少年，某天她收到了一封神秘的邀请函：

> *"欢迎加入传说勇者公会！在这里，你将面对各种挑战。只有掌握数据的力量，才能成为真正的公会大师。"*

公会需要记录成员的姓名和等级，以下哪个结构体定义是正确的？

- A. `struct Member { string name, int level; };`
- B. `struct Member { string name; int level; };`
- C. `struct Member ( string name; int level; );`
- D. `struct Member { string name int level; };`

- A. 单关键字排序
- B. 逆序排序
- C. 多关键字排序
- D. 随机排序

## 第 4 题【函数调用】(10分)

```cpp
}
```

- A. 30
- B. 70
- C. 20
- D. 50

## 第 9 题【数组初始化】(10分)
