### 1、<a href="https://fslong.iok.la/problem/P9342" target="_blank">车轮滚一圈</a>（100 分）

**题目描述**  
自行车车轮的半径是 r 厘米。自行车每往前滚一圈，车轮就绕着轴心转了一圈。请你算一算，车轮滚一圈能走多少厘米（周长），以及车轮的圆面有多大（面积）。

圆周率 π 取 3.14。

周长公式：$C = 2 \times \pi \times r$  
面积公式：$S = \pi \times r \times r$

**输入**  
一行，一个整数 r（1 ≤ r ≤ 1000），表示车轮半径（厘米）。

**输出**  
一行，两个浮点数，依次为周长和面积，中间用空格分隔，结果保留一位小数。

**样例输入**
```
10
```

**样例输出**
```
62.8 314.0
```
2 × 3.14 × 10 = 62.8，3.14 × 10 × 10 = 314.0

**提示**
- 用 `double` 存储计算结果
- 用 `printf("%.1f %.1f", c, s)` 保留一位小数
- π 可以定义为 `double pi = 3.14;`

**知识点**：算术运算、浮点数、格式化输出

### 2、<a href="https://fslong.iok.la/problem/P9333" target="_blank">三人比身高</a>（100 分）

**题目描述**  
课间三个好朋友比身高，已知他们的身高分别是 a、b、c 厘米。请你找出谁最高，输出他的身高。

**输入**  
一行，三个整数 a、b、c（100 ≤ 身高 ≤ 220），表示三个人的身高（厘米）。

**输出**  
一行，一个整数，表示最高的身高。

**样例输入**
```
165 178 170
```

**样例输出**
```
178
```

**样例输入 2**
```
180 180 175
```

**样例输出 2**
```
180
```

**提示**
- 先用 if 比较 a 和 b 得出较大者，再和 c 比
- 或者用嵌套 if-else 两两比较
- 也可以用 `max` 函数：`max(max(a, b), c)`

**知识点**：分支结构、关系运算

### 3、<a href="https://fslong.iok.la/problem/P9334" target="_blank">大小写转换</a>（100 分）

**题目描述**  
聊天时同桌发来一个小写字母，你想把它转成大写字母怼回去。请编写程序：输入一个小写字母，输出对应的大写字母。

提示：小写字母和大写字母的 ASCII 码相差 32，'a' = 97，'A' = 65。

**输入**  
一行，一个小写字母（'a' ~ 'z'）。

**输出**  
一行，对应的大写字母。

**样例输入**
```
b
```

**样例输出**
```
B
```

**样例输入 2**
```
z
```

**样例输出 2**
```
Z
```

**提示**
- 大写字母 = 小写字母 − 32
- 或者用 `char(ch - 32)` 转换

**知识点**：ASCII码、字符运算、类型转换

### 4、<a href="https://fslong.iok.la/problem/P9335" target="_blank">跳绳统计</a>（100 分）

**题目描述**  
体育老师让全班 n 个同学跳绳 1 分钟，记录每个人的跳绳个数。老师想知道两件事：有多少人跳过了 100 个（超过 100 就算过关），以及跳得最多的 4 位同学的总和是多少。

**输入**  
第一行，一个整数 n（4 ≤ n ≤ 30），表示同学人数。  
第二行，n 个整数，表示每个同学的跳绳个数（0 ≤ 个数 ≤ 200）。

**输出**  
一行，两个整数，依次为超过 100 的人数和前 4 名跳绳总数，中间用空格分隔。

**样例输入**
```
6
85 120 95 150 60 110
```

**样例输出**
```
3 475
```
超过 100 的有 120、150、110，共 3 人。  
前 4 名：150 + 120 + 110 + 95 = 475。

**提示**
- 统计超 100 人数：每读一个数，判断 > 100 就加 1
- 记前 4 名：用四个变量 m1≥m2≥m3≥m4，每读一个数依次比较并插入合适位置
- 初始值都设成 0

**知识点**：循环、分支、比较

### 5、<a href="https://fslong.iok.la/problem/P9336" target="_blank">两点距离</a>（100 分）

**题目描述**  
体育委员在操场上画了一个直角坐标系，标记了两个打卡点的坐标 (x₁, y₁) 和 (x₂, y₂)。请计算这两个点之间的直线距离。

距离公式：$\sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$

**输入**  
一行，四个整数 x₁、y₁、x₂、y₂（−100 ≤ 坐标 ≤ 100）。

**输出**  
一行，一个浮点数，表示两点之间的距离，**四舍五入保留一位小数**。

**样例输入**
```
1 1 4 5
```

**样例输出**
```
5.0
```
$\sqrt{(1-4)^2 + (1-5)^2} = \sqrt{9 + 16} = \sqrt{25} = 5.0$

**样例输入 2**
```
0 0 3 4
```

**样例输出 2**
```
5.0
```

**提示**
- 用 `sqrt()` 函数计算平方根
- 用 `pow(a, 2)` 或 `a * a` 计算平方
- 用 `printf("%.1f", dist)` 保留一位小数
- 需要包含头文件 `<cmath>`

**知识点**：数学函数、算术运算、格式化输出
