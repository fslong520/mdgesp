    if (x >= 85) {
        cout << "良好";
    } else {
        cout << "中等";

}

### 1、<a href="https://fslong.iok.la/problem/P9249" target="_blank">【每日零花钱】</a>(100分)

**题目描述**  
小明每天存零花钱，第一天存 a 元，之后每天比前一天多存 d 元。坚持 n 天后，他总共存了多少钱？

等差数列求和公式:Sn = n × a₁ + n×(n-1)×d / 2

例如:从 2 开始,公差为 3,前 5 项是 2,5,8,11,14,和为 40。

**输入格式**  
一行,三个整数 a、d、n (1 ≤ a,d ≤ 100, 1 ≤ n ≤ 1000),分别表示首项、公差和项数。

**输出格式**  
一行,一个整数,表示前 n 项的和。

**样例输入**  
```
2 3 5
```

**样例输出**  
```
40
```

**样例输入 2**  
```
1 1 100
```

**样例输出 2**  
```
5050
```

**提示**  
直接套用公式:Sn = n × a + n × (n-1) × d / 2,注意先算乘法再算除法。

### 4、<a href="https://fslong.iok.la/problem/P9252" target="_blank">【成绩倒序】</a>(100分)

**题目描述**  
老师把考试成绩按录入顺序存进了数组，现在需要从后往前打印出来核对。

例如输入 1,2,3,4,5,逆序后变成 5,4,3,2,1。

**输入格式**  
第一行,一个整数 n (1 ≤ n ≤ 100)。  
第二行,n 个整数,每个数不超过 1000。

**输出格式**  
一行,n 个整数,用空格分隔,表示倒序后的成绩。

**样例输入**  
```
5
1 2 3 4 5
```

**样例输出**  
```
5 4 3 2 1
```

**样例输入 2**  
```
3
10 20 30
```

**样例输出 2**  
```
30 20 10
```

**提示**  
方法1:从后往前遍历数组输出。方法2:先用临时数组存储逆序结果再输出。

### 5、<a href="https://fslong.iok.la/problem/P9253" target="_blank">【成绩排序】</a>(100分)

**题目描述**  
期中考试结束了,班主任王老师想看看全班同学的数学成绩分布情况。请把所有成绩从小到大排好序输出。

第二行,n 个整数,表示每个同学的成绩(0 到 100 之间)。

**输出格式**  
一行,n 个整数,用空格分隔,表示从小到大排好序的成绩。

**样例输入**  
```
5
85 92 78 92 88
```

**样例输出**  
```
78 85 88 92 92
```

**样例输入 2**  
```
3
60 100 75
```

**样例输出 2**  
```
60 75 100
```

**提示**  
可以使用插入排序或冒泡排序实现。注意成绩可能有重复值。
