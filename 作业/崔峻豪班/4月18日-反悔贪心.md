### 1、<a href="https://fslong.iok.la/problem/AWC0047_A" target="_blank">Temperature Changes on a Mountain Trail</a>(100分)

**题目描述**  
小明正在参加登山探险活动。山路上有 N 个检查点,编号从 1 到 N,每个检查点都有一个温度值 Ti。小明从检查点 1 出发,按顺序经过所有检查点到达山顶。

在检查点 i(i≥2),如果与前一个检查点的温度差绝对值至少是阈值 D,即 |Ti - Ti-1| ≥ D,则称在检查点 i 发生了**温度突变**。

小明需要你帮他计算在整个登山过程中,发生了多少次温度突变。

**输入格式**  
- 第一行包含两个整数 N 和 D,用空格分隔,分别表示检查点数量和突变阈值。
- 接下来 N 行,每行一个整数 Ti,表示第 i 个检查点的温度。

**输出格式**  
一行,输出发生温度突变的检查点数量。

**样例输入 1**  
```
5 3
10
12
15
14
20
```

**样例输出 1**  
```
2
```

**样例输入 2**  
```
4 10
5
8
3
12
```

**样例输出 2**  
```
0
```

**提示**  
对于样例 1:
- 检查点 2: |12-10| = 2 < 3,未突变
- 检查点 3: |15-12| = 3 ≥ 3,突变
- 检查点 4: |14-15| = 1 < 3,未突变
- 检查点 5: |20-14| = 6 ≥ 3,突变
共 2 次突变。

数据范围: 2 ≤ N ≤ 100, 1 ≤ D ≤ 100, -100 ≤ Ti ≤ 100

### 2、<a href="https://fslong.iok.la/problem/AWC0047_B" target="_blank">Road Closure on a One-Way Street</a>(100分)

**题目描述**  
小明所在的小镇有一条直线道路,上面有 N 个点从左到右排列,编号为点 1 到点 N。相邻点 i 和 i+1 之间的路段用 Wi 表示是否封闭:
- Wi = 1 表示该路段封闭,无法通行
- Wi = 0 表示该路段可以自由通行

小明有 K 张施工完成许可证,可以在出发前选择最多 K 个封闭路段解除封闭。

小明从点 1 出发,可以在未封闭的路段上双向移动。请问在使用最优策略的情况下,小明从点 1 最多可以到达多少个点?

**输入格式**  
- 第一行包含两个整数 N 和 K,用空格分隔。
- 第二行包含 N-1 个整数 W1, W2, ..., W(N-1),用空格分隔。

**输出格式**  
一行,输出可以到达的点数。

**样例输入 1**  
```
7 2
0 1 0 1 1 0
```

**样例输出 1**  
```
5
```

**样例解释 1**  
最优策略:打通路段 2-3(W[2]=1)和路段 4-5(W[4]=1),可以到达点 1-5,共 5 个点。

**样例输入 2**  
```
10 0
0 0 1 0 0 0 1 0 0
```

**样例输出 2**  
```
3
```

**提示**  
使用贪心策略:优先打通靠近起点的封闭路段,这样可以用最少的许可证到达更多的点。

数据范围: 1 ≤ N ≤ 100, 0 ≤ K ≤ N-1, Wi ∈ {0, 1}

### 4、<a href="https://fslong.iok.la/problem/AWC0049_A" target="_blank">Progressive Taxation Simulation</a>(100分)

**题目描述**  
小明正在为一个国家开发税收计算系统。在这个国家,收入税额按照累进税制计算:
- 对收入的前 L 日元,应用 P1% 的税率
- 对超过 L 日元的收入部分,应用 P2% 的税率

具体计算规则:
- 当 X ≤ L 时:税额 = floor(X × P1 / 100)
- 当 X > L 时:税额 = floor((L × P1 + (X - L) × P2) / 100)

其中 floor(x) 表示向下取整。

给定 Q 个纳税人的收入 Xi,请计算每个人的税额。

**输入格式**  
- 第一行包含四个整数 Q, L, P1, P2,用空格分隔。
- 接下来 Q 行,每行一个整数 Xi,表示第 i 个纳税人的收入。

**输出格式**  
输出 Q 行,第 i 行包含第 i 个纳税人的税额。

**样例输入 1**  
```
3 100 10 23
50
100
150
```

**样例输出 1**  
```
5
10
21
```

**样例解释 1**  
- 收入 50: 50 ≤ 100,税额 = floor(50 × 10 / 100) = 5
- 收入 100: 100 ≤ 100,税额 = floor(100 × 10 / 100) = 10
- 收入 150: 150 > 100,税额 = floor((100 × 10 + 50 × 23) / 100) = floor(2150 / 100) = 21

**样例输入 2**  
```
4 200 30 10
100
200
300
1
```

**样例输出 2**  
```
30
60
70
0
```

**提示**  
注意使用整数运算,除法会自动向下取整。

数据范围: 1 ≤ Q ≤ 10^5, 0 ≤ L ≤ 10^9, 0 ≤ P1, P2 ≤ 100, 0 ≤ Xi ≤ 10^9

### 5、<a href="https://fslong.iok.la/problem/AWC0049_B" target="_blank">Cell Cultivation</a>(100分)

**题目描述**  
小明在生物实验室进行细胞培养实验。实验开始时,培养皿中有 A 个细胞。

这些细胞有一个特殊性质:当执行一次"有丝分裂促进"操作时,所有细胞同时分裂,细胞数量恰好翻倍。

由于营养限制,这种操作只能执行 1 到 K 次。小明的目标是使细胞数量恰好为 B。

请判断是否可能将细胞数量从 A 变为恰好 B。如果可能,找出所需的最少操作次数。

**输入格式**  
一行包含三个整数 A, B, K,用空格分隔,分别表示初始细胞数、目标细胞数和最大操作次数。

**输出格式**  
如果可能达到 B 个细胞,输出最少操作次数;否则输出 -1。

**样例输入 1**  
```
3 24 5
```

**样例输出 1**  
```
3
```

**样例解释 1**  
3 → 6 → 12 → 24,操作 3 次后恰好等于目标 24,且 3 ≤ 5。

**样例输入 2**  
```
7 100 10
```

**样例输出 2**  
```
-1
```

**样例解释 2**  
从 7 开始翻倍:7 → 14 → 28 → 56 → 112...,永远不会等于 100。

**样例输入 3**  
```
1000000000000 8000000000000 3
```

**样例输出 3**  
```
3
```

**提示**  
- 每次操作后细胞数量变为原来的 2 倍
- 检查 B 是否能被 A 整除,且 B/A 是否为 2 的幂次
- 注意使用 long long 类型存储大数

数据范围: 1 ≤ A ≤ 10^12, 1 ≤ B ≤ 10^12, 1 ≤ K ≤ 60
