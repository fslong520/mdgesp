## 第 5 题【递推与递归的区别】(10分)
小明在算法学院看到了两种计算阶乘的方法:

**方法一(递归):**
```cpp
int fact(int n) {
    if (n <= 1) return 1;
    return n * fact(n - 1);
}
```

**方法二(递推):**
```cpp
int fact(int n) {
    int ans = 1;
    for (int i = 2; i <= n; i++) ans *= i;
    return ans;
}
```

教授问:"这两种方法的主要区别是什么?"

- A. 两者完全相同,没有区别
- B. 递归效率更高,递推效率更低
- C. 递归用循环,递推用函数调用
- D. 递归用函数调用自身,递推用循环迭代

### 3、<a href="https://fslong.iok.la/problem/P9120" target="_blank">乘式分解</a>(100分)

**题目描述**  
小明在数学世界学习分解数字。把正整数 n 分解成若干个正整数相乘,要求后面的数≥前面的数(保证不降序),输出所有方案。

例如:把 12 分解,有以下 4 种方案:
- 2×2×3
- 2×6
- 3×4
- 12

注意:1 不参与分解,因为任何数乘1还是自身。n 本身也算一种方案。

**输入格式**  
一行,一个正整数 n。

**输出格式**  
输出所有方案,每种方案占一行,数之间用"×"连接。按字典序输出。

**样例输入 1**  
```
12
```

**样例输出 1**  
```
2×2×3
2×6
3×4
12
```

**样例输入 2**  
```
8
```

**样例输出 2**  
```
2×2×2
2×4
8
```

**提示**  
- 定义递归函数 `void dfs(int x, int start, string path)`
- x:还剩多少要分解
- start:当前因子最小从几开始(保证不降序,且从2开始)
- path:已选的因子组成的路径
- 终止条件:遍历完所有可能的因子后输出路径

数据范围: 2 ≤ n ≤ 30

### 4、<a href="https://fslong.iok.la/problem/P9121" target="_blank">密码锁</a>(100分)

**题目描述**  
小明有一个 4 位密码锁,每位可以是 1 到 n 之间的数字。密码的规则是:相邻两位的数字之差的绝对值不能超过 1。

请输出所有合法的 4 位密码,按字典序排列。

**输入格式**  
一行,一个正整数 n(1 ≤ n ≤ 5),表示每位数字的最大值。

**输出格式**  
输出所有合法的 4 位密码,每行一个。按字典序输出。

**样例输入 1**  
```
2
```

**样例输出 1**  
```
1111
1112
1121
1122
1211
1212
1221
1222
2111
2112
2121
2122
2211
2212
2221
2222
```


**提示**  
- 定义递归函数 `void dfs(int pos, int last, string path)`
- pos:当前填第几位
- last:上一位填的数字
- path:已填的数字序列
- 终止条件:pos==4 时输出
- 递归:枚举当前位可以填的数字 i,要求 |i-last| ≤ 1

数据范围: 1 ≤ n ≤ 5

### 5、<a href="https://fslong.iok.la/problem/P9122" target="_blank">选苹果</a>(100分)

**题目描述**  
小明面前有 n 个苹果,编号从 1 到 n。他想要从中选出一些苹果(可以不选,也可以全选),输出所有可能的选法。

例如:n=3 时,有以下 8 种选法:
- 空(一个都不选)
- 1
- 2
- 3
- 1 2
- 1 3
- 2 3
- 1 2 3

**输入格式**  
一行,一个正整数 n。

**输出格式**  
输出所有选法,每种选法占一行,数字之间用空格分隔。按字典序输出。

**样例输入 1**  
```
3
```

**样例输出 1**  
```

1
1 2
1 2 3
1 3
2
2 3
3
```

**样例输入 2**  
```
2
```

**样例输出 2**  
```

1
1 2
2
```

**提示**  
- 定义递归函数 `void dfs(int i, string s)`,i 表示当前考虑第几个苹果,s 表示已选的苹果
- 终止条件:i==n+1 时输出
- 递归:对于第 i 个苹果,有两种选择
  - 不选:dfs(i+1, s)
  - 选:dfs(i+1, s + to_string(i) + " ")

数据范围: 1 ≤ n ≤ 10
