### 5、<a href="https://fslong.iok.la/problem/P9186" target="_blank">数字组合（可重复）</a>(100分)
**题目描述**  
从 1,2,...,n 中选出 m 个数，每个数可以重复选，输出所有方案。

要求：方案内**从小到大**排列，方案间**字典序**输出。

**输入**  
1. n m（1 ≤ m ≤ n ≤ 20）

**输出**  
1. 每行一个方案，数字间空格隔开

**样例输入**  
```
4 2
```
**样例输出**  
```
1 1
1 2
1 3
1 4
2 2
2 3
2 4
3 3
3 4
4 4
```

**提示**  
- 基于课堂组合题变种，允许重复选数
- 递归函数 `void dfs(int k, int start)`
- k：当前选第几个数
- start：当前可以从哪个数开始选（允许重复，传i而非i+1）
- 终止条件：选够m个数

数据范围: 1 ≤ n ≤ 20, 1 ≤ m ≤ n

### 6、<a href="https://fslong.iok.la/problem/P9187" target="_blank">不含连续1的二进制串</a>(100分)
**题目描述**  
生成长度为 n 的所有二进制串，要求不包含连续的1。

**输入**  
1. n（1 ≤ n ≤ 15）

**输出**  
1. 每行一个二进制串，按字典序输出

**样例输入**  
```
3
```
**样例输出**  
```
000
001
010
100
101
```

**提示**  
- 基于课堂二进制串题变种，增加剪枝条件
- 递归函数 `void dfs(int k, int last)`
- k：当前填第几位
- last：上一位填的是0还是1（上一位是1则当前只能填0）
- 终止条件：填够n位

数据范围: 1 ≤ n ≤ 15
