## 一、选择题（每题 10 分，共 50 分）

### 第 1 题 【对称构造】(10分)
要构造 n 个不同的整数使它们的和为 0，当 n 为偶数时，正确的构造方法是？

- A. 放一个 0，其余成对放正负数
- B. 所有数都放正数
- C. 成对放置正负数，如 -1, 1, -2, 2, ...
- D. 随机生成 n 个数再调整

---

### 第 2 题 【大小交替构造】(10分)
将数组排序后按"小-大-小-大"交替放置，可以避免什么情况？

- A. 数组中出现重复元素
- B. 中间元素等于两边元素的平均值
- C. 数组元素之和为 0
- D. 数组中存在负数

---

### 第 3 题 【递归】(10分)
以下递归函数的输出是？
```cpp
int f(int n, int sum) {
    if (n == 0) return sum;
    return f(n - 1, sum + n);
}
cout << f(10, 0);
```

- A. `10`
- B. `45`
- C. `55`
- D. `5050`

---

### 第 4 题 【前缀和】(10分)
已知数组 a 的前缀和数组 s（s[0]=0, s[i]=s[i-1]+a[i-1]），求区间 [l, r]（0-indexed）的元素和，正确的公式是？

- A. `s[r] - s[l]`
- B. `s[r+1] - s[l]`
- C. `s[r] - s[l-1]`
- D. `s[r+1] - s[l+1]`

---

### 第 5 题 【快速幂】(10分)
计算 $3^{13}$，使用快速幂算法需要多少次乘法运算？

- A. `13`
- B. `6`
- C. `5`
- D. `3`
