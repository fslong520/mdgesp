## 一、选择题（每题 10 分，共 100 分）

### 第 1 题【循环结构】(10分)
以下代码段执行后，变量 sum 的值是？
```cpp
int sum = 0;
for (int i = 1; i <= 10; i++) {
    if (i % 3 == 0) continue;
    sum += i;
}
```

- A. 37
- B. 55
- C. 27
- D. 45

### 第 2 题【数组与循环】(10分)
定义 int a[6] = {5, 3, 8, 2, 9, 1}，执行以下代码后，a[2] 的值是？
```cpp
for (int i = 0; i < 5; i++) {
    if (a[i] > a[i+1]) {
        int t = a[i];
        a[i] = a[i+1];
        a[i+1] = t;
    }
}
```

- A. 3
- B. 5
- C. 8
- D. 2
