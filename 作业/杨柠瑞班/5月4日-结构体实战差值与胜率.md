### 1、<a href="https://fslong.iok.la/problem/P9197" target="_blank">蛋糕店老板</a>(100分)

**题目描述**
你开了一家蛋糕店,可以制作两种蛋糕:

| 材料 | A型蛋糕(每个) | B型蛋糕(每个) |
|------|-------------|-------------|
| 面粉 | 250g | 200g |
| 香蕉 | 2根 | 0根 |
| 糖 | 75g | 150g |
| 黄油 | 100g | 150g |
| 可可粉 | 0g | 75g |

A型蛋糕每个卖400元,B型蛋糕每个卖450元。

现在告诉你店里各种材料的库存量,请问怎么安排A型和B型蛋糕的制作数量,才能让总收入最高?输出最高收入。

**输入格式**
一行,五个整数,分别表示面粉、香蕉、糖、黄油、可可粉的库存量(单位:g或根)。

**输出格式**
一行,一个整数,表示最高收入(元)。

**样例输入**
```
1000 5 300 400 150
```

**样例输出**
```
1450
```

**提示**
用两重循环枚举A型和B型蛋糕的制作数量,检查材料是否够用,记录最大收入。枚举范围不用太大,材料总量有限。

### 2、<a href="https://fslong.iok.la/problem/P9198" target="_blank">无尽的滑滑梯</a>(100分)

**题目描述**
游乐园的滑滑梯被排成了一个环形!从最后一个滑滑梯可以回到第一个。

给定 n 个滑滑梯的高度,如果当前滑滑梯比下一个高,就可以滑下去(下坡)。

请问在这个环形排列中,最长的一段连续下坡有多长?

**输入格式**
第一行一个整数 n (3 ≤ n ≤ 100),表示滑滑梯的数量。
第二行 n 个整数,表示每个滑滑梯的高度。

**输出格式**
一行,一个整数,表示最长连续下坡的长度(滑下去的次数)。

**样例输入**
```
5
5 3 2 4 1
```

**样例输出**
```
3
```

**解释**
从高度5开始:5→3→2是连续下坡(长度2),但2→4不是下坡。
从高度4开始:4→1→5→3→2是连续下坡(长度4)。
所以最长下坡长度为3(注:实际从4开始可以连续下坡3次:4→1,1→5不行;需要从环形角度考虑)。

**提示**
环形数组,最后一个元素的下一个是第一个元素。枚举每个起点,用循环找最长下坡。

### 3、<a href="https://fslong.iok.la/problem/P9198" target="_blank">最佳预算组合</a>(100分)

**题目描述**
你有 n 件商品可以选择,每件商品有一个价格。你的预算是 k 元。

你想从中选出两件商品,使它们的价格之和不超过预算 k,并且尽可能接近 k。

请问这两件商品的价格之和最大是多少?

**输入格式**
第一行两个整数 n 和 k (2 ≤ n ≤ 1000, 1 ≤ k ≤ 100000)。
第二行 n 个整数,表示每件商品的价格。

**输出格式**
一行,一个整数,表示不超过 k 的最大两件商品价格之和。

**样例输入**
```
5 10
3 7 2 8 5
```

**样例输出**
```
10
```

**解释**
选择价格为2和8的商品,2+8=10,恰好等于预算k=10,是最接近预算的组合。

**提示**
先将价格排序,然后枚举第一件商品,在剩余商品中找合适的第二件。

### 4、<a href="https://fslong.iok.la/problem/P9200" target="_blank">建筑师的矩形花园</a>(100分)

**题目描述**
一位建筑师要用一段固定长度的围栏围出一个矩形花园。

她定义了 Rectangle 结构体来存储花园的长和宽:

```cpp
struct Rectangle {
    int length;
    int width;
};
```

请帮她计算这个矩形花园的面积和周长。

**输入格式**
一行,两个整数,分别表示矩形的长和宽 (1 ≤ length, width ≤ 100)。

**输出格式**
两行:
- 第一行:`Area: ` 加上面积
- 第二行:`Perimeter: ` 加上周长

**样例输入**
```
5 3
```

**样例输出**
```
Area: 15
Perimeter: 16
```

**提示**
面积 = 长 × 宽,周长 = 2 × (长 + 宽)。用结构体存储长和宽,编写函数分别计算面积和周长。

### 5、<a href="https://fslong.iok.la/problem/P9201" target="_blank">光荣榜</a>(100分)

期末考试结束了,教务处要打印一张光荣榜,按成绩从高到低展示所有学生的信息。

每个学生有学号、姓名和成绩三个属性。请按成绩从高到低排序输出,成绩相同的学生按输入顺序排列。

**输入格式**
第一行一个整数 n (1 ≤ n ≤ 100),表示学生人数。
接下来 n 行,每行一个整数、一个字符串和一个整数,分别表示学号、姓名和成绩。

**输出格式**
输出 n 行,按排序顺序输出学生信息,每行格式为:学号 姓名 成绩

**样例输入**
```
5
1001 Zhang 95
1002 Li 87
1003 Wang 95
1004 Zhao 78
1005 Chen 92
```

**样例输出**
```
1001 Zhang 95
1003 Wang 95
1005 Chen 92
1002 Li 87
1004 Zhao 78
```

**提示**
定义结构体存储学号、姓名和成绩,按成绩降序排序。成绩相同时保持原输入顺序(稳定排序)。
