## 第 1 题【指针基础】(10分)

快递站王叔叔想记录每个包裹的编号，他用指针写了一段代码：

```cpp
int id = 1001;
int *p = &id;
```

以下哪个操作能够**正确**输出包裹编号？

- A. `cout << p;`
- B. `cout << *p;`
- C. `cout << &p;`
- D. `cout << &id;`

```cpp
p++;
cout << *p;
```

以上代码输出的是？

- A. 10
- B. 20
- C. 30
- D. 40

## 第 3 题【指针与数组关系】(10分)

冯阿姨写了一段代码，让同学们猜输出：

```cpp
int a[3] = {5, 10, 15};
int *p = a;
cout << p[2];
```

以下哪种说法正确？

- A. 编译报错，指针不能用 `[]` 访问
- B. 输出 15，`p[2]` 等价于 `*(p+2)`，即 `a[2]`
- C. 输出 5，`p[2]` 等价于 `p` 指向的值加 2
- D. 运行时错误，指针越界

## 第 4 题【const 与指针】(10分)

同桌用两种方式记录今天的温度：

```cpp
int temp = 28;
const int *p1 = &temp;
int * const p2 = &temp;
```

以下哪种操作**合法**？

- A. `*p1 = 30;` —— 通过 `p1` 修改温度
- B. `p2 = &temp;` 之后再 `*p2 = 30;` —— 让 `p2` 重新指向再修改
- C. `p1 = &temp;` 之后 `*p1` 能读到 28
- D. `p2 = nullptr;` —— 让 `p2` 指向空

## 第 10 题【模拟法】(10分)

- A. 只看最高楼层：8 - 1 = 7

### 第 16 题（顺序结构）

冯阿姨买了 3 斤苹果和 5 斤香蕉，苹果每斤 a 元，香蕉每斤 b 元。

**题目描述**
输入苹果单价 a 和香蕉单价 b，计算冯阿姨一共需要付多少钱。

**输入格式**
一行两个正整数 a 和 b（1 ≤ a, b ≤ 100）。

**输出格式**
一行一个整数，表示总金额（元）。

**样例输入**
```
4 3
```

**样例输出**
```
27
```

**提示**
总价 = 3 × a + 5 × b，只用算术运算和输入输出即可。

### 第 18 题（循环结构）

包子铺第一天卖了 a 个包子，之后每天比前一天多卖 b 个。

**题目描述**
输入 a、b、n，输出第 n 天销量和前 n 天总销量。

**输入格式**
一行三个正整数 a、b、n（1 ≤ a, b, n ≤ 100）。

**输出格式**
一行两个整数：第 n 天销量 和 前 n 天总销量。

**样例输入**
```
3 2 5
```

**样例输出**
```
11 35
```

**提示**
第 n 天 = a+(n-1)×b，总和 = n×(首项+末项)÷2

### 第 19 题（指针 + 数组）

课代表整理了全班 n 个同学的数学成绩，用指针找最高分和最低分。

**题目描述**
输入 n 个成绩，用指针遍历数组找出最高分和最低分。必须用指针访问元素。

**输入格式**
第一行 n（2 ≤ n ≤ 100）。
第二行 n 个整数（0 ≤ 成绩 ≤ 100）。

**输出格式**
一行两个整数：最高分 和 最低分。

**样例输入**
```
6
85 92 78 95 88 90
```

**样例输出**
```
95 78
```

**提示**
`int *p = a;` 指向首元素，用 `*(p+i)` 访问，遍历比较更新最值。

### 第 20 题（指针 + 数组）

体育老师用指针把跳绳成绩从低到高排序（选择排序）。

**题目描述**
输入 n 个成绩，用指针实现选择排序后输出。必须用指针访问和交换元素。

**输入格式**
第一行 n（1 ≤ n ≤ 100）。
第二行 n 个整数（1 ≤ 成绩 ≤ 300）。

**输出格式**
一行 n 个整数，从小到大排序，空格隔开。

**样例输入**
```
5
150 175 168 180 155
```

**样例输出**
```
150 155 168 175 180
```

**提示**
`*(arr+i)` 访问，`*(arr+j)` 比较，找到最小值下标后与当前位置交换。
