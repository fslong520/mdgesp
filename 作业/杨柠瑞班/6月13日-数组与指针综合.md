## 第 10 题【模拟法】(10分)

- A. 只看最高楼层：8 - 1 = 7
- B. 将每次到达的楼层加起来：5 + 2 + 8 = 15
- C. 模拟每段位移取绝对值再相加：|5-1|+|2-5|+|8-2| = 13
- D. 只看最低和最高楼层：8 - 1 = 7

### 第 16 题 [ynr01] — <a href="https://fslong.iok.la/problem/YNR01" target="_blank">🧑‍💻 在线答题</a>

王老师统计班里同学的学号，想看看每个数字（0~9）在学号中出现了多少次。

**题目描述**
输入一个正整数 n，统计 n 的每一位数字（0~9）出现的次数。

**输入格式**
一行，一个正整数 n（1 ≤ n ≤ 10^9）。

**输出格式**
一行，10 个整数，分别表示数字 0、1、2、…、9 出现的次数，用空格隔开。

**样例输入**
```
122333
```

**样例输出**
```
0 1 2 3 0 0 0 0 0 0
```

**提示**
- 想一想如何逐位取出一个整数的每一位数字
- 用一个长度为 10 的数组记录每个数字出现的次数

### 第 17 题 [ynr02] — <a href="https://fslong.iok.la/problem/YNR02" target="_blank">🧑‍💻 在线答题</a>

体育老师要对全班同学按座位顺序分组做仰卧起坐测试，每组计算平均成绩。

**题目描述**
输入 n 个学生的成绩，按顺序分成 g 组（按输入顺序依次分组，最后一组人数不足 g 人时仍需计算），输出每组的平均成绩（保留一位小数）。

**输入格式**
第一行两个正整数 n 和 g（1 ≤ n ≤ 100，1 ≤ g ≤ n）。
第二行 n 个整数，表示每个学生的成绩（0 ≤ 成绩 ≤ 100）。

**输出格式**
若干行，每行一个浮点数（保留一位小数），表示每组的平均成绩。

**样例输入**
```
8 3
75 82 90 68 95 71 88 76
```

**样例输出**
```
82.3
78.0
82.0
```

**提示**
- 每组 g 人，最后一组可能不足 g 人，计算平均时注意实际人数
- 可以用两层循环：外层遍历每组，内层累加该组成绩

### 第 18 题 [ynr03] — <a href="https://fslong.iok.la/problem/YNR03" target="_blank">🧑‍💻 在线答题</a>

班主任需要快速了解班级同学的性别分布和转学生人数，以便安排春游分组。

**题目描述**
输入 n 个同学的信息，每行包含姓名和性别（'M' 表示男生，'F' 表示女生），若姓名末尾带有 `_T` 后缀则表示转学生。统计男生、女生和转学生的人数并输出。

**输入格式**
第一行一个正整数 n（1 ≤ n ≤ 100）。
接下来 n 行，每行一个字符串和一个字符，用空格隔开，分别表示姓名和性别。

**输出格式**
三行，依次为男生人数、女生人数、转学生人数。

**样例输入**
```
5
Wang M
Li F
Zhao_T M
Chen F
Liu_T F
```

**样例输出**
```
2
3
2
```

**提示**
- 用 `string` 的 `size()` 和 `substr()` 检查姓名末尾是否为 `_T`
- 性别字符只有 'M' 或 'F'，不会出现其他值

### 第 19 题 [ynr04]（数组题）— <a href="https://fslong.iok.la/problem/YNR04" target="_blank">🧑‍💻 在线答题</a>

学校举办"最受欢迎歌手"投票活动，每位同学只能投一票，候选人编号从 1 开始。请统计每位候选人的得票数，并找出得票最多的获胜者。

**题目描述**
输入投票人数 m 和 m 张选票（每张选票是一个候选人编号），统计每个候选人的得票数，输出得票最多的候选人编号。若最高票数相同，输出编号最小的候选人。

**输入格式**
第一行一个正整数 m（1 ≤ m ≤ 1000）。
第二行 m 个正整数，表示每张选票的候选人编号（1 ≤ 编号 ≤ 100）。

**输出格式**
一行，一个整数，表示获胜者的候选人编号。

**样例输入**
```
8
3 1 3 2 1 3 2 3
```

**样例输出**
```
3
```

**解释**
1 号得 2 票，2 号得 2 票，3 号得 4 票。3 号票数最多，获胜。

**提示**
- 用数组记录每位候选人的得票数，候选人编号作为数组下标
- 统计完票数后遍历数组，找出得票最多的候选人

### 第 20 题 [ynr05] — <a href="https://fslong.iok.la/problem/YNR05" target="_blank">🧑‍💻 在线答题</a>

图书管理员需要在有序书架上快速找到某本书的位置。

**题目描述**
书架上有 n 本书按编号从小到大排列（不重复）。输入一个目标编号，如果找到则输出它的位置（从 0 开始），找不到输出 -1。你能帮图书管理员设计一个高效的查找方法吗？

**输入格式**
第一行两个正整数 n 和 target（1 ≤ n ≤ 1000，1 ≤ target ≤ 10000）。
第二行 n 个升序排列的正整数。

**输出格式**
一行，一个整数，表示目标值的下标或 -1。

**样例输入**
```
6 5
1 2 3 4 5 6
```

**样例输出**
```
4
```

**样例输入 2**
```
5 7
1 3 5 7 9
```

**样例输出 2**
```
3
```

**样例输入 3**
```
4 6
1 3 5 7
```

**样例输出 3**
```
-1
```

**提示**
- 每次取中间位置与目标编号比较，根据大小缩小查找范围
- 可以试试用指针来标记查找区间的起始和结束位置
