## 第 2 题【成员访问】(10分)
已知结构体定义：
```cpp
struct Hero 
{
    string name;
    int atk;
    int hp;
};
Hero h;
```
要设置 h 的攻击力为 100，正确的写法是？

- A. `h[atk] = 100;`
- B. `h:atk = 100;`
- C. `h.atk = 100;`
- D. `h::atk = 100;`

- D. `hs(2).hp`
