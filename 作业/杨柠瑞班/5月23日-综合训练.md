### 1、<a href="https://fslong.iok.la/problem/P9346" target="_blank">分苹果</a>（100 分）

**题目描述**  
妈妈买了一袋苹果要分给家里的人。如果每人分 3 个，还剩几个？如果每人分 4 个，还差几个？

**输入**  
一行，一个整数 n（1 ≤ n ≤ 1000）。

**输出**  
一行，两个整数，中间用空格分隔：每人分 3 个剩余的数量、每人分 4 个还差的数量。

**样例输入**
```
10
```
每人分 3 个：10 ÷ 3 = 3 余 1 → 剩 1 个  
每人分 4 个：10 ÷ 4 = 2 余 2 → 还差 4−2=2 个

**样例输出**
```
1 2
```

**知识点**：算术运算、取余

### 2、<a href="https://fslong.iok.la/problem/P9347" target="_blank">蜗牛爬井</a>（100 分）

**题目描述**  
一只蜗牛掉进了 h 米深的枯井里。它白天向上爬 a 米，晚上向下滑 b 米（a > b）。问第几天能爬出井口？

**输入**  
一行，三个正整数 h、a、b（1 ≤ b < a ≤ h ≤ 10000）。

**输出**  
一行，一个整数，表示蜗牛爬出井口需要的天数。

**样例输入**
```
10 3 1
```
第 1 天：白天到 3 米，晚上滑到 2 米  
第 2 天：白天到 5 米，晚上滑到 4 米  
第 3 天：白天到 7 米，晚上滑到 6 米  
第 4 天：白天到 9 米，晚上滑到 8 米  
第 5 天：白天到 11 米 → 爬出！

**样例输出**
```
5
```

**知识点**：循环、模拟

### 4、<a href="https://fslong.iok.la/problem/P9349" target="_blank">不孪生素数</a>（100 分）

**题目描述**  
数学课上老师讲孪生素数（相差 2 的一对素数），同桌突发奇想：有没有素数是没有孪生兄弟的？即它减 2 不是素数，加 2 也不是素数。输入 n，统计 2 到 n 之间有多少个这样的"不孪生素数"。

**输入**  
一行，一个整数 n（2 ≤ n ≤ 1000）。

**输出**  
一行，一个整数，表示不孪生素数的个数。

**样例输入**
```
20
```
2 到 20 之间的素数：2、3、5、7、11、13、17、19  
检查每个：
- 2：2−2=0 不是素数，2+2=4 不是素数 → 不孪生 ✓
- 3：3+2=5 是素数 → 孪生 ✗
- 5：5−2=3 是素数 → 孪生 ✗
- ……
结果只有 2 是"不孪生"的 → 共 1 个

**样例输出**
```
1
```

**知识点**：枚举、素数判断

### 5、<a href="https://fslong.iok.la/problem/P9350" target="_blank">相邻相同元素</a>（100 分）

**题目描述**  
体育课排队时，老师发现相邻两个同学如果身高相同就很"默契"。给定一个班 n 个同学的身高，请你统计有多少对相邻同学身高相同。

**输入**  
第一行，一个整数 n（2 ≤ n ≤ 500）。  
第二行，n 个正整数，表示每个同学的身高（1 ≤ 身高 ≤ 200）。

**输出**  
一行，一个整数，表示相邻身高相同的对数。

**样例输入**
```
8
165 170 170 170 168 172 172 165
```
第 2-3 位都是 170（一对），第 3-4 位都是 170（一对），第 6-7 位都是 172（一对）→ 共 3 对

**样例输出**
```
3
```

**知识点**：数组、遍历
