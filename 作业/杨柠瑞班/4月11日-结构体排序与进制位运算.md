### 4、<a href="https://fslong.iok.la/problem/P8946" target="_blank">电梯运行</a> (100分)

**题目描述**  
一栋大楼只有一部电梯，初始在第 0 层。电梯上升一层需要 6 秒，下降一层需要 4 秒，每到达一个目标楼层会停留 5 秒开门。

现在给你一个乘坐顺序列表，请计算电梯运行完所有请求总共需要多少秒。

**输入格式**  
第一行一个整数 n (1 ≤ n ≤ 100)，表示乘坐次数。  
第二行 n 个整数，表示电梯依次要到达的楼层（每层楼 ≤ 100）。

**输出格式**  
一行，一个整数，表示总耗时（秒）。

**样例输入**  
```
3
2 5 3
```

**样例输出**  
```
53
```

**提示**  
0→2：上升2层(12秒)+停留(5秒)=17秒  
2→5：上升3层(18秒)+停留(5秒)=23秒  
5→3：下降2层(8秒)+停留(5秒)=13秒  
总计：17+23+13=53秒

### 5、<a href="https://fslong.iok.la/problem/P8947" target="_blank">外星文明密码</a> (100分)

**题目描述**  
科学家接收到了一段来自外星文明的信号，这段信号是一个十进制数字。经过研究发现，外星人使用二进制来传递信息的“能量密度”——也就是二进制表示中1所占的比例。

能量密度的计算方法是：将数字转换为二进制后，统计其中1的个数，然后除以二进制的总位数。

例如：13 的二进制是 1101，有 4 位，其中 3 位是 1，能量密度 = 3/4 = 0.75

现在给你一个十进制数字，请计算它的能量密度（保留2位小数）。

**输入格式**  
一行，一个正整数 n (1 ≤ n ≤ 1000)。

**输出格式**  
一行，一个实数，表示能量密度（保留2位小数）。

**样例输入**  
```
13
```

**样例输出**  
```
0.75
```

**样例输入 2**  
```
8
```

**样例输出 2**  
```
0.25
```

**提示**  
13的二进制是1101，4位中有3个1，3÷4=0.75。
8的二进制是1000，4位中有1个1，1÷4=0.25。

### 6、<a href="https://fslong.iok.la/problem/P8948" target="_blank">特殊的计算器</a> (100分)

**题目描述**  
小华有一个特殊的计算器，这个计算器只能进行位运算操作。现在小华想用这个计算器来计算两个数的平均值，但不允许使用加法后再除以2的方法。

他发现了一个巧妙的办法：
- 先用按位与运算 (&) 找出两个数相同的部分
- 再用按位异或运算 (^) 找出两个数不同的部分，然后右移1位（相当于除以2）
- 最后将两部分相加就是平均值

公式：(a & b) + ((a ^ b) >> 1)

现在给你两个整数，请用位运算的方法计算它们的平均值（向下取整）。

**输入格式**  
一行，两个整数 a 和 b (0 ≤ a, b ≤ 1000)。

**输出格式**  
一行，一个整数，表示平均值。

**样例输入**  
```
10 20
```

**样例输出**  
```
15
```

**样例输入 2**  
```
7 8
```

**样例输出 2**  
```
7
```

**提示**  
10和20的平均值是15。
7和8的平均值是7.5，向下取整是7。

位运算过程（以10和20为例）：
- 10的二进制：01010
- 20的二进制：10100
- 10 & 20 = 00000 = 0
- 10 ^ 20 = 11110 = 30
- 30 >> 1 = 15
- 0 + 15 = 15
