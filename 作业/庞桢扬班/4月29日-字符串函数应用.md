# 一、选择题（每题 10 分，共 100 分）

## 第 1 题【查找神器】(10分)
小明在数据世界的寻宝图中发现了一个神秘的查找工具 `find()`。

```cpp
string s = "ABCDEFG";
cout << s.find("C");
```

寻宝图的显示屏会显示什么数字？

- A. 1
- B. 3
- C. 2
- D. 4

### 1、<a href="https://fslong.iok.la/problem/P9164" target="_blank">查找子串位置</a>(100分)

**题目描述**  
小明在数据世界的搜索中心工作。他需要找出一个子串在原字符串中第一次出现的位置。

**输入格式**  
第一行：一个字符串（不含空格）  
第二行：要查找的子串（不含空格）

**输出格式**  
一个整数，表示子串第一次出现的位置（从 0 开始）。如果没找到，输出 -1。

**样例输入**  
```
HelloWorld
World
```

**样例输出**  
```
5
```

**提示**  
使用 `s.find(sub)` 函数，如果找不到会返回 `string::npos`，需要转换为 -1。

### 6、<a href="https://fslong.iok.la/problem/p9168" target="_blank">空格大扫除</a>(100分)

**题目描述**  
小明在数据世界的文件管理局实习。这里的文件名不允许有空格，
需要用连字符 `_`（下划线）代替空格。

> *"下划线就像一座桥，连接本该分开的单词。在计算机的世界里，文件名、变量名都离不开它。"*

小明需要编写一个程序：找到第一个空格的位置，
将所有空格替换为下划线，并统计替换了多少个空格。

**输入格式**  
一行字符串，包含至少一个空格。

**输出格式**  
共三行：
第一行：第一个空格的位置（从 0 开始）
第二行：替换后的字符串
第三行：替换的空格数量

**样例输入**  
```
Hello World
```

**样例输出**  
```
5
Hello_World
1
```

**样例输入 2**  
```
I Love CSP
```

**样例输出 2**  
```
1
I_Love_CSP
2
```

**提示**  
用 `find(" ")` 找第一个空格，用循环或 `replace()` 替换所有空格，并计数。
