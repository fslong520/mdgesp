# 一、选择题（每题 10 分，共 100 分）

## 第 1 题【ASCII码计算】(10分)
小明来到了数据世界的字符档案馆。管理员告诉他：

> *"每个字符都有一个独特的数字编码，叫做ASCII码。掌握了这个编码表，你就能在字符和数字之间自由转换。"*

已知字符 'A' 的 ASCII 码为 65，字符 'a' 的 ASCII 码为 97。那么字符 'D' 和字符 'd' 的 ASCII 码相差多少？

- A. 28
- B. 30
- C. 32
- D. 36

---

## 第 2 题【string与char[]区别】(10分)
小明在数据世界的编程工坊看到两段代码：

```cpp
// 代码甲
char a[] = "hello";
char b[] = "hello";
if(a == b) cout << "相等";

// 代码乙
string a = "hello";
string b = "hello";
if(a == b) cout << "相等";
```

工坊的老师问："这两段代码，哪一段会输出'相等'？"

- A. 只有代码甲会输出
- B. 只有代码乙会输出
- C. 两段代码都会输出
- D. 两段代码都不会输出

### 4、<a href="https://fslong.iok.la/problem/P9080" target="_blank">大写转小写</a>(100分)

### 6、<a href="https://fslong.iok.la/problem/P9082" target="_blank">查找字符位置</a>(100分)

**题目描述**  
小明在数据世界的搜索中心工作。他需要找出某个字符在字符串中第一次出现的位置。

**输入格式**  
第一行：一个字符串（不含空格）  
第二行：一个字符

**输出格式**  
一个整数，表示该字符第一次出现的位置（从 0 开始）。如果没找到，输出 -1。

**样例输入**  
```
Hello
l
```

**样例输出**  
```
2
```

**样例输入 2**  
```
Hello
x
```

**样例输出 2**  
```
-1
```

**提示**  
遍历字符串，找到第一个等于目标字符的位置；或者使用 `s.find(c)`，如果找不到会返回 `string::npos`。
