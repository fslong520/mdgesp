### 1、<a href="https://fslong.iok.la/problem/P9337" target="_blank">器材室租借</a>（100 分）

**题目描述**  
体育课代表去器材室借篮球和足球。器材室有 a 个篮球、b 个足球。每个班可以借 3 个篮球和 2 个足球，所有班借的数量一样。课代表想知道最多能借给几个班，以及借完后还剩几个篮球、几个足球。

**输入**  
一行，两个正整数 a、b（1 ≤ a,b ≤ 1000），分别表示篮球数和足球数。

**输出**  
一行，三个整数，依次为：借给的班级数、剩余篮球数、剩余足球数，中间用空格分隔。

**样例输入**
```
20 15
```

**样例输出**
```
7 1 1
```

**样例解释**
20 个篮球 ÷ 3 = 6 个班……剩 2 个？不对，还要看足球：15 ÷ 2 = 7 个班……剩 1 个。篮球只能借给 6 个班（20÷3=6 余 2），足球能借给 7 个班（15÷2=7 余 1），取较小的 6？不对，正确答案是：
20 ÷ 3 = 6（篮球够 6 个班），15 ÷ 2 = 7（足球够 7 个班），两个条件都满足的班数是 min(6,7)=6……等等，再算算。20 个篮球，每班 3 个，最多 6 个班（余 2）；15 个足球，每班 2 个，最多 7 个班（余 1）。两种球都得够，所以只能借给 6 个班，剩 2 个篮球和 3 个足球——被刚才的足球余数 1 误导了！

**正确答案：**
篮球够 20÷3=6 班余 2，实际借 6 个班用 6×3=18 个篮球剩 2，6×2=12 个足球剩 3。

**提示**
- 班级数 = min(a÷3, b÷2)
- 剩余篮球 = a − 班级数 × 3，剩余足球 = b − 班级数 × 2

**知识点**：算术运算、整数除法、取最小值

### 2、<a href="https://fslong.iok.la/problem/P9338" target="_blank">折纸挑战</a>（100 分）

**题目描述**  
午休时同桌跟你打赌：一张纸厚度只有 0.1 毫米，但每对折一次厚度就翻倍。他对折了若干次之后发现厚度超过了 h 毫米，得意地说自己成功了。你知道他至少对折了多少次吗？

**输入**  
一行，一个整数 h（1 ≤ h ≤ 1000000），表示目标厚度（毫米）。

**输出**  
一行，一个整数，表示至少对折的次数。

**样例输入**
```
10
```
折法：0.1→0.2→0.4→0.8→1.6→3.2→6.4→12.8，第 7 次后超过 10mm。

**样例输出**
```
7
```

**样例输入 2**
```
1
```
折法：0.1→0.2→0.4→0.8→1.6，第 4 次后 0.8＜1，第 5 次后 1.6≥1。

**样例输出 2**
```
5
```

**提示**
- 用循环模拟对折过程，厚度反复乘以 2，直到超过 h
- 注意厚度是浮点数，用 `double` 存储
- 答案不会超过 31（因为 2³⁰ 约 10 亿倍，远大于题目上限）

**知识点**：循环、浮点数运算、累乘

### 3、<a href="https://fslong.iok.la/problem/P9339" target="_blank">快递运费</a>（100 分）

**题目描述**  
班长帮全班代购参考书，快递公司按重量收费：
- 1 kg 以内（含 1 kg）：8 元
- 1 kg 以上到 3 kg（含 3 kg）：12 元
- 3 kg 以上到 5 kg（含 5 kg）：18 元
- 5 kg 以上：每 kg 加收 5 元，不足 1 kg 按 1 kg 算（比如 6.3 kg，超出的 1.3 kg 按 2 kg 算，加收 10 元）

**输入**  
一行，一个浮点数 w（0.1 ≤ w ≤ 100），表示包裹重量（kg）。

**输出**  
一行，一个整数，表示运费（元）。

**样例输入**
```
2.5
```

**样例输出**
```
12
```

**样例输入 2**
```
6.3
```

**样例输出 2**
```
28
```
5 kg 以内 18 元，超了 1.3 kg 按 2 kg 算，加 2×5=10 元，共 28 元。

**提示**
- 用 if-else if 判断重量区间
- 超过 5 kg 的部分用 `ceil` 函数向上取整：`int extra = ceil(w - 5);`

**知识点**：分支结构、浮点数比较、向上取整

### 5、<a href="https://fslong.iok.la/problem/P9341" target="_blank">相邻数最小差</a>（100 分）

**题目描述**  
食堂阿姨记录了一周每天的用餐人数，发现如果按人数从小到大排好，相邻两天的人数差最小的那天，就是食堂最"舒服"的日子——不用那么累。请你帮忙找出这个最小差值。

**输入**  
第一行，一个整数 N（2 ≤ N ≤ 500）。  
第二行，N 个正整数，每个数不超过 10000。

**输出**  
一行，一个整数，表示排序后相邻两数之差的最小值。

**样例输入**
```
6
320 150 480 200 510 390
```

**样例输出**
```
20
```
排序后：150 200 320 390 480 510  
相邻差：50、120、70、90、30，最小为 30（480−450？不不不）。  
重算：150→200=50，200→320=120，320→390=70，390→480=90，480→510=30  
最小差为 30。

**提示**
- 先排序，再遍历找相邻最小差
- 核心三步：**输入** → **排序** → **遍历求最小差并输出**

**知识点**：数组、冒泡排序、遍历
