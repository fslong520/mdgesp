# 三、编程题(每题 100 分,共 500 分)


### 1、<a href="https://fslong.iok.la/problem/P9050" target="_blank">【勇者的力量】(100分)</a>

**题目描述**  
小明在勇者公会接受了力量试炼。试炼的内容是:举起一块巨石,巨石有一个基础重量,但小明可以服用力量药水来提升自己的举重能力。

现在告诉你小明的基础力量、力量药水提供的额外力量、以及巨石的重量,请判断小明能否举起巨石。

如果能举起,输出"YES",否则输出"NO"。

**输入格式**  
一行,三个整数,分别表示小明的基础力量、药水的额外力量、巨石的重量(都在 1 到 1000 之间)。

**输出格式**  
一行,"YES" 或 "NO"。

**样例输入**  
```
50 30 70
```

**样例输出**  
```
YES
```

**样例输入 2**  
```
50 10 70
```

**样例输出 2**  
```
NO
```

**提示**  
总力量 = 基础力量 + 药水力量,如果总力量 ≥ 巨石重量,就能举起。

### 2、<a href="https://fslong.iok.la/problem/P9051" target="_blank">【勇者的评级】</a>(100分)

**题目描述**  
小明参加了 3 项试炼,每项试炼都有一个成绩。勇者公会会根据平均分来为小明评级,规则如下:
- 平均分 ≥ 90 分:评级为 S
- 平均分 ≥ 80 分:评级为 A
- 平均分 ≥ 70 分:评级为 B
- 平均分 ≥ 60 分:评级为 C
- 平均分 < 60 分:评级为 D

现在告诉你小明 3 项试炼的成绩,请计算平均分(保留整数部分,直接舍去小数),然后输出对应的评级。

**输入格式**  
一行,3 个整数,分别表示 3 项试炼的成绩(每个成绩在 0 到 100 之间)。

**输出格式**  
一行,一个字母,表示评级(S/A/B/C/D)。

**样例输入**  
```
85 90 78
```

**样例输出**  
```
B
```

**样例输入 2**  
```
95 92 88
```

**样例输出 2**  
```
A
```

**提示**  
先计算总分,然后除以 3 得到平均分(整数除法会自动舍去小数)。再用 if-else if 判断评级。

### 3、<a href="https://fslong.iok.la/problem/P9052" target="_blank">【勇者的训练】(100分)</a>

**题目描述**  
小明需要在训练场进行 n 天的训练。每天训练后会获得经验值,但有一个特殊规则:
- 如果当天的天数是**奇数**(1, 3, 5...),获得 5 点经验
- 如果当天的天数是**偶数**(2, 4, 6...),获得 3 点经验

请计算小明在 n 天后总共获得了多少经验值。

**输入格式**  
一行,一个整数 n (1 ≤ n ≤ 100),表示训练天数。

**输出格式**  
一行,一个整数,表示总经验值。

**样例输入**  
```
5
```

**样例输出**  
```
19
```

**样例输入 2**  
```
4
```

**样例输出 2**  
```
16
```

**提示**  
第 1 天:5, 第 2 天:3, 第 3 天:5, 第 4 天:3, 第 5 天:5,总计 5+3+5+3+5=19。
使用循环遍历每一天,用 if 判断奇偶数,奇数加 5,偶数加 3。

### 4、<a href="https://fslong.iok.la/problem/P9053" target="_blank">【勇者的最高成绩】(100分)</a>

**题目描述**  
小明参加了 5 项试炼,每项试炼都有一个成绩。现在请你帮他找出这 5 项试炼中的最高成绩。

**输入格式**  
一行,5 个整数,表示 5 项试炼的成绩(每个成绩在 0 到 100 之间)。

**输出格式**  
一行,一个整数,表示最高成绩。

**样例输入**  
```
75 82 90 68 85
```

**样例输出**  
```
90
```

**提示**  
用一个变量记录最大值,初始设为第 1 个成绩,然后依次和后面的成绩比较,如果发现更大的就更新。

### 5、<a href="https://fslong.iok.la/problem/P9054" target="_blank">【勇者的最低消耗】(100分)</a>

**题目描述**  
小明在冒险中遇到了 5 个怪物,打败每个怪物需要消耗不同的魔力值。现在请你帮他找出消耗魔力值最少的那个怪物,输出最少消耗多少魔力。

**输入格式**  
一行,5 个整数,表示打败 5 个怪物分别需要的魔力值(每个值在 1 到 1000 之间)。

**输出格式**  
一行,一个整数,表示最少的魔力消耗。

**样例输入**  
```
120 85 200 150 95
```

**样例输出**  
```
85
```

**提示**  
和找最大值类似,用一个变量记录最小值,初始设为第 1 个值,然后依次比较,发现更小的就更新。
