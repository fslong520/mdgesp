# 勇者的背包

小明的冒险还在继续，这次他来到了一个神秘的仓库，里面摆满了各种道具。仓库的墙上写着一行字：

> *"真正的勇者不仅要会战斗，还要学会整理装备。将这些道具放入你的背包，用数组的力量管理它们吧！"*

小明打开了自己的冒险背包，准备学习如何更好地管理装备……

}

# 三、编程题（每题 100 分，共 600 分）



### 1、<a href="https://fslong.iok.la/problem/P8949" target="_blank">装备升级</a>(100分)

**题目描述**  
小明的武器有一个初始攻击力。每次升级可以提升一定的攻击力，提升的幅度是固定的。现在告诉你武器的初始攻击力、每次升级提升的攻击力、以及升级次数，请计算升级后的最终攻击力。

**输入格式**  
一行，三个整数，分别表示初始攻击力、每次提升的攻击力、升级次数（都在1到100之间）。

**输出格式**  
一行，一个整数，表示最终攻击力。

**样例输入**  
```
50 10 3
```

**样例输出**  
```
80
```

**提示**  
初始50，升级3次，每次+10，最终 = 50 + 10×3 = 80

### 2、<a href="https://fslong.iok.la/problem/P8950" target="_blank">勇者属性计算</a>(100分)

**题目描述**  
勇者小明有四个基础属性：力量、敏捷、智力和体力。现在告诉你这四个属性的数值，请计算勇者的总属性值（四个属性之和）以及平均属性值（保疙1位小数）。

**输入格式**  
一行，四个整数，分别表示力量、敏捷、智力和体力（每个属性值在1到100之间）。

**输出格式**  
两行：
第一行输出总属性值。
第二行输出平均属性值（保疙1位小数）。

**样例输入**  
```
15 20 18 22
```

**样例输出**  
```
75
18.8
```

### 3、<a href="https://fslong.iok.la/problem/P8951" target="_blank">勇者评分</a>(100分)

**题目描述**  
系统根据勇者的战斗表现为其评分。评分规则如下：
- 如果战斗次数小于5次，评分为D
- 如果战斗次数在5到9次之间，评分为C
- 如果战斗次数在10到19次之间，评分为B
- 如果战斗次数在20到49次之间，评分为A
- 如果战斗次数达到50次及以上，评分为S

现在告诉你勇者的战斗次数，请输出对应的评分。

**输入格式**  
一行，一个整数 n (0 ≤ n ≤ 100)，表示战斗次数。

**输出格式**  
一行，一个字母，表示评分等级（S/A/B/C/D）。

**样例输入**  
```
15
```

**样例输出**  
```
B
```

**样例输入 2**  
```
55
```

**样例输出 2**  
```
S
```
