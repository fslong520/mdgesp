### 第 1 题 植树问题 [zkh06]（100 分）— <a href="https://fslong.iok.la/problem/ZKH06" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

学校门口有一条马路，园林工人要在路的一边种树。树与树之间的间隔相同。

已知马路总长 L 米，每隔 d 米种一棵树（L 是 d 的整数倍），计算三种情况下各需要多少棵树：
1. 两端都种树
2. 两端都不种树
3. 一端种树、另一端不种

**输入格式**

一行，两个正整数 L 和 d（1 ≤ d ≤ L ≤ 1000，L % d = 0）。

**输出格式**

一行，三个整数，分别表示"两端都种""两端不种""一端种"的棵数，中间用空格隔开。

**样例输入 1**

```
50 5
```

**样例输出 1**

```
11 9 10
```

**样例输入 2**

```
20 4
```

**样例输出 2**

```
6 4 5
```

**提示**

树的数量 = 段数 + 两端是否种树的调整。段数 = L ÷ d。两端都种 = 段数 + 1，两端不种 = 段数 − 1，一端种 = 段数。

### 第 2 题 年龄问题 [zkh07]（100 分）— <a href="https://fslong.iok.la/problem/ZKH07" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

小华今年 a 岁，爸爸今年 b 岁（b > a）。爸爸说："等你长大后，我的年龄会变成你的 2 倍。"

问经过多少年后，爸爸的年龄正好是小华年龄的 2 倍？如果永远不可能，输出 -1。

**输入格式**

一行，两个正整数 a 和 b（1 ≤ a < b ≤ 200），分别表示小华和爸爸今年的年龄。

**输出格式**

一行，一个整数。如果存在这样的年份，输出年数（正整数）；如果不可能，输出 -1。

**样例输入 1**

```
8 36
```

**样例输出 1**

```
20
```

**样例输入 2**

```
30 40
```

**样例输出 2**

```
-1
```

**样例解释**

- 样例 1：设 x 年后，36 + x = 2 × (8 + x)。解得 x = 20。爸爸 56 岁，小华 28 岁。
- 样例 2：设 x 年后，40 + x = 2 × (30 + x)。解得 x = −20。无法回到过去，输出 -1。

**提示**

列方程：b + x = 2 × (a + x)，解出 x = b − 2a。如果 x < 0，说明不可能。注意 x 必须是整数（题目保证输入数据使 x 为整数）。

### 第 3 题 彩灯颜色 [zkh08]（100 分）— <a href="https://fslong.iok.la/problem/ZKH08" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

国庆节到了，学校在操场上挂了一排彩灯。彩灯按照"红、黄、蓝、绿"四种颜色循环排列。也就是说：
- 第 1 盏：红（Red）
- 第 2 盏：黄（Yellow）
- 第 3 盏：蓝（Blue）
- 第 4 盏：绿（Green）
- 第 5 盏：红（Red）
- 第 6 盏：黄（Yellow）
- ……

输入一个正整数 n，输出第 1 盏到第 n 盏彩灯的颜色（用英文首字母 R/Y/B/G 表示）。

**输入格式**

一行，一个正整数 n（1 ≤ n ≤ 100）。

**输出格式**

一行，n 个字符，表示第 1 到第 n 盏灯的颜色，中间不要空格。

**样例输入 1**

```
7
```

**样例输出 1**

```
RYBGRYB
```

**样例输入 2**

```
10
```

**样例输出 2**

```
RYBGRYBGRY
```

**提示**

周期是 4。第 i 盏灯的颜色取决于 i % 4 的结果：1→R, 2→Y, 3→B, 0→G。用 if-else 判断 i % 4 的值，分别输出对应的字母即可。

### 第 4 题 分数统计 [zkh09]（100 分）— <a href="https://fslong.iok.la/problem/ZKH09" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

小测验满分 10 分。老师想统计每个分数各有多少人。

输入 n 个同学的成绩（0-10 分），统计每个分数出现的人数，并从 10 分到 0 分依次输出。

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 1000）。
第二行，n 个整数，表示每位同学的成绩（0 ≤ 成绩 ≤ 10）。

**输出格式**

11 行，每行格式为 `分数: 人数`，从 10 分到 0 分。

**样例输入**

```
8
3 5 3 7 7 1 10 3
```

**样例输出**

```
10: 1
9: 0
8: 0
7: 2
6: 0
5: 1
4: 0
3: 3
2: 0
1: 1
0: 0
```

**提示**

建一个大小为 11 的计数数组 `cnt[11]`，下标 0~10 对应每个分数。读一个成绩就在对应的下标上 +1。最后从 10 循环到 0 输出。

### 第 5 题 成绩排名 [zkh10]（100 分）— <a href="https://fslong.iok.la/problem/ZKH10" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

满分 10 分的测验成绩出来了。老师想把成绩从高到低排个名次，相同分数重复输出。

使用前一道题（分数统计）的方法——先统计每个分数的人数，再从高到低输出对应次数。

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 1000）。
第二行，n 个整数，表示每位同学的成绩（0 ≤ 成绩 ≤ 10）。

**输出格式**

一行，n 个整数，表示从高到低排序后的成绩，两两之间用空格隔开。

**样例输入**

```
8
3 5 3 7 7 1 10 3
```

**样例输出**

```
10 7 7 5 3 3 3 1
```

**提示**

先建计数数组统计每个分数的人数。然后从 10 分往下到 0 分，对于每个分数，用一个内层循环输出该分数对应的次数。这就叫**计数排序**——不用两两比较，而是靠"数数"来排序。
