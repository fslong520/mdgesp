### 第 1 题 鸡兔同笼 [zkh06]（100 分）— <a href="https://fslong.iok.la/problem/ZKH06" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

《孙子算经》中记载了这样一个问题：笼子里有鸡和兔。从上面数，共有 n 个头；从下面数，共有 m 条腿。已知鸡有 2 条腿，兔有 4 条腿。

输入 n 和 m，编程计算鸡和兔各有多少只。若无解，输出 `No solution`。

**输入格式**

一行，两个整数 n（1 ≤ n ≤ 100）和 m（1 ≤ m ≤ 400）。

**输出格式**

- 若有解，输出两个整数 x y，分别表示鸡和兔的只数，中间用空格隔开
- 若无解，输出 `No solution`

**样例输入 1**

```
35 94
```

**样例输出 1**

```
23 12
```

**样例输入 2**

```
10 32
```

**样例输出 2**

```
No solution
```

**提示**

设鸡 x 只，兔 y 只，则 x + y = n，2x + 4y = m。消元得 y = (m − 2n) / 2，x = n − y。注意判断 y 是否为正整数且 x 是否为非负整数。

### 第 2 题 爬楼梯 [zkh07]（100 分）— <a href="https://fslong.iok.la/problem/ZKH07" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

小明爬楼梯，每步可以跨 1 级或 2 级台阶。例如一个 3 级的楼梯，有 3 种上法：

```
1 + 1 + 1
1 + 2
2 + 1
```

输入楼梯的级数 n（1 ≤ n ≤ 30），计算一共有多少种不同的上法。

**输入格式**

一行，一个整数 n（1 ≤ n ≤ 30）。

**输出格式**

一行，一个整数，表示不同的上法总数。

**样例输入 1**

```
3
```

**样例输出 1**

```
3
```

**样例输入 2**

```
4
```

**样例输出 2**

```
5
```

**提示**

设 f(i) 表示上到第 i 级台阶的上法总数。要到达第 i 级，可以从第 i−1 级跨 1 级上来，也可以从第 i−2 级跨 2 级上来。所以 f(i) = f(i−1) + f(i−2)。初始：f(1) = 1，f(2) = 2。用数组存储每个结果即可。

### 第 3 题 空瓶换水 [zkh08]（100 分）— <a href="https://fslong.iok.la/problem/ZKH08" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

超市搞活动：每 3 个空瓶可以换 1 瓶水。小赵一开始买了 n 瓶水，他和家人把水喝完就有了空瓶，空瓶又可以拿去换水……如此一直换下去，直到空瓶不够 3 个为止。

问小赵一共能喝到多少瓶水（包括一开始买的）？

**输入格式**

一行，一个整数 n（1 ≤ n ≤ 1000）。

**输出格式**

一行，一个整数，表示总共能喝到的水瓶数。

**样例输入 1**

```
6
```

**样例输出 1**

```
8
```

**样例输入 2**

```
7
```

**样例输出 2**

```
10
```

**样例输入 3**

```
3
```

**样例输出 3**

```
4
```

**样例解释**

- 买 6 瓶 → 喝 6 瓶 → 6 空瓶换 2 瓶 → 喝 2 瓶 → 2 空瓶不够换 → 共喝 6+2 = 8 瓶
- 买 7 瓶 → 喝 7 瓶 → 7 空瓶换 2 瓶（剩 1 空瓶）→ 喝 2 瓶 → 共 3 空瓶 → 换 1 瓶 → 喝 1 瓶 → 1 空瓶 → 共喝 7+2+1 = 10 瓶

**提示**

用变量记录当前空瓶数和总喝的水数。只要空瓶数 ≥ 3，就继续换：新换的水 = 空瓶数 ÷ 3（向下取整），剩余空瓶 = 空瓶数 % 3 + 新换的水。

### 第 4 题 成绩排名 [zkh09]（100 分）— <a href="https://fslong.iok.la/problem/ZKH09" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

期中考试结束了，老师想要统计全班同学的成绩。满分 100 分，已知全班有 n 个同学，每个同学的成绩都是 0 到 100 之间的整数。

请使用计数排序，将成绩从高到低输出。

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 1000）。
第二行，n 个整数，表示每位同学的成绩（0 ≤ 成绩 ≤ 100）。

**输出格式**

一行，n 个整数，表示从高到低排序后的成绩，两两之间用空格隔开。

**样例输入**

```
10
66 72 88 55 93 100 45 88 72 60
```

**样例输出**

```
100 93 88 88 72 72 66 60 55 45
```

**提示**

分数范围只有 0~100。建一个大小为 101 的计数数组，统计每个分数出现几次，然后从 100 分往下依次输出对应次数的分数。不要用两层循环排序，那样太慢！

### 第 5 题 年龄统计 [zkh10]（100 分）— <a href="https://fslong.iok.la/problem/ZKH10" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

幼儿园有 n 个小朋友，每个小朋友的年龄在 1 到 6 岁之间。陈老师想统计每个年龄各有多少个小朋友，并按照年龄从大到小输出结果。

请使用计数排序的思路完成统计。

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 1000）。
第二行，n 个整数，表示每个小朋友的年龄（1 ≤ 年龄 ≤ 6）。

**输出格式**

6 行，每行两个整数，分别表示年龄和该年龄的人数，从大到小排列。

**样例输入**

```
12
3 5 2 3 6 1 4 3 5 2 6 3
```

**样例输出**

```
6 2
5 2
4 1
3 4
2 2
1 1
```

**提示**

年龄范围很小（1~6），建一个大小为 7 的计数数组（下标从 1 到 6）。统计完成后，从 6 岁往下到 1 岁，依次输出年龄和对应的人数。
