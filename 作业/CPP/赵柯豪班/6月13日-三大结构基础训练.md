### 第 16 题 游乐园套票 [zkh01]（100 分）— <a href="https://fslong.iok.la/problem/ZKH01" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

周末爸爸带小明去游乐园。成人票每张 a 元，儿童票每张 b 元。游乐园推出亲子套票优惠：一大一小同时买，总价为 (a + b) × 2 − b（即儿童票减半）。买完亲子套票后妹妹也来了，需要再买一张成人票再玩一次，同样按套票规则计算。

输入成人票价 a 和儿童票价 b，先输出亲子套票（一大一小）的价格，再输出加上妹妹的成人票后的总价格。

**输入格式**

一行，两个整数 a 和 b（1 ≤ a, b ≤ 1000）。

**输出格式**

两行，第一行为亲子套票的价格，第二行为最终总价。

**样例输入 1**

```
3 5
```

**样例输出 1**

```
11
25
```

**样例输入 2**

```
10 6
```

**样例输出 2**

```
22
38
```

**提示**

先算出亲子套票的价格 `r1 = (a + b) * 2 - b`，再用 r1 作为新的"成人价"继续算妹妹的那份 `r2 = (r1 + a) * 2 - a`。分两步计算即可。

输入需要购买的笔记本数量 n（1 ≤ n ≤ 60），输出到哪家店买更便宜。如果价格相同，输出 Same。

**输出格式**

一行：
- 如果 A 店更便宜，输出 `A`

```
5
```

**样例输出 1**

```
Same
```

**样例输入 2**

```
6
```

**样例输出 2**

```
B
```

**样例输入 3**

```
1
```

**样例输出 3**

```

```

### 第 18 题 趣味运动会号码牌 [zkh03]（100 分）— <a href="https://fslong.iok.la/problem/ZKH03" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

学校举办趣味运动会，每个参赛运动员都有一个编号。王老师想统计几个指定区间内，编号为奇数的运动员有多少人，他们的编号之和是多少。

输入 T 组数据，每组输入两个正整数 a 和 b（1 ≤ a ≤ b ≤ 100），表示编号从 a 到 b 的区间。对每组数据，输出该区间内所有奇数的和。

**输入格式**

第一行，一个整数 T（1 ≤ T ≤ 10）。
接下来 T 行，每行两个整数 a 和 b（1 ≤ a ≤ b ≤ 100）。

**输出格式**

T 行，每行一个整数，表示对应区间内奇数编号之和。

**样例输入**

```
3
1 5
2 10
5 5
```

**样例输出**

```
9
24
5
```

**样例解释**

- 1到5的奇数：1+3+5=9
- 2到10的奇数：3+5+7+9=24
- 5到5的奇数：5=5

**提示**

需要处理多组数据，每组数据遍历区间中的每个数，判断是否为奇数后累加。想一想怎么判断一个数是奇数。

### 第 20 题 价格区间统计 [zkh05]（100 分）— <a href="https://fslong.iok.la/problem/ZKH05" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

王老师统计完价格后想做进一步分析，将商品按价格分为三档：

- 便宜（价格 < 50 元）
- 中等（50 元 ≤ 价格 ≤ 100 元）
- 昂贵（价格 > 100 元）

输入 n 种商品的价格，分别统计各档次的商品数量。

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 100）。
第二行，n 个正整数，表示各商品价格（1 ≤ 价格 ≤ 10000）。

**输出格式**

一行，三个整数，分别表示便宜、中等、昂贵的商品数量，中间用空格隔开。

**样例输入**

```
8
12 45 68 120 33 99 150 25
```

**样例输出**

```
4 2 2
```

**样例解释**

便宜（12,45,33,25）共 4 个，中等（68,99）共 2 个，昂贵（120,150）共 2 个。

**提示**

用三个变量分别统计三个档次的数量。每读一个价格，判断它属于哪个档次，对应的计数器加 1。
