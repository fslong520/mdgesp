### 1、<a href="https://fslong.iok.la/problem/P9123" target="_blank">【魔法药水配方】(100分)</a>

**题目描述**  
小明在炼金学院学习制作魔法药水。药水的配方很特别:
- 需要 3 种基础材料:火焰草、冰凌花、闪电叶
- 火焰草需要 a 克,冰凌花需要 b 克,闪电叶需要 c 克
- 药水的魔力值 = (火焰草 + 冰凌花) × 闪电叶 ÷ 2 (整数除法)

现在告诉你三种材料的用量,请计算这瓶药水的魔力值。

**输入格式**  
一行,三个整数 a, b, c (1 ≤ a, b, c ≤ 100),分别表示三种材料的克数。

**输出格式**  
一行,一个整数,表示药水的魔力值。

**样例输入**  
```
10 20 6
```

**样例输出**  
```
90
```

**样例输入 2**  
```
5 15 8
```

**样例输出 2**  
```
80
```

**提示**  
样例1: (10 + 20) × 6 ÷ 2 = 30 × 6 ÷ 2 = 180 ÷ 2 = 90  
注意运算顺序:先算括号里的加法,再乘法,最后除法。

### 2、<a href="https://fslong.iok.la/problem/P9124" target="_blank">【魔法商店购物】(100分)</a>

**题目描述**  
小明来到了魔法商店,想买一些道具。商店有一个优惠活动:
- 如果消费金额 ≥ 100 金币,可以享受 8 折优惠
- 如果消费金额 ≥ 50 金币且 < 100 金币,可以享受 9 折优惠  
- 如果消费金额 < 50 金币,没有折扣,按原价支付

现在告诉你小明要购买的商品原价,请计算他实际需要支付多少金币(结果保留整数部分,直接舍去小数)。

**输入格式**  
一行,一个整数 price (1 ≤ price ≤ 200),表示商品的原价。

**输出格式**  
一行,一个整数,表示实际需要支付的金币数。

**样例输入**  
```
120
```

**样例输出**  
```
96
```

**样例输入 2**  
```
80
```

**样例输出 2**  
```
72
```

**样例输入 3**  
```
30
```

**样例输出 3**  
```
30
```

**提示**  
样例1: 120 ≥ 100,打8折,120 × 8 ÷ 10 = 96  
样例2: 80 ≥ 50 且 < 100,打9折,80 × 9 ÷ 10 = 72  
样例3: 30 < 50,没有折扣,支付30  
使用 if-else if 语句判断不同的价格区间,计算对应的折扣。

-8

8

### 4、<a href="https://fslong.iok.la/problem/P9125" target="_blank">【温差计算】(100分)</a>

**题目描述**  
小明记录了一周每天的最高温度,他想知道这一周中最高温度和最低温度的差距是多少(温差 = 最高温度 - 最低温度)。

**输入格式**  
第一行,一个整数 n (1 ≤ n ≤ 100),表示天数。  
第二行,n 个整数,表示每天的最高温度(温度范围 -50 到 50)。

**输出格式**  
一行,一个整数,表示温差(最高温度 - 最低温度)。

**样例输入**  
```
7
15 20 8 25 12 18 10
```

**样例输出**  
```
17
```

**样例输入 2**  
```
5
-5 10 3 -8 7
```

**样例输出 2**  
```
18
```

**提示**  
需要同时找出最大值和最小值,然后用最大值减去最小值。可以用两个变量分别记录。
样例2:最高温度是10,最低温度是-8,温差 = 10 - (-8) = 18。

### 5、<a href="https://fslong.iok.la/problem/P9127" target="_blank">【双倍经验值】(100分)</a>

**题目描述**  
小明在游戏里打败了 n 只怪物,每只怪物会给予一定的经验值。今天是双倍经验日,每只怪物给的经验值都会变成原来的 2 倍。请计算小明在今天总共能获得多少经验值。

**输入格式**  
第一行,一个整数 n (1 ≤ n ≤ 100),表示怪物数量。  
第二行,n 个整数,表示每只怪物的基础经验值(每个经验值在 1 到 100 之间)。

**输出格式**  
一行,一个整数,表示双倍经验后的总经验值。

**样例输入**  
```
5
10 20 15 25 30
```

**样例输出**  
```
200
```

**样例输入 2**  
```
4
5 8 12 3
```

**样例输出 2**  
```
56
```

**提示**  
方法1:先把所有经验值加起来,最后乘以2。  
方法2:遍历数组时,每个元素先乘以2再累加。  
样例1: (10+20+15+25+30) × 2 = 100 × 2 = 200
