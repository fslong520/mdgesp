### 1、<a href="https://fslong.iok.la/problem/P9296" target="_blank">小小收银员</a>（100 分）

**题目描述**  
小明去文具店买笔和橡皮。请你帮收银员算账：输入笔的数量、橡皮的数量、每支笔的价格（元）、每块橡皮的价格（元），以及小明付的钱数，计算小明应付多少钱、应找零多少钱。

**输入**  
一行，五个正整数 a、b、c、d、e，分别表示笔的数量、橡皮的数量、笔的单价（角）、橡皮的单价（角）、付的钱数（元）。

说明：价格以"角"为单位输入，但输出时以"元"为单位（1 元 = 10 角）。

**输出**  
两行，第一行输出应收金额（元），第二行输出找零金额（元）。结果保留一位小数。

**样例输入**
```
3 2 25 15 20
```

**样例输出**
```
10.5
9.5
```

**样例解释**
3 支笔 × 25 角 + 2 块橡皮 × 15 角 = 75 + 30 = 105 角 = 10.5 元  
20 元 - 10.5 元 = 9.5 元

**提示**
- 先把总价算成"角"，再除以 10 转换成"元"
- 用 `double` 或 `float` 存储元数

**知识点**：算术运算、浮点数、格式化输出

### 2、<a href="https://fslong.iok.la/problem/P9297" target="_blank">数字侦探</a>（100 分）

输入一个正整数 n，判断它是否是"幸运数"：一个数既是 3 的倍数，又大于 50，就是幸运数。如果是幸运数，输出 `Lucky`，否则输出 `Not lucky`。

### 3、<a href="https://fslong.iok.la/problem/P9298" target="_blank">体温监测</a>（100 分）

**题目描述**  
社区诊所需要根据体温判断就诊者的健康状况。输入体温（摄氏度），输出对应的健康状态：
- 体温 < 35.0 → `Low`
- 35.0 ≤ 体温 ≤ 37.2 → `Normal`
- 37.3 ≤ 体温 ≤ 38.5 → `Mild Fever`
- 体温 > 38.5 → `High Fever`

**输入**  
一行，一个浮点数 t，表示体温（35.0 ≤ t ≤ 42.0）。

**输出**  
一行，对应的健康状态。

**样例输入**
```
36.5
```

**样例输出**
```
Normal
```

**样例输入 2**
```
39.2
```

**样例输出 2**
```
High Fever
```

**提示**
- 用 if-else if-else 实现多条件判断
- 注意浮点数比较可以直接用 `>=` 和 `<=`
- 注意区间的衔接：35.0 应归为 Normal，37.3 应归为 Mild Fever

**知识点**：多分支结构、浮点数比较

### 4、<a href="https://fslong.iok.la/problem/P9299" target="_blank">成绩分差</a>（100 分）

**题目描述**  
老师录入了全班同学的成绩，想知道最高分和最低分相差多少分，以便了解成绩的两极分化情况。

**输入**  
第一行一个整数 n（1 ≤ n ≤ 100），表示班级人数。  
第二行 n 个整数，表示每个人的成绩（0 ≤ 成绩 ≤ 100）。

**输出**  
一行，一个整数，表示最高分与最低分的差值。

**样例输入**
```
5
78 92 65 88 70
```

**样例输出**
```
27
```

**样例解释**
最高分 92，最低分 65，差值为 92 − 65 = 27。

**提示**
- 用数组存储成绩
- 遍历数组找出最大值和最小值
- 最大值减最小值即答案

**知识点**：一维数组、循环遍历、求最值

### 5、<a href="https://fslong.iok.la/problem/P9300" target="_blank">排序与最值</a>（100 分）

**题目描述**  
输入 n 个整数，先将它们从小到大排序，然后输出排序后的结果，再输出其中的最小值和最大值。

**输入**  
第一行一个整数 n（2 ≤ n ≤ 50）。  
第二行 n 个整数，每个数的绝对值不超过 1000。

**输出**  
两行。第一行输出排序后的 n 个数，每两个数之间用一个空格隔开。第二行输出最小值和最大值，中间用空格隔开。

**样例输入**
```
5
3 1 4 1 5
```

**样例输出**
```
1 1 3 4 5
1 5
```

**提示**
- 用**插入排序**（Insertion Sort）手动实现排序
- 插入排序思想：将待排序元素插入到已排序序列的正确位置
- 排序后第一个元素即最小值，最后一个即最大值
- 输出数组时注意最后一个数后面不要多余空格

**知识点**：数组、插入排序、遍历输出
