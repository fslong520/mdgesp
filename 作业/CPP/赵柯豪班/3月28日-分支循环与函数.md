# 一、选择题（每题 10 分，共 50 分）

## 第 1 题【分支结构】(10分)
以下代码的输出结果是？
```cpp
#include <iostream>
using namespace std;

int main()
{
    int x = 15;
    if (x > 10)
    {
        cout << "A";
    }
    else if (x > 20)
    {
        cout << "B";
    }
    else
    {
        cout << "C";
    }
    return 0;
}
```

- A. A
- B. B
- C. C
- D. AB

## 第 2 题【for循环】(10分)
以下代码执行后，变量 `s` 的值是多少？
```cpp
int s = 0;
for (int i = 1; i <= 5; i++)
{
    s += i;
}
```

- A. 10
- B. 15
- C. 20
- D. 5

## 第 3 题【while循环】(10分)
以下代码的输出结果是？
```cpp
#include <iostream>
using namespace std;

int main()
{
    int n = 10;
    while (n > 1)
    {
        n /= 2;
    }
    cout << n << endl;
    return 0;
}
```

- A. 0
- B. 1
- C. 2
- D. 5

## 第 4 题【break和continue】(10分)
以下代码的输出结果是？
```cpp
#include <iostream>
using namespace std;

int main()
{
    for (int i = 1; i <= 5; i++)
    {
        if (i == 3)
        {
            continue;
        }
        cout << i << " ";
    }
    return 0;
}
```

- A. 1 2 3 4 5
- B. 1 2 4 5
- C. 1 2
- D. 4 5
