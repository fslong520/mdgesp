### 1、<a href="https://fslong.iok.la/problem/P9391" target="_blank">奶茶第二杯半价</a>（100 分）

**题目描述**  
奶茶店促销：方案 A 是"第二杯半价"，方案 B 是"满 30 减 5"。输入要买的杯数 n 和单价 p，输出更省钱的那个金额。

方案 A：每两杯一组，每组 1.5×p 元，余 1 杯 p 元  
方案 B：总价 = n × p，每满 30 元减 5 元

**输入**  
一行，两个整数 n、p，表示杯数和单价（1 ≤ n,p ≤ 100）。

**输出**  
一个整数，最少需要付的钱。

**样例输入**
```
5 12
```

**样例输出**
```
48
```
方案 A：(12+6)×2+12=48，方案 B：60−5−5=50，取 48

**样例输入 2**
```
3 10
```

**样例输出 2**
```
25
```
方案 A：(10+5)+10=25，方案 B：30−5=25，取 25

**知识点**：算术运算、分支、min

### 5、<a href="https://fslong.iok.la/problem/P9395" target="_blank">搬砖流水线</a>（100 分）

**题目描述**  
工地上 n 块砖按重量从小到大排序，用冒泡排序（加 flag 优化）。砖头基本已经有序，只有少数几块位置不对。输出实际需要跑多少趟。

**输入**  
第一行，n（1 ≤ n ≤ 100）。  
第二行，n 个整数，表示砖头重量。

**输出**  
一个整数，表示排序趟数。

**样例输入**
```
8
1 2 3 5 4 6 7 8
```

**样例输出**
```
2
```
第 1 趟：5 和 4 交换 → 1 2 3 4 5 6 7 8（有交换）  
第 2 趟：无交换，flag 未变，break。共 2 趟。

**知识点**：冒泡排序、flag 优化
