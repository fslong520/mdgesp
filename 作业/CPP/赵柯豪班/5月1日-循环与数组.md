### 1、<a href="https://fslong.iok.la/problem/P9174" target="_blank">【技能伤害计算】(100分)</a>

**题目描述**  
小明在玩一款 RPG 游戏,他的角色释放技能时,伤害计算公式如下:  
**最终伤害 = (基础攻击力 + 武器攻击力) × 技能倍率**

例如:基础攻击力 50,武器攻击力 30,技能倍率 2,则最终伤害 = (50 + 30) × 2 = 160。

现在告诉你这三个数值,请计算最终伤害。

**输入格式**  
一行,三个整数,分别表示基础攻击力、武器攻击力、技能倍率(都在 1 到 100 之间)。

**输出格式**  
一行,一个整数,表示最终伤害。

**样例输入**  
```
50 30 2
```

**样例输出**  
```
160
```

**样例输入 2**  
```
80 20 3
```

**样例输出 2**  
```
300
```

**提示**  
直接套用公式计算,注意先算括号里的加法,再乘法。

### 2、<a href="https://fslong.iok.la/problem/P9175" target="_blank">【职业推荐】(100分)</a>

**题目描述**  
小明刚进入一款 RPG 游戏,创建角色时需要分配三点属性:力量、敏捷、智力。游戏会根据最高的属性推荐适合的职业:
- 如果**力量**最高:推荐 `Warrior`(战士)
- 如果**敏捷**最高:推荐 `Archer`(弓箭手)
- 如果**智力**最高:推荐 `Mage`(法师)
- 如果有两个或三个属性**并列最高**:推荐 `All-rounder`(全能选手)

现在告诉你小明分配的力量、敏捷、智力值,请输出游戏推荐职业。

**输入格式**  
一行,三个整数,分别表示力量、敏捷、智力(都在 1 到 100 之间)。

**输出格式**  
一行,一个字符串,表示推荐职业(`Warrior`/`Archer`/`Mage`/`All-rounder`)。

**样例输入**  
```
80 60 70
```

**样例输出**  
```
Warrior
```

**样例输入 2**  
```
50 90 50
```

**样例输出 2**  
```
Archer
```

**样例输入 3**  
```
70 70 70
```

**样例输出 3**  
```
All-rounder
```

**提示**  
先找出三个数中的最大值,然后判断有几个属性等于最大值。如果只有1个,输出对应职业;如果有2个或3个,输出 `All-rounder`。

### 3、<a href="https://fslong.iok.la/problem/P9176" target="_blank">【闯关得分查询】(100分)</a>

**题目描述**  
小明玩闯关游戏,记录了前 n 关的得分。现在他想查询:从第 l 关到第 r 关(包含 l 和 r),总共得了多少分?

**输入格式**  
第一行,一个整数 n (1 ≤ n ≤ 100),表示总关数。  
第二行,n 个整数,表示每关的得分(0 到 100 之间)。  
第三行,两个整数 l 和 r (1 ≤ l ≤ r ≤ n),表示查询的关卡区间。

**输出格式**  
一行,一个整数,表示第 l 关到第 r 关的总得分。

**样例输入**  
```
5
10 20 15 25 30
2 4
```

**样例输出**  
```
60
```

**样例输入 2**  
```
6
5 8 12 3 7 10
1 3
```

**样例输出 2**  
```
25
```

**提示**  
方法1:直接遍历从第 l-1 个元素到第 r-1 个元素累加。  
方法2(前缀和):先预处理前缀和数组 `s[i]` 表示前 i 关的总分,则区间 [l, r] 的总分 = `s[r] - s[l-1]`。

### 4、<a href="https://fslong.iok.la/problem/P9177" target="_blank">【抽卡统计】(100分)</a>

**题目描述**  
小明玩抽卡游戏,抽到了 n 张卡牌。每张卡牌有一个稀有度等级:
- 1 代表 N 卡(普通)
- 2 代表 R 卡(稀有)
- 3 代表 SR 卡(超稀有)
- 4 代表 SSR 卡(特级稀有)

请统计每种稀有度的卡牌各有多少张。

**输入格式**  
第一行,一个整数 n (1 ≤ n ≤ 100),表示抽卡次数。  
第二行,n 个整数,每张卡的稀有度等级(1 到 4 之间)。

**输出格式**  
一行,4 个整数,用空格分隔,分别表示 N 卡、R 卡、SR 卡、SSR 卡的数量。

**样例输入**  
```
8
1 2 3 1 4 2 1 3
```

**样例输出**  
```
3 2 2 1
```

**样例输入 2**  
```
5
4 4 4 4 4
```

**样例输出 2**  
```
0 0 0 5
```

**提示**  
定义数组 `cnt[5] = {0}`,遍历每张卡,`cnt[稀有度]++` 即可。最后输出 `cnt[1]` 到 `cnt[4]`。

### 5、<a href="https://fslong.iok.la/problem/P9179" target="_blank">【道具图鉴】(100分)</a>

**题目描述**  
小明在游戏中收集了 n 个道具,每个道具有一个类型编号(1 到 5 之间)。他想看看自己的道具图鉴完成度——即 1 到 5 这 5 种道具中,他收集到了几种?

例如:收集到的道具类型为 1、3、5、2、3、1、5、4,则类型 1~5 都出现过,图鉴完成度为 5。  
再如:收集到的道具类型为 2、2、2、2、2,则只出现过类型 2,图鉴完成度为 1。

**输入格式**  
第一行,一个整数 n (1 ≤ n ≤ 100),表示道具数量。  
第二行,n 个整数,表示每个道具的类型编号(1 到 5 之间)。

**输出格式**  
一行,一个整数,表示收集到了几种不同类型的道具。

**样例输入**  
```
8
1 3 5 2 3 1 5 4
```

**样例输出**  
```
5
```

**样例输入 2**  
```
5
2 2 2 2 2
```

**样例输出 2**  
```
1
```

**提示**  
定义数组 `vis[6] = {0}` 作为"桶",遍历每个道具,把 `vis[道具类型]` 标记为 1。最后统计 `vis[1]` 到 `vis[5]` 中有几个 1,就是收集到了几种。
