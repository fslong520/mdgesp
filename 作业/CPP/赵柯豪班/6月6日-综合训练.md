## 第 1 题【体育课挑人】(10分)

体育老师让全班站成一排，先挑出最矮的同学站到第一位，然后从剩下的人中再挑最矮的站第二位……如此重复，直到全班按身高从矮到高排好。

这种"每轮选一个最值放到对应位置"的排序方法，叫什么名字？

A. 插入排序 —— 把新元素插到已排好序列的正确位置

C. 选择排序 —— 每轮选一个最值，放到它该去的位置

### 第 16 题 <a href="https://fslong.iok.la/problem/P9437" target="_blank">时间换算（100 分）</a>

**题目描述**

小明的电子表显示他跑步用了 3661 秒，但他看不懂这是多久。请你帮他换算成**X小时X分钟X秒**的格式。

**输入格式**
- 一个正整数 N(1 ≤ N ≤ 100000)，表示总秒数

**输出格式**
- 三行，分别为小时、分钟、秒

**样例输入**

```
3661
```

**样例输出**

```
1
1
1
```

**提示**
- 1小时 = 3600秒，1分钟 = 60秒
- 先算小时：N ÷ 3600，再用余数算分钟和秒
- 这是纯数学公式计算，不用任何判断！

### 第 18 题 <a href="https://fslong.iok.la/problem/P9439" target="_blank">存钱罐（100 分）</a>

**题目描述**

小明想买一个 N 元的玩具。他第一天存 1 元，第二天存 2 元，第三天存 3 元……每天比前一天多存 1 元。

请问：最少需要多少天，存的钱**大于等于** N 元？

**输入格式**
- 一个正整数 N(1 ≤ N ≤ 10000)

**输出格式**
- 一个整数，表示最少需要的天数

**样例输入**

```
10
```

**样例输出**

```
4
```

**样例解释**
1+2+3+4 = 10，第 4 天刚好够，所以需要 4 天。

**样例输入2**

```
15
```

**样例输出2**

```
5
```

**样例解释2**
1+2+3+4=10 不够，1+2+3+4+5=15，刚好第5天够。

### 第 19 题 <a href="https://fslong.iok.la/problem/P9440" target="_blank">社团报名排序（100 分）</a>

**题目描述**  
学校有 n 个社团报名，每个社团有一个报名人数。老师想按报名人数从少到多排好顺序。

**输入格式**  
第一行，一个整数 n（1 ≤ n ≤ 100）。  
第二行，n 个正整数，表示各社团报名人数。

**输出格式**  
一行，n 个从小到大排列的整数，空格分隔。

**样例输入**
```
6
12 5 8 20 3 15
```

**样例输出**
```
3 5 8 12 15 20
```

**要求**  
必须用**选择排序**实现。每轮找最小值放到前面。

**提示**  
选择排序标准实现：外层循环 i 从 0 到 n-2，内层找最小下标并交换。

**要求**  
选择排序只需排 2 轮，找到最矮的两个即可，不用全部排完。

第一行，一个整数 n（2 ≤ n ≤ 100）。

**输出格式**  
一行，两个整数，从小到大排列，空格分隔。

```
6
165 172 158 180 163 170
```

**样例输出**
```
158 163
```

**提示**  
选择排序外层循环只跑 2 次，每轮找到最小值换到前面，最后输出前两个元素。
