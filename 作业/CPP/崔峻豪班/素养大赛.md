📝 第一套：基础语法与变量逻辑
场景：丝路新程（贸易计算）



题目：在模拟丝绸之路货币兑换程序中，需要存储精确到小数点后两位的汇率，最合适的变量类型是？ 

A. int

B. char

C. double

D. bool

答案：C



解析：double 为双精度浮点型，适合处理带小数的贸易计算 。

题目：执行语句 int silk_rolls = 10; silk_rolls %= 3; 后，silk_rolls 的值是？

A. 1

B. 3

C. 0

D. 3.33

答案：A

解析：% 是取余运算符，10 除以 3 余 1。



题目：在 C++11 标准下，判断商队人数 people 是否在 10 到 50 之间（含边界），正确表达式是？ 

A. 10 <= people <= 50

B. people >= 10 || people <= 50

C. people >= 10 && people <= 50

D. people => 10 and <= 50

答案：C



解析：逻辑与运算符 && 用于连接两个必须同时满足的条件 。



题目：下列哪个选项是 C++ 中合法的变量名（用于记录非遗匠人编号）？ 

A. 2025_worker

B. worker-name

C. _artisanID

D. default

答案：C



解析：变量名不能以数字开头，不能含减号，不能是保留字 。

题目：若要使用 cout 输出“文化传承”字样，需要包含哪个头文件？

A. <cstdio>

B. <iostream>

C. <string>

D. <cmath>

答案：B

B. 使用 for 循环执行 12 次迭代

题目：执行 for(int i=0; i<5; i++) { if(i==3) break; } 后，循环体总共执行了几次？

A. 2

B. 3

C. 4

D. 5

答案：B

解析：当 i 为 0, 1, 2 时执行，当 i 等于 3 时触发 break 跳出循环。



题目：在路径优化算法中，常需要表示“无穷大”，若 dist 是 int 型，通常设置的值是？

D. -1

题目：C++ 中实现“当物资不为空时持续分发”的逻辑，合适的循环是？

B. while(!inventory.empty())

C. if

D. switch

题目：在模拟“糖画熬制”温度监控时，需实现精准调控，下列哪个 C++ 表达式能正确表示“温度 t 在 150.5 到 160.0 之间”？

C. t > 150.5 || t < 160.0

C. stack

题目：在 C++ 中，bool 类型的变量可以取的值是？
