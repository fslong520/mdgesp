## 一、选择题（8题，选自历年真题题库）

### 第 1 题（2025 真题）
十进制720₁₀和八进制270₈的和用十六进制表示？
- A. 388₁₆
- B. 3DE₁₆
- C. 288₁₆
- D. 990₁₆
- **答案：A**
- **解析**：270₈=184，720+184=904 → 904/16=56余8，56/16=3余8，3/16=0余3 → 388₁₆。

### 第 2 题（2024 真题）
计算(14₈-1010₂)×D₁₆-1101₂的十进制值？
- A. 13
- B. 14
- C. 15
- D. 16
- **答案：A**
- **解析**：14₈=12，1010₂=10，D₁₆=13，1101₂=13。(12-10)×13-13=2×13-13=13。

### 第 1 题 2021 CSP-J 第二轮 T2 插入排序 ★★ [cspj2021t2]

**题目描述**
给定长度为 n 的数组 a，下标从 1 到 n。有 q 次操作：
- `1 x v`：将 a[x] 修改为 v
- `2 x`：查询 a[x] 在数组排序后的排名（稳定排序，相同值保持原相对顺序）

**输入格式**：第一行 n, q；第二行 n 个整数 a[1..n]；接下来 q 行每行一个操作
**输出格式**：每个 2 操作输出一行一个整数

**样例**
- 输入：`3 4 / 3 2 1 / 2 3 / 1 3 2 / 2 2 / 2 3` → 输出：`1 / 1 / 2`
- 输入：`3 4 / 2 1 2 / 2 2 / 1 1 3 / 1 2 1 / 2 2` → 输出：`2 / 3`

**提示**：n ≤ 8000, q ≤ 2×10^5。每次排序 O(n) 即可，注意稳定排序含义。

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, q; cin >> n >> q;
    vector<int> a(n + 1);
    for (int i = 1; i <= n; i++) cin >> a[i];
    while (q--) {
        int op; cin >> op;
        if (op == 1) {
            int x, v; cin >> x >> v;
            a[x] = v;
        } else {
            int x; cin >> x;
            vector<pair<int,int>> b;
            for (int i = 1; i <= n; i++) b.emplace_back(a[i], i);
            sort(b.begin(), b.end());
            for (int i = 0; i < n; i++)
                if (b[i].second == x) { cout << i + 1 << "\n"; break; }
        }
    }
    return 0;
}
```

### 第 2 题 NOIP2018 普及组 T2 龙虎斗 ★★ [noip2018pjt2]

**题目描述**
n 个兵营编号 1~n，第 i 个兵营有 c[i] 位士兵。以 m 为分界，龙方为 1~m-1，虎方为 m+1~n。气势计算公式：气势 = Σ c[i] × |i - m|。m 号兵营中立。

给定 s1 位神兵加到 p1 号兵营。还有 s2 位神兵，你可指定加到任意兵营（不能加 m）。求加到哪个兵营使龙虎双方气势差绝对值最小。若有多个解，输出编号最小的。

**输入格式**：第一行 n；第二行 n 个整数 c[1..n]；第三行 m, p1, s1, s2
**输出格式**：一行一个整数，最佳兵营编号

**样例**
- 输入：`6 / 2 3 2 3 2 3 / 4 6 5 2` → 输出：`2`
- 输入：`5 / 1 1 1 1 1 / 3 2 1 2` → 输出：`4`

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
int main() {
    int n; cin >> n;
    vector<ll> c(n + 1);
    for (int i = 1; i <= n; i++) cin >> c[i];
    int m, p1, s1, s2; cin >> m >> p1 >> s1 >> s2;
    c[p1] += s1;
    ll dr = 0, hu = 0;
    for (int i = 1; i < m; i++) dr += c[i] * (m - i);
    for (int i = m + 1; i <= n; i++) hu += c[i] * (i - m);
    ll best = LLONG_MAX;
    int ans = 1;
    for (int i = 1; i <= n; i++) {
        if (i == m) continue;
        ll d2 = dr, h2 = hu;
        if (i < m) d2 += s2 * (m - i);
        else h2 += s2 * (i - m);
        ll diff = abs(d2 - h2);
        if (diff < best) { best = diff; ans = i; }
    }
    cout << ans << endl;
    return 0;
}
```

### 第 3 题 2024 CSP-J 第二轮 T2 地图探险 ★★ [cspj2024t2]

**题目描述**
n 行 m 列地图，`.` 为空地、`x` 为障碍。机器人初始位置 (x0,y0)，朝向 d（0=东、1=南、2=西、3=北）。进行 k 次操作，每次：
1. 计算前进一步的新位置 (nx, ny)
2. 若新位置在地图内且为空地，则前进，朝向不变
3. 否则右转（d = (d+1)%4），位置不变
求 k 步后机器人经过的不同位置个数（含起点）。

**输入格式**：第一行 T 组数据。每组数据：第一行 n m k，第二行 x0 y0 d0，接下来 n 行地图
**输出格式**：每组数据一行整数

**样例**
- 输入：`1 / 3 4 4 / 1 1 0 / .xxx / .x.. / ...x` → 输出：3
- 输入：`1 / 5 5 14 / 1 1 0 / ..... / ..... / ..... / ..... / .....` → 输出：14

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int dx[4] = {0, 1, 0, -1};
int dy[4] = {1, 0, -1, 0};
int main() {
    int T; cin >> T;
    while (T--) {
        int n, m, k; cin >> n >> m >> k;
        int x, y, d; cin >> x >> y >> d; x--; y--;
        vector<string> g(n);
        for (int i = 0; i < n; i++) cin >> g[i];
        vector<vector<bool>> vis(n, vector<bool>(m, false));
        vis[x][y] = true;
        while (k--) {
            int nx = x + dx[d], ny = y + dy[d];
            if (nx >= 0 && nx < n && ny >= 0 && ny < m && g[nx][ny] == '.') {
                x = nx; y = ny;
            } else d = (d + 1) % 4;
            vis[x][y] = true;
        }
        int ans = 0;
        for (int i = 0; i < n; i++)
            for (int j = 0; j < m; j++)
                if (vis[i][j]) ans++;
        cout << ans << endl;
    }
    return 0;
}
```

### 第 4 题 2025 CSP-J 第二轮 T2 座位 ★★ [cspj2025t2]

**题目描述**
n 行 m 列考场，n×m 名考生成绩互不相同。按成绩从高到低蛇形分配座位：
- 第 1 列：从第 1 行到第 n 行（↓）
- 第 2 列：从第 n 行到第 1 行（↑）
- 第 3 列：从第 1 行到第 n 行（↓）
- 以此类推

给定 n, m 和所有考生成绩 a1...a_{n×m}，其中 a1 是小 R 的成绩，求小 R 的座位（列, 行）。

**输入格式**：第一行 n, m；第二行 n×m 个整数 a1...a_{n×m}
**输出格式**：一行两个整数 c, r 表示第 c 列第 r 行

**样例**
- 输入：`2 2 / 100 99 98 97` → 输出：`1 2`
- 输入：`3 3 / 100 99 98 97 96 95 94 93 92` → 输出：`2 2`

**提示**：成绩降序排序后，排名 rk，列 = (rk-1)/n + 1，行 = 列奇数 ? (rk-1)%n+1 : n-(rk-1)%n。

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n, m; cin >> n >> m;
    int tot = n * m;
    vector<pair<int,int>> a(tot);
    for (int i = 0; i < tot; i++) { cin >> a[i].first; a[i].second = i; }
    sort(a.rbegin(), a.rend());
    for (int i = 0; i < tot; i++) {
        if (a[i].second == 0) {
            int col = i / n + 1;
            int row = (col % 2) ? (i % n + 1) : (n - i % n);
            cout << col << " " << row << endl;
            return 0;
        }
    }
}
```
