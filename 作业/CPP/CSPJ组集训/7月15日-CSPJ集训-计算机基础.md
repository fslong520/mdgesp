### 第 1 题 2021 CSP-J J2 T1 分糖果 ★ [cspj2021t1]（100 分）— <a href="https://fslong.iok.la/problem/CSPJ2021T1" target="_blank">🧑‍💻 在线答题</a>

【题目描述】

红太阳幼儿园有 n 个小朋友，你是园长。现在你有一些糖果，要把这些糖果分给小朋友。

给定 L, R，你需要选择一个正整数 k（L ≤ k ≤ R），使得分给小朋友后，你手中剩下的糖果最少。

换句话说，你要最大化 k mod n。

请你求出 k mod n 的最大可能值。

【输入格式】

从标准输入读入数据。

输入一行三个正整数 n, L, R。

【输出格式】

输出到标准输出。

输出一行一个整数，表示 k mod n 的最大可能值。

【样例 1 输入】

```
7 16 23
```

【样例 1 输出】

```
6
```

【样例 1 解释】

在 [16, 23] 中选 k = 20，20 mod 7 = 6，可以证明这是最大可能值。

【数据范围】

对于所有测试数据，保证 1 ≤ n ≤ 10⁹，1 ≤ L ≤ R ≤ 10⁹。

### 第 2 题 2020 CSP-J J2 T1 优秀的拆分 ★ [cspj2020t1]（100 分）— <a href="https://fslong.iok.la/problem/CSPJ2020T1" target="_blank">🧑‍💻 在线答题</a>

【题目描述】

一般来说，一个正整数可以拆分成若干个正整数的和。

现在定义"优秀的拆分"：将一个正整数拆分成若干个**不同的 2 的正整数次幂**的和。注意 1 不是 2 的正整数次幂。

给定一个正整数 n，如果 n 有优秀的拆分，请输出其拆分方案（从大到小输出）；如果不存在优秀的拆分，输出 -1。

【输入格式】

从标准输入读入数据。

输入一行一个正整数 n。

【输出格式】

输出到标准输出。

如果存在优秀的拆分，从大到小输出拆分中的每个数，相邻两个数之间用一个空格隔开；如果不存在，输出 -1。

【样例 1 输入】

```
6
```

【样例 1 输出】

```
4 2
```

【样例 1 解释】

6 = 4 + 2，4 和 2 都是 2 的正整数次幂且互不相同。

【样例 2 输入】

```
7
```

【样例 2 输出】

```
-1
```

【样例 2 解释】

7 无法拆分成若干个不同的 2 的正整数次幂之和。

【数据范围】

对于所有测试数据，保证 1 ≤ n ≤ 10⁷。

### 第 3 题 2019 CSP-J J2 T1 数字游戏 ★ [cspj2019t1]（100 分）— <a href="https://fslong.iok.la/problem/CSPJ2019T1" target="_blank">🧑‍💻 在线答题</a>

【题目描述】

小 K 同学向小 P 同学发送了一个长度为 8 的 01 字符串来玩数字游戏，小 P 同学需要统计其中字符 '1' 的个数。

例如：字符串 `00010100` 中有 2 个 '1'，因此小 P 同学可以答出 2。

现在给定一个长度为 8 的 01 字符串，请统计其中 '1' 的个数。

【输入格式】

从标准输入读入数据。

输入一行，一个长度为 8 的 01 字符串。

【输出格式】

输出到标准输出。

输出一行一个整数，表示字符串中字符 '1' 的个数。

【样例 1 输入】

```
00010100
```

【样例 1 输出】

```
2
```

【样例 2 输入】

```
11111111
```

【样例 2 输出】

```
8
```

【数据范围】

对于所有测试数据，字符串长度为 8，仅包含字符 0 和 1。

### 第 4 题 2018 NOIP 普及组复赛 T1 标题统计 ★ [noip2018pjt1]（100 分）— <a href="https://fslong.iok.la/problem/NOIP2018PJT1" target="_blank">🧑‍💻 在线答题</a>

【题目描述】

凯凯刚写了一篇美妙的作文，请问这篇作文的标题中有多少个字符？

注意：标题中可能包含大、小写英文字母、数字字符、空格和换行符。统计标题字符数时，空格和换行符不计算在内。

【输入格式】

从标准输入读入数据。

输入文件只有一行，一个字符串 s。

【输出格式】

输出到标准输出。

输出一行一个整数，即作文标题的字符数（不含空格和换行符）。

【样例 1 输入】

```
234
```

【样例 1 输出】

```
3
```

【样例 2 输入】

```
Hello  world!
```

【样例 2 输出】

```
12
```

【样例 2 解释】

字符串 `Hello  world!` 中有两个空格不计，有效字符共 12 个。

【数据范围】

对于所有测试数据，1 ≤ |s| ≤ 5（字符串可能包含空格，|s| 表示字符串长度）。
