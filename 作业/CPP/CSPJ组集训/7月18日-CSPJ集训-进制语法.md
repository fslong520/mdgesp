## 一、进制·位运算（7题）

### 第 1 题（2023 真题）
二进制101010₂+八进制166₈=？
- A. 1101010₂
- B. 1011010₂
- C. 1110000₂
- D. A0₁₆
- **答案：D**
- **解析**：101010₂=42，166₈=118，42+118=160=A0₁₆。

### 第 2 题（2016 真题）
二进制00101100+00010101=？
- A. 00101000
- B. 01000001
- C. 01000100
- D. 00111000
- **答案：B**
- **解析**：00101100+00010101=01000001。

### 第 3 题（2016 真题）
二进制0.1相等的八进制数是？
- A. 0.8
- B. 0.4
- C. 0.2
- D. 0.1
- **答案：B**
- **解析**：二进制0.1=0.5，转八进制：0.5×8=4.0，故0.4₈。

### 第 4 题（2025 真题）
```cpp
int x=255;
cout << (x & (x-1));
```
- A. 255
- B. 254
- C. 128
- D. 0
- **答案：B**
- **解析**：x=255=11111111₂，x-1=254=11111110₂，x&(x-1)=11111110₂=254。消去最右1。

### 第 5 题（2024 真题）
0至7的4位二进制格雷码是？
- A. 0000,0001,0011,0010,0110,0111,0101,1000
- B. 0000,0001,0011,0010,0110,0111,0100,0101
- C. 0000,0001,0011,0010,0100,0101,0111,0110
- D. 0000,0001,0011,0010,0110,0111,0101,0100
- **答案：D**
- **解析**：相邻格雷码只有1位不同。D项0000→0001→0011→0010→0110→0111→0101→0100满足性质。

### 第 6 题（2019 真题）
二进制数11 1011 1001 0111和01 0110 1110 1011逻辑与的结果？
- A. 01 0010 1000 1011
- B. 01 0010 1000 0001
- C. 01 0010 1000 0011
- D. 01 0010 1001 0011
- **答案：C**
- **解析**：逐位与，全1为1否则为0，得01 0010 1000 0011。

### 第 7 题（2018 真题）
统计二进制中1个数的缺失代码是？
- A. x>>=1
- B. x&=x-1
- C. x|=x>>1
- D. x<<=1
- **答案：B**
- **解析**：x&=x-1消除最右1，循环次数即1的个数（popcount）。

### 第 13 题（2022 真题）
```cpp
int x=101, y=201;
int *p=&x, *q=&y;
p=q;
```
执行后p指向？
- A. p指向x，值为201
- B. p指向y，值为101
- C. p指向x，p指向y的地址
- D. p指向y的地址
- **答案：D**
- **解析**：p=q让p存储y的地址，即p指向y。

### 第 14 题（2025 真题）
关于C++ string类的说法正确的是？
- A. string长度创建后不能改变
- B. 可用+连接string和char
- C. length()和size()返回值可能不同
- D. string以\0结尾且计入length()
- **答案：B**
- **解析**：string长度可动态改变（A错），length()=size()（C错），\0不计入length（D错）。+可连接string与char（B对）。

### 第 15 题（2025 真题）
```cpp
void solve(int &a, int b){
    a = a + b; b = a - b; a = a - b;
}
int main(){
    int x=5, y=10;
    solve(x, y);
    cout << x << "," << y;
}
```
- A. 5,10
- B. 10,5
- C. 10,10
- D. 5,5
- **答案：C**
- **解析**：a是x的引用（同步修改），b是y的副本（不影响y）。a=15,b=5,a=10 → x=10,y=10。

### 第 16 题（2016 真题）
以下循环执行后s的值等价于？
```cpp
int s = a;
for(int i=1; i<=c; i++) s++;
```
- A. s=a+b
- B. s=a+c
- C. s=s+c
- D. s=b+c
- **答案：B**
- **解析**：循环c次每次s+1，等价于s=a+c。

### 第 17 题（2016 真题）
```cpp
int k=4, n=0;
while(n < k){
    n++;
    if(n % 3 != 0) continue;
    k--;
}
```
- A. 2,2
- B. 2,3
- C. 3,2
- D. 3,3
  - **答案：D**
  - **解析**：n=1(跳过)→n=2(跳过)→n=3(k→3)→n=4(n<3?否)退出。最终k=3,n=3。

### 第 18 题（2018 真题）
循环按下CapsLock,A,S,D,F键（CapsLock切换大小写），输出按下的字符。第81个输出什么？
- A. A
- B. S
- C. D
- D. a
- **答案：D**
- **解析**：每5次一个循环(CapsLock,A,S,D,F)，第80个为F，第81个CapsLock切换大小写输出小写a。

### 第 19 题（2016 真题）
循环按下CapsLock,A,S,D键（CapsLock切换大小写），输出按下的字符。第81个输出什么？
- A. A
- B. S
- C. D

- **答案：C**

### 第 20 题（2019 真题）

- D. s = a - c

### 第 1 题 NOIP2017 普及组 T2 图书管理员 ★★ [noip2017pjt2]

**题目描述**
图书馆有 n 本书，每本书有一个图书编码（正整数）。有 q 位读者，每位读者的需求码长度为 len，需求码为 code。找出图书编码末尾 len 位与 code 匹配的最小图书编码。若无匹配输出 -1。

**输入格式**：第一行 n, q；第二行 n 个图书编码；接下来 q 行每行 len code
**输出格式**：每行一个整数表示最小匹配编码，无匹配输出 -1

**样例**
- 输入：`4 2 / 12345 67890 12300 12000 / 2 45 / 3 123` → 输出：`12345 / 12300`
- 输入：`3 2 / 111 222 333 / 1 4 / 2 12` → 输出：`-1 / 111`

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n, q; cin >> n >> q;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    sort(a.begin(), a.end());
    while (q--) {
        int len, code; cin >> len >> code;
        int mod = 1;
        for (int i = 0; i < len; i++) mod *= 10;
        int ans = -1;
        for (int x : a)
            if (x % mod == code) { ans = x; break; }
        cout << ans << endl;
    }
    return 0;
}
```

### 第 2 题 NOIP2016 普及组 T2 回文日期 ★★ [noip2016pjt2]

**题目描述**
牛牛习惯用 8 位数字表示一个日期，格式为 yyyymmdd。给定两个日期，求这两个日期之间（含两端）有多少个日期是回文的（正读倒读一样）。

**输入格式**：两行，每行一个 8 位数字表示起始和结束日期
**输出格式**：一行一个整数

**样例**
- 输入：`20110101 / 20111231` → 输出：1
- 输入：`20000101 / 20101231` → 输出：2

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int days[13] = {0,31,28,31,30,31,30,31,31,30,31,30,31};
bool isLeap(int y) {
    return (y%4==0 && y%100!=0) || y%400==0;
}
int main() {
    int s, e; cin >> s >> e;
    int ans = 0;
    for (int y = s/10000; y <= e/10000; y++) {
        int m = (y%10)*10 + (y/10)%10;
        int d = ((y/100)%10)*10 + y/1000;
        if (m < 1 || m > 12) continue;
        int lim = days[m];
        if (m == 2 && isLeap(y)) lim = 29;
        if (d < 1 || d > lim) continue;
        int date = y*10000 + m*100 + d;
        if (date >= s && date <= e) ans++;
    }
    cout << ans << endl;
    return 0;
}
```

### 第 3 题 2020 CSP-J 第二轮 T2 直播获奖 ★★ [cspj2020t2]

**题目描述**
比赛共有 n 名选手，获奖率 w%。每公布一个成绩，就计算一次获奖分数线。获奖人数 = max(1, ⌊当前人数 × w%⌋)，分数线为第获奖人数名的成绩。

**输入格式**：第一行 n, w；第二行 n 个整数
**输出格式**：一行 n 个整数

**样例**
- 输入：`10 60 / 200 300 400 500 600 600 0 300 200 100` → 输出：`200 300 400 400 400 500 400 400 300 300`

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int cnt[601];
int main() {
    int n, w; cin >> n >> w;
    for (int i = 1; i <= n; i++) {
        int x; cin >> x; cnt[x]++;
        int win = max(1, i * w / 100);
        int sum = 0;
        for (int s = 600; s >= 0; s--) {
            sum += cnt[s];
            if (sum >= win) { cout << s << " "; break; }
        }
    }
    return 0;
}
```

### 第 4 题 2019 CSP-J 第二轮 T2 公交换乘 ★★ [cspj2019t2]

**题目描述**
小明每天乘坐公共交通上下班。有以下优惠规则：
- 坐地铁（type=0）得一张优惠券，45分钟内可免费坐一次价格不超过该地铁票价的公交（type=1）
- 坐公交（type=1）时，若有45分钟内的未使用优惠券且公交票价≤优惠券对应地铁票价，则免费；否则付钱

给定 n 条记录，求总花费。

**输入格式**：第一行 n；接下来 n 行每行 type, price, time
**输出格式**：一行一个整数

**样例**
- 输入：`6 / 0 5 0 / 1 4 10 / 1 6 20 / 0 7 30 / 1 5 40 / 1 8 50` → 输出：`16`

**参考代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
struct Ticket { int price, time; };
int main() {
    int n; cin >> n;
    queue<Ticket> q;
    int ans = 0;
    while (n--) {
        int type, price, time; cin >> type >> price >> time;
        while (!q.empty() && time - q.front().time > 45) q.pop();
        if (type == 0) {
            q.push({price, time});
            ans += price;
        } else {
            bool used = false;
            int sz = q.size();
            for (int i = 0; i < sz; i++) {
                Ticket t = q.front(); q.pop();
                if (!used && t.price >= price) { used = true; continue; }
                q.push(t);
            }
            if (!used) ans += price;
        }
    }
    cout << ans << endl;
    return 0;
}
```
