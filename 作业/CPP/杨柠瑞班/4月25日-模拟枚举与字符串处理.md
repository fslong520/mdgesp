### 3、<a href="https://fslong.iok.la/problem/P9129" target="_blank">单词首字母大写</a>(100分)

**题目描述**  
小红在编辑一篇文章,需要将每个单词的首字母变成大写。

给定一个由小写字母和空格组成的句子,将每个单词的首字母转换为大写,其他字符保持不变。

单词之间用空格分隔,首字母是指每个单词的第一个字母。

**输入格式**  
一行,一个字符串,只包含小写字母和空格,长度不超过 200。

**输出格式**  
一行,表示转换后的字符串。

**样例输入**  
```
hello world from cpp
```

**样例输出**  
```
Hello World From Cpp
```

**提示**  
遍历字符串,如果当前字符是字母,且它是第一个字符或前一个字符是空格,则转换为大写。

### 4、<a href="https://fslong.iok.la/problem/P9130" target="_blank">自动售货机模拟</a>(100分)

**题目描述**  
小明设计了一台自动售货机,规则如下:

- 售货机里有 n 种商品,每种商品有价格
- 顾客投入 m 元钱
- 售货机会优先卖出最便宜的商品(如果钱够的话)
- 买完一件后,如果还有钱,继续买剩下商品中最便宜的
- 直到钱不够买任何商品为止

给定商品价格序列和顾客投入的钱,计算顾客能买到多少件商品。

**输入格式**  
第一行两个整数 n 和 m (1 ≤ n ≤ 100, 1 ≤ m ≤ 10000),表示商品数量和投入的钱。  
第二行 n 个整数,表示每种商品的价格。

**输出格式**  
一行,一个整数,表示能买到的商品数量。

**样例输入**  
```
5 10
3 5 2 8 4
```

**样例输出**  
```
3
```

**解释**  
商品价格排序后:2, 3, 4, 5, 8  
- 买价格2的商品,剩8元
- 买价格3的商品,剩5元
- 买价格4的商品,剩1元
- 1元不够买任何商品,停止
- 共买到3件商品

**提示**  
先将商品价格排序,然后从最便宜的开始依次购买。

### 5、<a href="https://fslong.iok.la/problem/P9132" target="_blank">图书管理系统</a>(100分)

**题目描述**  
学校图书馆需要管理图书信息,并按规则排序展示。

每本图书有书号、书名、价格三个属性。排序规则:
1. 首先按价格从高到低排序
2. 如果价格相同,按书号从小到大排序
3. 如果还相同,按书名字典序从小到大排序

请你编写程序完成排序任务。

**输入格式**  
第一行一个整数 n (1 ≤ n ≤ 100),表示图书数量。  
接下来 n 行,每行包含一个整数、一个字符串和一个整数(可能带小数),分别表示书号、书名、价格。

**输出格式**  
输出 n 行,按排序顺序输出图书信息,格式为:书号 书名 价格(保留2位小数)

**样例输入**  
```
4
101 算法导论 89.50
102 C++程序设计 45.00
103 数据结构 89.50
104 计算机网络 55.00
```

**样例输出**  
```
101 算法导论 89.50
103 数据结构 89.50
104 计算机网络 55.00
102 C++程序设计 45.00
```

**提示**  
- 算法导论和数据结构价格相同(89.50),按书号排序,101在前
- 使用结构体存储图书信息,自定义比较函数
- 价格用 double 类型存储,输出时用 `fixed` 和 `setprecision(2)` 控制精度
