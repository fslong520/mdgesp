### 第 1 题 奶茶订单 ★ [P9728]（100 分）— 字符串遍历 <a href="https://fslong.iok.la/problem/P9728" target="_blank">🧑‍💻 在线答题</a>

【题目描述】

冯老师编程课表现好的同学，可以获得一杯奶茶奖励。每个同学选一种口味，点单记录是一个字符串 s，其中每种字母代表一种口味：

- `C` 原味
- `B` 波霸
- `R` 红茶
- `G` 绿茶
- `M` 抹茶

冯老师想统计每种口味各被点了多少杯，请你帮帮他。

【输入格式】

一行，一个字符串 s，长度不超过 1000，仅包含大写字母 C、B、R、G、M。

【输出格式】

一行，五个整数，分别表示口味 C、B、R、G、M 被点的杯数，用空格隔开。

【样例 1 输入】

```
CCBRMGCB
```

【样例 1 输出】

```
3 2 1 1 1
```

【样例 1 解释】

字符串中有 3 个 C、2 个 B、1 个 R、1 个 G、1 个 M。

【样例 2 输入】

```
MMMMM
```

【样例 2 输出】

```
0 0 0 0 5
```

【数据范围】

1 ≤ |s| ≤ 1000。

【提示】

用 if-else 或 switch 判断每个字符，对应计数变量加 1，最后按顺序输出五个计数。

### 第 2 题 跳蚤市场 ★ [P9729]（100 分）— 暴力枚举 <a href="https://fslong.iok.la/problem/P9729" target="_blank">🧑‍💻 在线答题</a>

【题目描述】

小区举办跳蚤市场，有 n 件物品。每件物品有价格 a[i] 和价值 b[i]。

小华攒了 m 元零花钱，他想买两件物品（不能买同一件），在不超过预算的前提下，使总价值最大。请你帮小华算一算，最多能获得多少总价值？

【输入格式】

第一行，两个整数 n 和 m（2 ≤ n ≤ 1000，1 ≤ m ≤ 1000）。

第二行，n 个正整数 a[i]，表示每件物品的价格（1 ≤ a[i] ≤ m）。

第三行，n 个正整数 b[i]，表示每件物品的价值（1 ≤ b[i] ≤ 1000）。

【输出格式】

一行，一个整数，表示能获得的最大总价值。如果买不到两件物品，输出 0。

【样例 1 输入】

```
5 20
8 10 5 15 6
100 200 50 300 80
```

【样例 1 输出】

```
300
```

【样例 1 解释】

最优方案：买第 3 件（5 元/50 价值）和第 4 件（15 元/300 价值），总价 20 ≤ 20，总价值 350。

【样例 2 输入】

```
3 5
6 7 8
10 20 30
```

【样例 2 输出】

```
0
```

【样例 2 解释】

最便宜的两件也要 6+7=13 > 5，买不了两件。

【数据范围】

2 ≤ n ≤ 1000，1 ≤ m ≤ 1000，1 ≤ a[i] ≤ m，1 ≤ b[i] ≤ 1000。

【提示】

两层循环枚举所有 i < j 的组合，判断 a[i] + a[j] ≤ m，更新最大总价值。

### 第 3 题 草莓采摘 ★★ [P9730]（100 分）— 网格枚举 <a href="https://fslong.iok.la/problem/P9730" target="_blank">🧑‍💻 在线答题</a>

【题目描述】

五一假期，冯老师带同学们去草莓采摘园。采摘园是一个 n 行 m 列的网格，每个格子里种了一定数量的草莓。

冯老师想找一块 2 行 2 列（即 2×2）的连续区域，让同学们一起采摘。请问能摘到的草莓总数最大是多少？

【输入格式】

第一行，两个整数 n 和 m（2 ≤ n, m ≤ 100）。

接下来 n 行，每行 m 个整数，表示每个格子里的草莓数量（1 ≤ 数量 ≤ 1000）。

【输出格式】

一行，一个整数，表示最大的 2×2 区域草莓总数。

【样例 1 输入】

```
2 3
1 100 1
1 1 1
```

【样例 1 输出】

```
103
```

【样例 1 解释】

所有可能的 2×2 区域：
- 左边：1 + 100 + 1 + 1 = 103
- 右边：100 + 1 + 1 + 1 = 103
最大值为 103。

【样例 2 输入】

```
3 3
1 2 3
4 5 6
7 8 9
```

【样例 2 输出】

```
28
```

【样例 2 解释】

所有 2×2 区域的和：
左上(1,1)：1+2+4+5=12
右上(1,2)：2+3+5+6=16
左下(2,1)：4+5+7+8=24
右下(2,2)：5+6+8+9=28
最大值为 28。

【数据范围】

2 ≤ n, m ≤ 100，格子中草莓数量 1 ≤ 数量 ≤ 1000。

【提示】

两层循环枚举所有 2×2 区域的左上角 (i,j)，计算 a[i][j]+a[i][j+1]+a[i+1][j]+a[i+1][j+1] 并更新最大值。

### 第 4 题 美食街评分 ★★ [P9731]（100 分）— 结构体排序 <a href="https://fslong.iok.la/problem/P9731" target="_blank">🧑‍💻 在线答题</a>
三
【题目描述】

学校美食节来了！有 n 家摊位参加了评选，同学们对每家摊位进行打分，包含两个维度：

- **口味分**：满分 100
- **性价比分**：满分 100

总分为口味分 × 0.6 + 性价比分 × 0.4（四舍五入取整数）。

现在需要按总分从高到低排名。如果总分相同，口味分高者排前面；如果总分和口味分都相同，则摊位编号小的排前面。

请你帮王老师写一个程序，输出排名结果。

【输入格式】

第一行，一个整数 n（1 ≤ n ≤ 1000）。

接下来 n 行，每行三个整数：摊位编号 id（1 ≤ id ≤ n）、口味分 t、性价比分 p（0 ≤ t, p ≤ 100）。

【输出格式】

输出 n 行，每行一个整数，表示按排名从高到低输出的摊位编号。

【样例 1 输入】

```
4
1 90 80
2 85 90
3 90 70
4 80 80
```

【样例 1 输出】

```
1
2
3
4
```

【样例 1 解释】

计算总分（口味 × 0.6 + 性价比 × 0.4，四舍五入）：
- 摊位 1：90×0.6 + 80×0.4 = 54 + 32 = 86
- 摊位 2：85×0.6 + 90×0.4 = 51 + 36 = 87
- 摊位 3：90×0.6 + 70×0.4 = 54 + 28 = 82
- 摊位 4：80×0.6 + 80×0.4 = 48 + 32 = 80

按总分排序：2(87) > 1(86) > 3(82) > 4(80)

【样例 2 输入】

```
3
2 95 80
1 95 80
3 90 90
```

【样例 2 输出】

```
1
2
3
```

【样例 2 解释】

摊位 2 和 1 总分相同（95×0.6+80×0.4=57+32=89），口味分也相同，编号小的 1 排前面。
摊位 3 总分 90×0.6+90×0.4=54+36=90，排第一。

【数据范围】

1 ≤ n ≤ 1000，1 ≤ id ≤ n，0 ≤ t, p ≤ 100。

【提示】

定义结构体 `struct Stall { int id, t, p, score; }`。用 `sort` + 自定义 `cmp` 排序：先比 score 降序，再比 t 降序，再比 id 升序。计算 score 时用 `int(0.6 * t + 0.4 * p + 0.5)` 实现四舍五入。
