### 1、<a href="https://fslong.iok.la/problem/P9396" target="_blank">租小船</a>（100 分）

**题目描述**  
学校组织春游，有 n 个同学要坐船游湖。每条船最多能坐 k 人，不能超载。请问至少需要租多少条船，才能让所有同学都上船？

**输入**  
一行，两个正整数 n 和 k（1 ≤ k ≤ n ≤ 10000），分别表示总人数和每条船最多能坐的人数。

**输出**  
一个整数，表示至少需要租的船数。

**样例输入**
```
23 5
```
**样例输出**
```
5
```
**样例解释**：23 个同学每条船坐 5 人，坐满 4 条船后还剩 3 人，这 3 人也需要 1 条船，所以一共要租 5 条船。

### 2、<a href="https://fslong.iok.la/problem/P9397" target="_blank">书架编号</a>（100 分）

**题目描述**  
学校图书馆新到了一批书，书架编号从 a 到 b。管理员说只有编号为偶数的书架才需要贴标签。她想知道，这批书架中所有偶数编号加起来是多少，以便申请对应数量的标签。

**输入**  
一行，两个正整数 a 和 b（1 ≤ a ≤ b ≤ 10000），表示书架的起始和结束编号。

**输出**  
一个整数，表示所有偶数编号之和。

**样例输入**
```
1 10
```
**样例输出**
```
30
```
**样例解释**：编号 1 到 10 中，偶数编号为 2、4、6、8、10，标签总数为 30。

### 4、<a href="https://fslong.iok.la/problem/P9399" target="_blank">单词分值</a>（100 分）

**题目描述**  
英语课上，老师发明了一种给单词打分的方法：
- 小写字母按字母表位置计正分：a=1、b=2、c=3 …… z=26
- 大写字母按 ASCII 码值计负分：A=-65、B=-66、C=-67 ……

一个单词的分值 = 每个字母的得分之和。例如 `Hello`：
- H = -72，e = 5，l = 12，l = 12，o = 15
- 总分 = -72 + 5 + 12 + 12 + 15 = -28

给定一个包含大小写字母的字符串，请计算它的分值。

**输入**  
一行，一个只包含大小写英文字母的字符串 s（长度 ≤ 1000）。

**输出**  
一个整数，表示该单词的分值。

**样例输入**
```
Hello
```
**样例输出**
```
-28
```

### 5、<a href="https://fslong.iok.la/problem/P9400" target="_blank">零花钱</a>（100 分）

**题目描述**  
小明记了他 n 天的零花钱收支（每天的金额为正整数，表示当天收到的零花钱）。他想知道，有多少个“连续日期段”的零花钱总和恰好等于 k 元。

例如，5 天的零花钱分别是 1、2、3、4、5，目标金额 k = 5：
- 第 1~2 天：1+2 = 3 ✗
- 第 1~3 天：1+2+3 = 6 ✗
- 第 2~3 天：2+3 = 5 ✓
- 第 4~4 天：4 ✗
- 第 5~5 天：5 ✓
- 还有第 1~2~3~4~5 天太多了……

请帮小明算出有多少个连续日期段的总和恰好是 k。

**输入**  
第一行两个正整数 n 和 k（1 ≤ n ≤ 1000，1 ≤ k ≤ 10000）。  
第二行 n 个正整数，表示每天的零花钱金额。

**输出**  
一个整数，表示总和恰好为 k 的连续日期段数量。

**样例输入**
```
5 5
1 2 3 2 1
```
**样例输出**
```
3
```
**样例解释**：和为 5 的连续日期段有 [2,3]、[3,2]、[1,2,3,2,1] 共 3 个。
