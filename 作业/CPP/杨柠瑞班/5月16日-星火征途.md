### 1、<a href="https://fslong.iok.la/problem/P9291" target="_blank">星火征途 · 物资统计</a>（100 分）

**故事背景**  
革命根据地需要统计各村庄的物资储备情况。每个村庄上缴一定数量的粮食和弹药，请你帮后勤部长算出总共收到了多少物资。

**题目描述**  
输入两个整数 a 和 b，分别表示粮食袋数和弹药箱数，输出它们的和与差（用大数减小数）。

**输入**  
一行，两个整数 a 和 b（1 ≤ a, b ≤ 1000）。

**输出**  
两行，第一行输出 a + b，第二行输出 a 与 b 的差的绝对值。

**样例输入**
```
30 12
```

**样例输出**
```
42
18
```

**提示**
- 用 `if` 判断哪个数大，用大数减小数
- 也可以用 `abs()` 函数

**知识点**：基本输入输出、算术运算、条件判断

### 2、<a href="https://fslong.iok.la/problem/P9292" target="_blank">星火征途 · 密码翻译</a>（100 分）

**故事背景**  
地下党使用了一种简单的字符替换密码：将每个字母替换为它在字母表中的后一位（'A'→'B'，'B'→'C'，……，'Z'→'A'）。请你编写程序自动翻译密文。

**题目描述**  
输入一个大写字母组成的字符串，将每个字母替换为它在字母表中的后一个字母（'Z' 替换为 'A'），输出替换后的结果。

**输入**  
一行，一个由大写字母组成的字符串 s（1 ≤ |s| ≤ 100）。

**输出**  
一行，替换后的字符串。

**样例输入**
```
ABC
```

**样例输出**
```
BCD
```

**样例输入 2**
```
ZZZ
```

**样例输出 2**
```
AAA
```

**提示**
- 利用 ASCII 码：'A'=65，'B'=66，……，'Z'=90
- 对每个字符 ch：若 ch == 'Z' 则输出 'A'，否则输出 ch + 1

**知识点**：ASCII 编码、字符处理、循环

### 3、<a href="https://fslong.iok.la/problem/P9293" target="_blank">星火征途 · 火力覆盖</a>（100 分）

**故事背景**  
红军炮兵需要计算炮弹的落点。已知炮弹的初速度为 v，发射角度为 45°，在不考虑空气阻力的情况下，水平射程公式为 R = v² / 10。请你计算不同速度下的射程。

**题目描述**  
输入 n 组速度值，对每组速度 v，计算水平射程 R = v × v / 10，并输出。结果保留一位小数。

**输入**  
第一行一个整数 n（1 ≤ n ≤ 10）。  
第二行 n 个整数 v₁, v₂, ..., vₙ（1 ≤ vᵢ ≤ 1000）。

**输出**  
n 行，每行一个保留一位小数的浮点数，表示对应速度的射程。

**样例输入**
```
3
10 25 30
```

**样例输出**
```
10.0
62.5
90.0
```

**提示**
- 计算 v × v 时可能超出 int 范围（1000×1000=1,000,000），用 `long long` 或 `double`
- 用 `printf("%.1f\n", R)` 或 `cout << fixed << setprecision(1) << R << endl;`

**知识点**：循环、浮点运算、格式化输出

### 4、<a href="https://fslong.iok.la/problem/P9294" target="_blank">星火征途 · 密文分析</a>（100 分）

**故事背景**  
情报员截获了一段密文，发现其中某些字母出现的频率异常高——这些字母很可能就是关键词的组成部分。请你统计一段密文中每个字母出现的次数，找出出现次数最多的字母。

**题目描述**  
输入一个只包含大写字母的字符串 s，统计每个字母（'A'~'Z'）出现的次数，输出出现次数最多的字母及其次数。如果有多个字母并列最多，按字母顺序输出最小的那个。

**输入**  
一行，一个由大写字母组成的字符串 s（1 ≤ |s| ≤ 1000）。

**输出**  
一行，先输出出现次数最多的字母，再输出它的出现次数，中间用空格隔开。

**样例输入**
```
XINGHUOZHENGTU
```

**样例输出**
```
G 2
```

**样例解释**
字符串中 G、H、N、T、U 各出现 2 次，但 G 是字母顺序最小的，故输出 G 2。

**提示**
- 定义 `int cnt[26] = {0};` 存储每个字母出现次数
- `cnt[s[i] - 'A']++` 即可统计
- 遍历 cnt 数组找出最大值对应的字母

**知识点**：一维数组、字符串遍历、统计

### 5、<a href="https://fslong.iok.la/problem/P9295" target="_blank">星火征途 · 战报统计</a>（100 分）

**故事背景**  
红军某部连续 n 天记录了每天的歼敌数量。上级需要了解哪几天的歼敌数超过了平均歼敌数，以便表彰有功部队。

**题目描述**  
输入 n 个整数表示连续 n 天的歼敌数量。先计算这 n 天的平均歼敌数（向下取整），然后输出所有超过平均值的天的歼敌数（按原顺序输出）。

**输入**  
第一行一个整数 n（1 ≤ n ≤ 100）。  
第二行 n 个整数 a₁, a₂, ..., aₙ（1 ≤ aᵢ ≤ 10000）。

**输出**  
第一行输出平均值（向下取整）。  
第二行输出超过平均值的所有数，用空格隔开。若没有超过平均值的数，则输出 `None`。

**样例输入**
```
6
12 45 23 67 34 56
```

**样例输出**
```
39
45 67 56
```

**样例解释**
平均值为 (12+45+23+67+34+56)/6 = 237/6 = 39（向下取整）。  
超过 39 的有 45、67、56。

**提示**
- 用数组存储每日歼敌数
- 先求和再计算平均值（整数除法自动向下取整）
- 遍历数组，判断每个数是否大于平均值

**知识点**：一维数组、遍历、平均值计算、条件输出
