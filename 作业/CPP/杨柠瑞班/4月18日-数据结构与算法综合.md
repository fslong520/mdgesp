### 1、<a href="https://fslong.iok.la/problem/P9044" target="_blank">星际探险家排名</a>(100分)

**题目描述**  
在未来世界,星际探险家公会每年都会举行探险大赛。每位探险家有姓名、探索星球数量和获得的积分。

比赛结束后,公会需要按照以下规则对探险家进行排名:
1. 首先按积分从高到低排序
2. 如果积分相同,按探索星球数量从多到少排序
3. 如果还相同,按姓名字典序从小到大排序

小明是公会的数据分析师,需要你编写程序完成排名任务。

**输入格式**  
第一行一个整数 n (1 ≤ n ≤ 100),表示探险家数量。  
接下来 n 行,每行包含一个字符串和两个整数,分别表示姓名、探索星球数量和积分。

**输出格式**  
输出 n 行,按排名顺序输出每位探险家的信息,格式为:姓名 星球数 积分

**样例输入**  
```
4
张三 5 100
李四 8 100
王五 5 120
赵六 3 90
```

**样例输出**  
```
王五 5 120
李四 8 100
张三 5 100
赵六 3 90
```

**提示**  
李四和张三积分相同(100),但李四探索的星球更多(8 > 5),所以李四排在前面。

### 2、<a href="https://fslong.iok.la/problem/P9045" target="_blank">魔法学院奖学金</a>(100分)

**题目描述**  
魔法学院每学期都会颁发奖学金,评选规则比较特殊:
- 首先按成绩从**低到高**排序(鼓励进步)
- 成绩相同时,按魔法能力值从**高到低**排序
- 如果还相同,按学号从**小到大**排序

学院有 n 名学生,每名学生有学号、成绩和魔法能力值三个属性。院长请你编写程序,输出获得奖学金的学生排名。

**输入格式**  
第一行一个整数 n (1 ≤ n ≤ 100),表示学生数量。  
接下来 n 行,每行包含一个整数和两个整数,分别表示学号、成绩和魔法能力值。

**输出格式**  
输出 n 行,按排名顺序输出学生信息,格式为:学号 成绩 魔法能力值

**样例输入**  
```
3
101 95 80
102 85 85
103 85 90
```

**样例输出**  
```
103 85 90
102 85 85
101 95 80
```

**提示**  
注意第一关键字是成绩**升序**,第二关键字是魔法能力值**降序**,第三关键字是学号**升序**。

### 3、<a href="https://fslong.iok.la/problem/P9046" target="_blank">神秘宝藏坐标</a>(100分)

**题目描述**  
传说在一片神秘的大陆上,埋藏着无数个宝藏。每个宝藏的位置用一个坐标 (x, y) 表示。

考古学家发现了一个规律:如果宝藏坐标的 x 和 y 都是正整数,且 x + y = S(一个固定值),那么这个位置可能有宝藏。

现在给定 S 的值,请你找出所有可能的宝藏坐标,并按 x 从小到大的顺序输出。

**输入格式**  
一行,一个正整数 S (2 ≤ S ≤ 100)。

**输出格式**  
若干行,每行两个整数 x 和 y,表示一个可能的宝藏坐标。

**样例输入**  
```
5
```

**样例输出**  
```
1 4
2 3
3 2
4 1
```

**提示**  
x 和 y 都必须是正整数(≥ 1),所以 x 的范围是 1 到 S-1。

### 5、<a href="https://fslong.iok.la/problem/P9048" target="_blank">外星语言翻译器</a>(100分)

**题目描述**  
科学家接收到了一段外星文明的信号,经过研究发现,外星人使用一种特殊的计数方式:
- 他们用**倒序**的八进制来记录数字

也就是说,需要先将十进制转换为八进制,然后将结果**翻转**过来。

例如:十进制的 156 转换为八进制是 234,翻转后变成 432。

**输入格式**  
一行,一个十进制正整数 n (1 ≤ n ≤ 10000)。

**输出格式**  
一行,表示转换并翻转后的八进制数。

**样例输入**  
```
156
```

**样例输出**  
```
432
```

**样例输入 2**  
```
64
```

**样例输出 2**  
```
001
```

**提示**  
先转换为八进制(不断除以8取余数),注意余数序列本身就是倒序的,直接输出即可。

### 6、<a href="https://fslong.iok.la/problem/P9049" target="_blank">时间机器校准</a>(100分)

**题目描述**  
小明发明了一台时间机器,但时间机器的控制系统使用了一种特殊的十六进制编码:
- 每个十六进制位都要**加 1**后再显示(A变成B, B变成C, ..., E变成F, F变成0)

例如:十进制的 15 转换为十六进制是 F,加 1 后变成 0。
十进制的 255 转换为十六进制是 FF,加 1 后变成 00。

十六进制使用 0-9 和 A-F 表示。

**输入格式**  
一行,一个十进制正整数 n (1 ≤ n ≤ 65535)。

**输出格式**  
一行,表示经过编码后的十六进制数(字母大写)。

**样例输入**  
```
15
```

**样例输出**  
```
0
```

**样例输入 2**  
```
255
```

**样例输出 2**  
```
00
```

**样例输入 3**  
```
26
```

**样例输出 3**  
```
2B
```

**提示**  
先正常转换为十六进制,然后将每个字符 +1。注意 F 加 1 后变成 0(循环)。
字符转换:0→1, 1→2, ..., 8→9, 9→A, A→B, ..., E→F, F→0。
