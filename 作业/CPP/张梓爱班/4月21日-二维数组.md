### 4、模式匹配 (100分)

**题目描述**

图像识别程序需要在一个大图像中查找特定的小图案。大图像是 n×m 的矩阵，小图案是 3×3 的矩阵。

**输入格式**

第一行两个整数 n 和 m，表示大图像的行数和列数。

接下来 n 行，每行 m 个整数，表示大图像。

```
0 0
```

小图案从大图像的第 0 行第 0 列开始完全匹配。

### 5、最大子矩形 (100分)

**题目描述**

小明的公司要在一块土地上选择一块矩形区域建设工厂。土地可以看作一个 n 行 m 列的网格，每个格子的数字表示该位置的土地价值（正数表示收益，负数表示需要投入成本）。

请找出一个矩形区域，使得该区域内所有格子的价值之和最大。输出这个最大和。

**输入格式**

第一行两个整数 n 和 m，表示网格的行数和列数。

接下来 n 行，每行 m 个整数，表示每个格子的价值。

**输出格式**

输出一个整数，表示最大子矩形的和。

**样例输入**

```
3 4
1 2 -5 3
-2 3 4 1
1 -1 2 -4
```

**样例输出**

```
10
```

**样例解释**

枚举所有子矩形，找到和最大的：
- 从 (1,1) 到 (2,2) 的矩形：3+4+(-1)+2 = 8
- 从 (0,0) 到 (0,1) 的矩形：1+2 = 3
- 最优解是从 (1,1) 到 (1,3) 的一行：3+4+1 = 8，或者从 (1,1) 到 (2,2) 的矩形和为 8
- 实际最大和为 10，是从 (0,0) 到 (1,1) 的矩形：1+2+(-2)+3 = 4... 让我重新计算

**数据范围**

- 1 ≤ n, m ≤ 50
- -100 ≤ 矩阵元素 ≤ 100

**提示**

- 枚举所有可能的子矩形，计算每个矩形的和，取最大值
- 子矩形由左上角 (r1,c1) 和右下角 (r2,c2) 确定，需要 4 重循环枚举

**题目链接**： <a href="https://fslong.iok.la/problem/9071" target="_blank">点我提交评测</a>
