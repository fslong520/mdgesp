今天我们学习的是《记忆化搜索》，本次课的教学目的有：

1. 理解记忆化搜索的核心思想——DFS + 备忘录数组
2. 掌握记忆化搜索的通用写法框架：边界 → 查备忘录 → 递归计算 → 存备忘录 → 返回
3. 能识别记忆化搜索的常见错误（只查不存、初值冲突、方向写反）
4. 体会记忆化搜索与递推 DP 之间的关系

## 一、重点知识

### 什么是记忆化搜索？

记忆化搜索 = DFS 递归搜索 + 备忘录数组。

**空间换时间**：将已经算过的子问题结果存下来，下次再遇到直接返回，避免重复计算。

**通用框架**：
```cpp
int dfs(状态参数) {
    // 1. 边界条件
    if (到达边界) return 基准值;
    // 2. 查备忘录——已经算过就直接返回
    if (mem[状态] != 初值) return mem[状态];
    // 3. 递归计算
    int res = 初始值;
    for (所有可能的选择) {
        res = 最优(res, dfs(新状态) + 代价);
    }
    // 4. 存备忘录 + 返回
    return mem[状态] = res;
}
```

### 1. 爬楼梯（4071）——记忆化的基础

**孩子代码**：
```cpp
int dfs(int n) {
    if (n <= 3) return n;
    if (shi[n] != 0) return shi[n];     // 查备忘录 ✅
    return dfs(n-1) + dfs(n-2);          // ❌ 只查不存！
}
```

**⚠️ Bug**：备忘录只查不存。`return shi[n] = dfs(n-1)+dfs(n-2);` 少了 `shi[n]=` 的赋值，记忆化形同虚设，n 大时会指数级递归。

**正确写法**：
```cpp
return shi[n] = dfs(n-1) + dfs(n-2);  // ✅ 查完也要存
```

### 2. 游戏题（P9379）——记忆化 + 取模

**题意**：`f[n] = f[n-a] + f[n-b]`，`n ≤ c` 时值为 1，答案模 1e9+7。

**孩子 AC 代码**：
```cpp
long long dfs(long long n) {
    if (n <= c) return 1;
    if (xzz[n] != 0) return xzz[n];
    xzz[n] = dfs(n - a) + dfs(n - b);
    return xzz[n] % (1000000000ll + 7);
}
```

**调试过程**：

| 版本 | 问题 | 原因 |
|:----:|:----:|:----:|
| CE/TLE/RE | 语法、数组大小 | 基础错误 |
| Partial 60 | **模数写错** | `10000000000ll/100+7 = 100000007`（少一个 0） |
| **AC** | 模数修正 | `1000000000ll+7 = 1000000007` ✅ |

记忆化框架**完全正确**——先查后存，符合标准模式。唯一问题：模数抄错了。

### 3. 数字三角形（P9381）——记忆化的经典应用

**孩子 AC 代码**：
```cpp
int dfs(int i, int j) {
    if (i == n - 1) return a[i][j];           // 边界：到底了
    if (f[i][j] != -1) return f[i][j];        // 查备忘录
    f[i][j] = a[i][j] + max(dfs(i+1, j), dfs(i+1, j+1));  // 计算并存储 ✅
    return f[i][j];
}
```

**评价**：记忆化搜索写法标准，一遍 AC。查→算→存三步齐全。

### 4. 滑雪（A1278）——记忆化搜索的王者题

**题意**：二维网格，从任意点出发找最长严格递减路径。四方向。

**孩子 AC 代码**：
```cpp
int dfs(int i, int j) {
    if (i < 0 || i >= n || j < 0 || j >= m) return 0;
    if (f[i][j]) return f[i][j];     // 查备忘录
    int l = 0;
    if (a[i-1][j] < a[i][j]) l = max(l, dfs(i-1, j));  // 上
    if (a[i+1][j] < a[i][j]) l = max(l, dfs(i+1, j));  // 下
    if (a[i][j+1] < a[i][j]) l = max(l, dfs(i, j+1));  // 右
    if (a[i][j-1] < a[i][j]) l = max(l, dfs(i, j-1));  // 左
    return f[i][j] = l + 1;          // 存备忘录 ✅
}
```

**⚠️ 7 次提交逐版精析**：

| 版本 | 结果 | 错误 | 原因 |
|:----:|:----:|:------|:----:|
| 99572-99576 | **CE**×4 | `mx` 未声明 | 代码里 `mx=max(mx,dfs(i,j))` 但没定义 `mx` |
| 99577-99580 | **WA**×3 | **方向递归写反** | 检查上格子 `a[i-1][j]`，却递归到下格子 `dfs(i+1,j)` |
| **99581** | **AC** ✅ | 修正 | 向上方向改为 `dfs(i-1,j)` |

**Bug 特写**（WA 版）：
```cpp
if (a[i-1][j] < a[i][j]) l = max(l, dfs(i+1, j));  // 检查上面，却往下面递归！
```
典型**复制粘贴后漏改坐标**——从下方方向复制过来，改了判断条件忘了改递归参数。

### 5. 采药（P9431 / NOIP2005 普及组）——记忆化解背包

**题意**：T 时间采 m 株药，求最大价值。经典 0-1 背包。

**孩子 AC 代码**：
```cpp
int dfs(int i, int j) {
    if (j == m) return 0;                     // 边界：采完了
    if (f[i][j] != -1) return f[i][j];        // 查备忘录
    int b = dfs(i, j + 1);                    // 不采
    if (i < sj[j]) return f[i][j] = b;        // 时间不够
    int c = dfs(i - sj[j], j + 1) + jz[j];    // 采
    return f[i][j] = max(b, c);               // 存备忘录
}
```

状态定义 `f[i][j]` = 剩余 i 时间、第 j 株起考虑的最大价值。记忆化搜索解背包，思路正确，一遍 AC。

### 记忆化搜索的易错点总结

| 易错点 | 说明 | 出现题 |
|:------:|:----:|:------:|
| **只查不存** | 查了备忘录但递归结果没存入数组 | 爬楼梯 |
| **初值冲突** | memo 初值 0 恰好是有效答案会误判 | 滑雪（此处刚好可用） |
| **方向写反** | 复制粘贴后漏改坐标参数 | 滑雪 |
| **模数抄错** | 1e9+7 少写一个 0 | 游戏题 |
