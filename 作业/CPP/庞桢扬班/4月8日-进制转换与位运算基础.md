### 1、开关状态计算 (100分)

**题目描述**  
有一排开关，编号从 1 到 n。初始时全部关闭。

第 1 轮，将所有编号是 1 的倍数的开关翻转一次；
第 2 轮，将所有编号是 2 的倍数的开关翻转一次；
第 3 轮，将所有编号是 3 的倍数的开关翻转一次；
...
第 n 轮，将所有编号是 n 的倍数的开关翻转一次。

现在给你 n 的值，请问最终有多少个开关是开启状态？

**提示**：思考一下，一个开关被翻转奇数次还是偶数次决定了它的最终状态。什么数的因子个数是奇数？

**输入格式**  
一行，一个正整数 n (1 ≤ n ≤ 10000)。

**输出格式**  
一行，一个整数，表示最终开启状态的开关数量。

**样例输入**  
```
10
```

**样例输出**  
```
3
```

**提示**  
10个开关，最终编号为1、4、9的开关是开启状态，共3个。

### 2、密码锁的组合数(100分)

**题目描述**  
某保险箱的密码是一个十进制数，但说明书上只告诉了你这个数的二进制形式。你需要根据二进制提示，计算出正确的十进制密码。

但为了安全起见，保险箱实际使用的密码是原密码加上它的二进制中 1 的个数。例如，如果二进制是 1101（十进制 13），其中 1 的个数是 3，那么实际密码是 13 + 3 = 16。

**输入格式**  
一行，一个十进制正整数 n (1 ≤ n ≤ 1000)。

**输出格式**  
两行：
第一行输出对应的二进制数。
第二行输出实际使用的密码（十进制）。

**样例输入**  
```
13
```

**样例输出**  
```
1101
16
```

### 3、LED灯亮度调节(100分)

**题目描述**  
某智能灯具的 RGB 三个颜色通道可以用一个整数来表示：
- 高 8 位表示红色 (R)
- 中间 8 位表示绿色 (G)  
- 低 8 位表示蓝色 (B)

例如，整数 0x00FF00 表示 R=0, G=255, B=0（纯绿色）。

现在给你一个十六进制数，请分别提取出 R、G、B 三个颜色通道的十进制值。

**输入格式**  
一行，一个六位（或更少位）的十六进制数（仅包含 0-9 和 A-F）。

**输出格式**  
一行，三个整数 R、G、B，用空格分隔。

**样例输入**  
```
FF8000
```

**样例输出**  
```
255 128 0
```

**样例输入 2**  
```
1A2B3C
```

**样例输出 2**  
```
26 43 60
```

**提示**  
使用位运算：
- R = (color >> 16) & 0xFF
- G = (color >> 8) & 0xFF
- B = color & 0xFF

**题目描述**  
在计算机底层操作中，左移运算可以实现数据的快速乘法。给定一个整数 n，将其左移 k 位，计算最终结果。

**输入格式**  
一行，两个整数 n 和 k (1 ≤ n ≤ 1000, 0 ≤ k ≤ 10)。

**输出格式**

### 6、奇偶数快速判断(100分)

**题目描述**  
在数据处理中，经常需要快速判断一组数的奇偶性。使用位运算的方法比使用取模运算更高效。给定 n 个整数，判断每个数是奇数还是偶数。

**输入格式**  
第一行，一个整数 n (1 ≤ n ≤ 100)，表示数字的个数。
第二行，n 个整数 ai (0 ≤ ai ≤ 10000)。

**输出格式**  
n 行，每行输出一个字符串："odd" 表示奇数，"even" 表示偶数。

**样例输入**  
```
5
3 8 15 22 7
```

**样例输出**  
```
odd
even
odd
even
odd
```

**提示**  
使用按位与运算：如果一个数与 1 进行按位与的结果是 1，则是奇数；否则是偶数。
