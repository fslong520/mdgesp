# 分数线划定

#### 题目描述

期末考试结束后，学校要划定奖学金分数线：  

- **一等奖**：总人数前 20%（向下取整）的学生获得  
- **二等奖**：排名在 20%∼50% 之间（向下取整）的学生获得，从一等奖之后开始算  

请从高到低排序后，输出一等奖和二等奖的最低分数线（即获得该奖的最低成绩）。  
如果某个奖项的获奖人数为 0，对应输出 -1。

#### 输入格式

第一行，一个整数 n（1 ≤ n ≤ 1000）。  
第二行，n 个整数，表示每位同学的成绩（0 ≤ 成绩 ≤ 100）。

#### 输出格式

一行，两个整数，分别表示一等奖最低分和二等奖最低分，用空格隔开。

#### 样例输入 #1

```
10
95 88 92 75 82 78 60 65 70 85
```

#### 样例输出 #1

```
92 82
```

#### 样例解释 #1

排序：95 92 88 85 82 78 75 70 65 60  
一等奖：floor(10×20%)=2人 → 最低 92  
二等奖：floor(10×30%)=3人 → 最低 82

#### 样例输入 #2

```
6
90 85 80 75 70 65
```

#### 样例输出 #2

```
90 85
```

#### 样例解释 #2

一等奖：floor(6×20%)=1人 → 90  
二等奖：floor(6×30%)=1人 → 85

#### 数据范围

1 ≤ n ≤ 1000，0 ≤ 成绩 ≤ 100。

#### 知识点与难度

本题涉及的知识点从属于 **GESP 3级**（排序、模拟），难度等级：**⭐⭐**。
