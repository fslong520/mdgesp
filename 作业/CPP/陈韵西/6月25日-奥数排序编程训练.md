### 第 1 题 神秘运算 ★ [cyx01]（100 分）— <a href="https://fslong.iok.la/problem/CYX01" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

数学课上老师定义了一种新运算符号 ★：

a ★ b = (a + b) × |a − b|

小西发现这个运算很有意思——它跟平方差公式有点像！现在给定一个正整数 n，请按以下步骤计算：

1. 先算 1 ★ 2，得到结果记为 r
2. 再算 r ★ 3，新的结果记为 r
3. 再算 r ★ 4……
4. 一直算到 r ★ n

请输出最终的 r。

**输入格式**

一行，一个正整数 n（2 ≤ n ≤ 20）。

**输出格式**

一行，一个整数。

**样例输入 1**

```
3
```

**样例输出 1**

```
0
```

**样例解释 1**

1 ★ 2 = (1+2) × |1−2| = 3 × 1 = 3

3 ★ 3 = (3+3) × |3−3| = 6 × 0 = 0

**样例输入 2**

```
5
```

**样例输出 2**

```
231
```

**样例解释 2**

1 ★ 2 = 3

3 ★ 3 = 0

0 ★ 4 = (0+4) × |0−4| = 4 × 4 = 16

16 ★ 5 = (16+5) × |16−5| = 21 × 11 = 231

**提示**

用变量 r 记录当前结果，从 2 循环到 n，每次用 r 和 i 做 ★ 运算。注意 ★ 运算的定义：先加再乘绝对值差。

### 第 2 题 采蘑菇 [cyx02]（100 分）— <a href="https://fslong.iok.la/problem/CYX02" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

小西去森林采蘑菇。第 1 天采 10 朵，之后每天比前一天多采 3 朵。

但蘑菇需要休养生息——**每连续采满 7 天，第 8 天就必须休整**（采 0 朵），
休整之后第 9 天继续按正常的递增计划采（即数量接着第 7 天之后继续递增）。

现在给定总天数 n，请算一算 n 天一共采了多少朵蘑菇。

**输入格式**

一行，一个正整数 n（1 ≤ n ≤ 100）。

**输出格式**

一行，一个整数，表示 n 天总共采的蘑菇数。

**样例输入 1**

```
10
```

**样例输出 1**

```
198
```

**样例解释 1**

第 1 天：10  第 2 天：13  第 3 天：16  第 4 天：19  第 5 天：22  第 6 天：25  第 7 天：28

小计 133 朵，连续采满 7 天→第 8 天休整（0 朵）

第 9 天：31（接第 7 天之后的递增）  第 10 天：34

总计：10+13+16+19+22+25+28+0+31+34 = 198

**样例输入 2**

```
7
```

**样例输出 2**

```
133
```

**提示**

用 for 循环遍历每一天。设一个计数器 work_days 记录已连续工作的天数。
- 如果 work_days == 7，当天采 0 朵，work_days 重置为 0
- 否则，当天采的数量 = 10 + work_days × 3，work_days 加 1
- 把每天采的数量累加到总和中

### 第 3 题 最长连续段 [cyx03]（100 分）— <a href="https://fslong.iok.la/problem/CYX03" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

小西在整理一组数字时，发现有些数字是连续的——比如 3, 4, 5, 6 就是连续的自然数（相邻两数相差 1）。

现在给定一组无序的正整数，请找出其中最长的连续自然数段的长度。

所谓"连续"，是指数字本身是连续的（相邻两数相差 1），不要求它们在输入顺序上是连续的。

例如：1 3 4 5 6 9 10

- 连续段①：3, 4, 5, 6 → 长度 4
- 连续段②：9, 10 → 长度 2
- 连续段③：1 → 长度 1

最长的是 3 4 5 6，长度 4。

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 1000）。

第二行，n 个正整数，每个数不超过 10000。

**输出格式**

一行，一个整数，表示最长连续自然数段的长度。

**样例输入 1**

```
7
1 3 4 5 6 9 10
```

**样例输出 1**

```
4
```

**样例输入 2**

```
5
100 200 300 400 500
```

**样例输出 2**

```
1
```

**样例解释 2**

任意两个数之间相差 100，都不是连续自然数，所以最长段长度为 1（每个数自成一段）。

**提示**

先排序！排序后相邻元素如果相差 1 就属于同一个连续段。遍历排序后的数组，统计当前连续段长度，遇到断点就更新答案并重置计数。

### 第 4 题 频次统计 [cyx04]（100 分）— <a href="https://fslong.iok.la/problem/CYX04" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

输入 n 个整数，先从小到大排序，然后统计每个不同的数各出现了多少次。
输出时按数值从小到大的顺序，每行输出一个数值和它的出现次数。

例如：5, 3, 5, 1, 3, 3, 1, 5

排序后：1, 1, 3, 3, 3, 5, 5, 5

统计结果：
- 1 出现了 2 次
- 3 出现了 3 次
- 5 出现了 3 次

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 1000）。

第二行，n 个整数，每个数不超过 10000。

**输出格式**

若干行，每行两个整数：数值 出现次数，用空格隔开。按数值从小到大输出。

**样例输入 1**

```
8
5 3 5 1 3 3 1 5
```

**样例输出 1**

```
1 2
3 3
5 3
```

**样例输入 2**

```
5
100 200 100 300 200
```

**样例输出 2**

```
100 2
200 2
300 1
```

**提示**

先排序。然后用一个变量 cnt 记录当前数的出现次数。遍历数组 i 从 0 到 n−1：
- 如果 a[i] == a[i−1]，cnt++（跟前一个数相同）
- 否则，输出上一个数和它的 cnt，重置 cnt = 1
循环结束后别忘了输出最后一个数。

### 第 5 题 分数线划定 [cyx05]（100 分）— <a href="https://fslong.iok.la/problem/CYX05" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

期末考试结束后，学校要划定奖学金分数线：

- **一等奖**：总人数前 20%（向下取整）的学生获得
- **二等奖**：排名在 20%∼50% 之间（向下取整）的学生获得，从一等奖之后开始算

请从高到低排序后，输出一等奖和二等奖的最低分数线（即获得该奖的最低成绩）。
如果某个奖项的获奖人数为 0，对应输出 -1。

**输入格式**

第一行，一个整数 n（1 ≤ n ≤ 1000）。

第二行，n 个整数，表示每位同学的成绩（0 ≤ 成绩 ≤ 100）。

**输出格式**

一行，两个整数，分别表示一等奖最低分和二等奖最低分，用空格隔开。

**样例输入 1**

```
10
95 88 92 75 82 78 60 65 70 85
```

**样例输出 1**

```
92 82
```

**样例解释 1**

从高到低排序：95 92 88 85 82 78 75 70 65 60

一等奖人数：floor(10 × 20%) = 2 人 → 第 1~2 名，最低分 92

二等奖人数：floor(10 × 30%) = 3 人 → 第 3~5 名，最低分 82

**样例输入 2**

```
6
90 85 80 75 70 65
```

**样例输出 2**

```
90 85
```

**样例解释 2**

从高到低排序：90 85 80 75 70 65

一等奖：floor(6 × 20%) = 1 人 → 90 分

二等奖：floor(6 × 30%) = 1 人 → 85 分

**提示**

先按从高到低排序（可以用 sort 后倒序输出，或用 greater<int>() 降序）。一等奖人数 a = n × 20 / 100（整数除法自动向下取整），二等奖人数 b = n × 30 / 100。如果 a == 0，一等奖输出 -1，否则输出 a[a-1]（第 a 个人的成绩）。二等奖同理——如果 b == 0 或 a + b > n，输出 -1，否则输出 a[a+b-1]。
