# 最长连续段

#### 题目描述

小西在整理一组数字时，发现有些数字是连续的——比如 3, 4, 5, 6 就是连续的自然数（相邻两数相差 1）。

现在给定一组无序的正整数，请找出其中最长的连续自然数段的长度。

所谓"连续"，是指数字本身是连续的（相邻两数相差 1），不要求它们在输入顺序上是连续的。

例如：1 3 4 5 6 9 10  
- 连续段：3, 4, 5, 6 → 长度 4  
- 连续段：9, 10 → 长度 2  
最长的是 3 4 5 6，长度 4。

#### 输入格式

第一行，一个整数 n（1 ≤ n ≤ 1000）。  
第二行，n 个正整数，每个数不超过 10000。

#### 输出格式

一行，一个整数，表示最长连续自然数段的长度。

#### 样例输入 #1

```
7
1 3 4 5 6 9 10
```

#### 样例输出 #1

```
4
```

#### 样例输入 #2

```
5
100 200 300 400 500
```

#### 样例输出 #2

```
1
```

#### 数据范围

1 ≤ n ≤ 1000，每个数不超过 10000。

#### 知识点与难度

本题涉及的知识点从属于 **GESP 3级**（排序、数组遍历），难度等级：**⭐⭐**。
