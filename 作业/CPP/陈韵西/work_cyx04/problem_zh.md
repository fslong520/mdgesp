# 频次统计

#### 题目描述

输入 n 个整数，先从小到大排序，然后统计每个不同的数各出现了多少次。  
输出时按数值从小到大的顺序，每行输出一个数值和它的出现次数。

例如：5, 3, 5, 1, 3, 3, 1, 5  
排序后：1, 1, 3, 3, 3, 5, 5, 5  
- 1 出现 2 次  
- 3 出现 3 次  
- 5 出现 3 次

#### 输入格式

第一行，一个整数 n（1 ≤ n ≤ 1000）。  
第二行，n 个整数，每个数不超过 10000。

#### 输出格式

若干行，每行两个整数：数值 出现次数，用空格隔开。按数值从小到大输出。

#### 样例输入 #1

```
8
5 3 5 1 3 3 1 5
```

#### 样例输出 #1

```
1 2
3 3
5 3
```

#### 样例输入 #2

```
5
100 200 100 300 200
```

#### 样例输出 #2

```
100 2
200 2
300 1
```

#### 数据范围

1 ≤ n ≤ 1000，每个数不超过 10000。

#### 知识点与难度

本题涉及的知识点从属于 **GESP 3级**（排序、数组遍历），难度等级：**⭐⭐**。
