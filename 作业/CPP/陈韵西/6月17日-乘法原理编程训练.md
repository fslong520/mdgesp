### 第 1 题 毕业旅行 [P9511]（100 分）— <a href="https://fslong.iok.la/problem/P9511" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

毕业旅行要规划路线。从起点到景点 A 有 a 条路，从景点 A 到景点 B 有 b 条路，从景点 B 到终点有 c 条路。问从起点经过 A、B 到终点共有多少种走法？

**输入格式**

一行三个整数 a, b, c（1 ≤ a,b,c ≤ 30）。

**输出格式**

一个整数。

**样例输入**

```
2 3 4
```

**样例输出**

```
24
```

**提示**

三步累乘。试试用 do-while 循环，先执行再判断。

### 第 2 题 抽奖活动 [P9512]（100 分）— <a href="https://fslong.iok.la/problem/P9512" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

商场抽奖有 n 类奖品，每类有 a_i 个。从每类中各抽一个奖品，有多少种不同的中奖组合？

**输入格式**

第一行一个整数 n（1 ≤ n ≤ 10）。
第二行 n 个整数 a_1, a_2, ..., a_n（1 ≤ a_i ≤ 20）。

**输出格式**

一个整数。

**样例输入**

```
3
4 5 2
```

**样例输出**

```
40
```

**提示**

先读 n，再用数组存每类的数量，最后用 for 循环累乘。跟课堂上的加法原理比一比：加法累加、乘法累乘。

### 第 3 题 上学路线 [P9513]（100 分）— <a href="https://fslong.iok.la/problem/P9513" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

小西记录了一周每天从家到学校的路线。第 i 天从家到车站有 a_i 条路，从车站到学校有 b_i 条路。问每天各有多少种走法？

**输入格式**

第一行一个整数 t（1 ≤ t ≤ 7）。
接下来 t 行，每行两个整数 a_i, b_i（1 ≤ a_i, b_i ≤ 20）。

**输出格式**

t 行，每行一个整数，表示当天的走法数。

**样例输入**

```
3
2 3
1 4
5 2
```

**样例输出**

```
6
4
10
```

**提示**

多组数据，每组用两层循环处理——外层遍历天数，内层累乘。

### 第 4 题 选择题答案 [P9514]（100 分）— <a href="https://fslong.iok.la/problem/P9514" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

一张试卷有 n 道选择题，每道题有 k 个选项。如果不看题目乱猜，有多少种不同的答案组合？

**输入格式**

一行两个整数 n, k（1 ≤ n ≤ 10, 2 ≤ k ≤ 5）。

**输出格式**

一个整数。

**样例输入**

```
5 4
```

**样例输出**

```
1024
```

**样例解释**

每道题 4 种选择，n 道题就是 4 × 4 × ... × 4（共 n 个 4）。4^5 = 1024。

**提示**

循环 n 次，每次累乘 k。跟课堂上的加法原理比一比：加法是"或"，乘法是"且"——这里每道题都要选，所以用乘法。

### 第 5 题 哪种方案更多 [P9515]（100 分）— <a href="https://fslong.iok.la/problem/P9515" target="_blank">🧑‍💻 在线答题</a>

**题目描述**

商场有两种抽奖方案：

- 方案 A：分 a 类奖品，每类分别有 a_1, a_2, ..., a_a 个，从每类中各抽一个。
- 方案 B：分 b 类奖品，每类分别有 b_1, b_2, ..., b_b 个，从每类中各抽一个。

问哪种方案的可能结果总数更多？如果相同输出 `Same`。

**输入格式**

第一行一个整数 a（1 ≤ a ≤ 10）。
第二行 a 个整数（1 ≤ a_i ≤ 20）。
第三行一个整数 b（1 ≤ b ≤ 10）。
第四行 b 个整数（1 ≤ b_i ≤ 20）。

**输出格式**

一行：
- 如果 A 方案更多，输出 `A`
- 如果 B 方案更多，输出 `B`
- 如果相同，输出 `Same`

**样例输入**

```
2
3 4
3
2 2 2
```

**样例输出**

```
A
```

**样例解释**

A 方案：3 × 4 = 12 种。B 方案：2 × 2 × 2 = 8 种。A 更多，输出 A。

**提示**

分别计算两种方案的总数再比较。需要两个数组、两次循环累乘、一个 if-else 判断。
