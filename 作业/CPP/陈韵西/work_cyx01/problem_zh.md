# 神秘运算★

#### 题目描述

数学课上老师定义了一种新运算符号 ★：  

**a ★ b = (a + b) × |a − b|**

给定一个正整数 n，请按以下步骤计算：

1. 先算 1 ★ 2，得到结果记为 r  
2. 再算 r ★ 3，新的结果记为 r  
3. 再算 r ★ 4……  
4. 一直算到 r ★ n  

输出最终的 r。

#### 输入格式

一行，一个正整数 n（2 ≤ n ≤ 20）。

#### 输出格式

一行，一个整数。

#### 样例输入 #1

```
3
```

#### 样例输出 #1

```
0
```

#### 样例解释 #1

1 ★ 2 = (1+2) × |1−2| = 3 × 1 = 3  
3 ★ 3 = (3+3) × |3−3| = 6 × 0 = 0

#### 样例输入 #2

```
5
```

#### 样例输出 #2

```
231
```

#### 样例解释 #2

1 ★ 2 = 3，3 ★ 3 = 0，0 ★ 4 = 16，16 ★ 5 = 231

#### 数据范围

2 ≤ n ≤ 20

#### 知识点与难度

本题涉及的知识点从属于 **GESP 2级**（循环、自定义运算），难度等级：**⭐⭐**。
