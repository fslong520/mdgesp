## 一、重点知识

### 什么是加法原理？

**加法原理（Addition Principle）**：做一件事有 n 类办法，第 i 类有 m[i] 种不同方法，则完成这件事共有 N = m[1] + m[2] + ... + m[n] 种方法。

本质就是——**分类计数，各类相加**。

**通用代码框架**：
```cpp
int sum = 0;
for (int i = 0; i < n; i++) {
    // 读入第 i 类的选项数
    // sum += 该选项数
}
cout << sum;
```

### 1. 社团招新（MATH03001）——加法原理的入门

**题意**：3 类社团（学术、体育、艺术）分别有 a、b、c 个项目，小明只选一个项目，求总选法数。

**孩子代码**：
```cpp
int a = 3, b = 3, c = 3;
cin >> a >> b >> c;
int total = a + b + c;
cout << total;
```

**评价**：加法原理最直接的应用——固定 3 个数相加，一遍 AC。代码极简，151B。

### 2. 超市采购（MATH03002）——泛化到 k 类

**题意**：k 类商品，每类有 a[i] 种，妈妈只买一件，求总选法数。

**孩子代码**：
```cpp
cin >> k;
for (int i = 0; i < k; ++i) cin >> a[i];
for (int i = 0; i < k; ++i) h += a[i];
cout << h;
```

**评价**：从固定 3 类→k 类，用数组 + 循环累加解决问题。这是加法原理的「泛化」一步——能用循环处理不定长度输入。

### 4. 文具选购（MATH03004）——多组测试

**题意**：t 组测试数据，每组 n 类文具各类有 a[i] 种，选一件求总选法数。

**孩子代码**：
```cpp
cin >> t;
for (int i = 0; i < t; ++i) {
    cin >> n;
    for (int j = 0; j < n; ++j) cin >> a[j];
    int h = 0;
    for (int o = 0; o < n; ++o) h = h + a[o];
    cout << h << endl;
}
```

**评价**：从单组到多组测试，外层套 t 次循环。一次性 AC，说明对多组测试模式已熟练。
