# 采蘑菇

#### 题目描述
小西去森林采蘑菇。第1天采10朵，之后每天比前一天多采3朵。
但是，每连续采满7天，第8天必须休整（采0朵）。
休整后第9天继续按正常递增计划采。

给定总天数 $n$，请计算小西一共能采多少朵蘑菇。

#### 输入格式
一行，一个整数 $n$ ($1 \le n \le 100$)，表示总天数。

#### 输出格式
一行，一个整数，表示采到的蘑菇总数。

#### 样例输入 #1
```
10
```

#### 样例输出 #1
```
198
```

#### 样例输入 #2
```
7
```

#### 样例输出 #2
```
133
```

#### 数据范围
$1 \le n \le 100$

#### 知识点与难度
本题涉及的知识点从属于 **GESP 2级**，难度等级：**⭐**。
