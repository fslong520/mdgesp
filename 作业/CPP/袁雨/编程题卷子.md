# 袁雨编程题专项练习

### 1、<a href="https://fslong.iok.la/problem/" target="_blank">多批物资采购</a>(100分)

**题目描述**

游园会筹备组需要计算多批物资的采购预算。
第一行输入一个整数 t，表示共有 t 批物资。
接下来 t 行，每行包含两个正整数：数量 n 和单价 p，请计算出每批的总预算。

**输入格式**

第一行一个整数 t，表示批次数。
接下来 t 行，每行两个整数 n 和 p，表示采购数量和单价。

**输出格式**

输出 t 行，每行一个整数，表示对应批次的总预算。

**样例输入**
```
3
3 10
5 20
10 15
```

**样例输出**
```
30
100
150
```

**数据范围**
- 1 ≤ t ≤ 100
- 1 ≤ n ≤ 10000
- 1 ≤ p ≤ 10000

### 2、<a href="https://fslong.iok.la/problem/" target="_blank">区间人流查询</a>(100分)

**题目描述**

游园会的人流统计系统记录了连续 N 个小时的人流数据，现在需要快速回答多次查询：每次查询区间 [l, r]（下标从 0 开始），求区间内的人流总和。
请使用一维前缀和来高效处理这些查询。

**输入格式**

第一行一个整数 n，表示记录时长（小时数）。
第二行 n 个整数，表示每小时的人流量。
第三行一个整数 q，表示查询次数。
接下来 q 行，每行两个整数 l 和 r，表示查询区间。

**输出格式**

输出 q 行，每行一个整数，表示对应区间的人流总和。

**样例输入**
```
6
10 20 30 40 50 60
3
0 2
1 4
2 5
```

**样例输出**
```
60
150
180
```

**数据范围**
- 1 ≤ n ≤ 100000
- 1 ≤ q ≤ 100000
- -10000 ≤ 人流量 ≤ 10000

### 3、<a href="https://fslong.iok.la/problem/" target="_blank">门票价格推算</a>(100分)

**题目描述**

游园会的门票价格与游客年龄有关。工作人员发现了一个规律：门票价格随着年龄的变化而线性变化。
现在告诉你两个不同年龄的门票价格，请你找出这个线性关系，并回答多个查询：给定年龄 x，计算对应的门票价格 y。

**输入格式**

第一行两个整数 x1, y1，表示第一个游客的年龄和对应票价。
第二行两个整数 x2, y2，表示第二个游客的年龄和对应票价。
第三行一个整数 q，表示查询次数。
接下来 q 行，每行一个整数 x，表示要查询的年龄。

**输出格式**

输出 q 行，每行一个整数，表示对应年龄的门票价格。
题目保证 x1 ≠ x2，且所有计算结果均为整数。

**样例输入**
```
10 30
20 50
3
15
25
5
```

**样例输出**
```
40
60
20
```

**数据范围**
- -1000 ≤ x1, x2, x ≤ 1000
- -10000 ≤ y1, y2 ≤ 10000
- 1 ≤ q ≤ 1000

### 4、<a href="https://fslong.iok.la/problem/" target="_blank">数字配对统计</a>(100分)

**题目描述**

游园会有一个数字配对游戏：给定一个长度为 n 的数组，请统计有多少对数字的和等于目标值 target。
每对数字必须满足：第一个数字的位置在第二个数字之前（即 i < j），且每个数字只能使用一次。

**输入格式**

第一行一个整数 n，表示数字个数。
第二行 n 个整数，表示数组 a。
第三行一个整数 target，表示目标值。

**输出格式**

输出一个整数，表示满足条件的配对数量。

**样例输入**
```
5
1 2 3 4 5
6
```

**样例输出**
```
2
```

**解释**：(1,5) 和 (2,4) 两对数字的和等于 6。

**数据范围**
- 2 ≤ n ≤ 100
- 1 ≤ a[i] ≤ 1000
- 2 ≤ target ≤ 2000

### 5、<a href="https://fslong.iok.la/problem/" target="_blank">跷跷板配对</a>(100分)

**题目描述**

游园会的跷跷板游戏：有 n 个小朋友，每个小朋友有一个体重。
现在要选两个小朋友坐在跷跷板的两端，使得两端体重差的绝对值最小。
请找出这个最小的体重差。

**输入格式**

第一行一个整数 n，表示小朋友的个数。
第二行 n 个整数，表示每个小朋友的体重。

**输出格式**

输出一个整数，表示最小的两端体重差绝对值。

**样例输入**
```
4
3 1 4 2
```

**样例输出**
```
1
```

**解释**：选体重为 3 和 4 的小朋友，差为 |3-4|=1，是最小的可能。

**数据范围**
- 2 ≤ n ≤ 100
- 1 ≤ 体重 ≤ 1000
